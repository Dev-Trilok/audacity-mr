4 4 The scripting module is a plug-in for advanced users that allows Audacity to be driven from an external Python or Perl script. स्क्रिप्टिंग मॉड्यूल प्रगत वापरकर्त्यांसाठी एक प्लग-इन आहे जे ऑडिटीला बाह्य पायथन किंवा पर्ल स्क्रिप्टवरून चालविण्याची परवानगी देतो.
5 5 Commands are sent to Audacity over a 'named pipe'. कमांडस ऑडसिटीला 'नामित पाईप' वर पाठविल्या जातात.
6 6 Any scripting language that supports named pipes can be used. नामित पाईप्सला समर्थन देणारी कोणतीही स्क्रिप्टिंग भाषा वापरली जाऊ शकते.
7 7 Scripting is most often used from Python. स्क्रिप्टिंग बहुतेक वेळा पायथन मधून वापरली जाते.
8 8 The full set of Scripting Commands is listed at Scripting Reference. स्क्रिप्टिंग कमांडचा पूर्ण सेट स्क्रिप्टिंग संदर्भात सूचीबद्ध आहे.
9 9 Many of the commands in the Scriptables I and Scriptables II menus were created because they are useful in scripting. स्क्रिप्टेबल I आणि स्क्रिप्टेबल II मेनूमधील अनेक आज्ञा तयार केल्या गेल्या कारण त्या स्क्रिप्टिंगमध्ये उपयुक्त आहेत.
10 10 Alternatives to Python scripting that use the same set of commands are: पायथन स्क्रिप्टिंगचे विकल्प जे समान कमांडचा वापर करतात:
11 11 Macros which follow a fixed set of steps, e.g. to convert many files from one format to another. मॅक्रो जे चरणांच्या निश्चित संचाचे अनुसरण करतात उदा. बर्‍याच फाईल्स एका फाँट मधून दुसर्‍या फॉरमॅटमध्ये रुपांतरित करण्यासाठी
12 12 Nyquist, using the ';type tool' option. Nyquist, '; टाइप टूल' पर्याय वापरुन.
13 13 Nyquist is a built in LISP based language in Audacity. ऑयडॅसिटी मध्ये NIquist एक LISP आधारित भाषेत अंगभूत आहे.
14 14 What Scripting can do स्क्रिप्टिंग काय करू शकते
15 15 Commands that Scripting uses are the same as in the Audacity macros feature. स्क्रिप्टिंग ज्या कमांड वापरतात त्या ऑडिटी मॅक्रो वैशिष्ट्या प्रमाणेच असतात.
16 16 You can for example: आपण उदाहरणार्थ:
17 17 Select audio ऑडिओ निवडा
18 18 Apply effects प्रभाव लागू करा
19 19 Rearrange clips क्लिप पुन्हा व्यवस्थित करा
20 20 Export the results. निकाल निर्यात करा.
21 21 Scripting goes beyond the simpler tasks and presets of macros. स्क्रिप्टिंग मॅक्रोच्या सोप्या कार्ये आणि प्रीसेटच्या पलीकडे जाते.
22 22 Using Python scripting can for example do calculations about regions to select, or can make decisions on the basis of number and types of tracks in a project. पायथन स्क्रिप्टिंगचा उपयोग उदाहरणार्थ निवडण्यासाठी क्षेत्रांविषयी गणना करू शकतो किंवा प्रकल्पातील ट्रॅकच्या संख्येच्या आधारे निर्णय घेऊ शकतो.
23 23 It is also possible to build additional user interface in Python, for example an extra toolbar, and have it send commands to Audacity over the pipe. पायथनमध्ये अतिरिक्त यूजर इंटरफेस बनविणे देखील शक्य आहे, उदाहरणार्थ अतिरिक्त टूलबार, आणि पाईपवरून ऑडसिटीला कमांड पाठवा.

34 34 Caveats and Warnings गुहेत आणि चेतावणी
35 35 Scripting drives Audacity from outside the Audacity user interface. ऑडसिटी वापरकर्ता इंटरफेसच्या बाहेरून स्क्रिप्टिंग ड्राइव्हस् ऑडॅसिटी.
36 36 If you use mod-script-pipe to enable scripting, this weakens computer security, as malicious software that already has a foothold on your computer could potentially use this feature to gain more control. आपण स्क्रिप्टिंग सक्षम करण्यासाठी मॉड-स्क्रिप्ट-पाईप वापरत असल्यास, यामुळे संगणक सुरक्षा कमकुवत होईल, कारण आपल्या संगणकावर आधीपासून पाय ठेवलेले दुर्भावनायुक्त सॉफ्टवेअर अधिक नियंत्रण मिळविण्यासाठी संभाव्यत: हे वैशिष्ट्य वापरू शकते.
37 37 See the advice below about not using scripting on a web server. वेब सर्व्हरवर स्क्रिप्टिंग न वापरण्याबद्दल खाली दिलेला सल्ला पहा.
38 38 Scripting support is mainly intended for use by developers. स्क्रिप्टिंग समर्थन मुख्यतः विकसकांच्या वापरासाठी आहे.
39 39 When things "don't work" you will rarely get a message from Audacity saying why. जेव्हा गोष्टी "कार्य करत नाहीत" तेव्हा ऑडिटीकडून क्वचितच असा संदेश येईल की असे का.
40 40 Details of the available scripting commands may well change between versions of Audacity. उपलब्ध स्क्रिप्टिंग कमांडचा तपशील ऑडॅसिटीच्या आवृत्त्यांमधील फरक बदलू शकतो.
41 41 There is a fuller list of limitations at the foot of this page. या पृष्ठाच्या पायथ्याशी असलेल्या मर्यादांची पूर्ण यादी आहे.
42 42 Scripting is NOT SUITABLE for providing a service on a web server. वेब सर्व्हरवर सेवा प्रदान करण्यासाठी स्क्रिप्टिंग उपयुक्त नाही.
43 43 The reason is that Audacity does not police or sanitise the instructions that arrive on the pipe. कारण असे आहे की ऑडॅसिटी पाईपवर येणार्‍या सूचना पोलिस करीत नाही किंवा स्वच्छता करीत नाही.
44 44 It just attempts to obey the instructions. तो फक्त सूचनांचे पालन करण्याचा प्रयत्न करतो.
45 45 If someone can write to the pipe they can get Audacity to read and write files and to execute code on the machine running Audacity. जर कोणी पाईपवर लिहू शकत असेल तर त्या फायली वाचण्यास आणि लिहिण्यासाठी ऑडसिटी मिळवू शकतात आणि मशीनवर चालू असलेल्या ऑडॅसिटीवर कोड कार्यान्वित करू शकतात.
46 46 Enabling mod-script-pipe allows Audacity to be controlled from outside the Audacity User Interface. मोड-स्क्रिप्ट-पाईप सक्षम केल्याने ऑडसिटीला ऑडसिटी वापरकर्ता इंटरफेसच्या बाहेरून नियंत्रित केले जाऊ शकते.
47 47 In some environments, such as a web server, that is too big a security risk. वेब सर्व्हरसारख्या काही वातावरणात, हे खूपच सुरक्षिततेचे धोका असते.
48 48 Getting Started प्रारंभ करणे
49 49 Enable mod-script-pipe मोड-स्क्रिप्ट-पाईप सक्षम करा
50 50 Audacity now comes with a plug-in module called "mod-script-pipe". ऑडसिटी आता प्लग-इन मॉड्यूलसह ​​येते ज्याला "मॉड-स्क्रिप्ट-पाइप" म्हणतात.
51 51 You enable it using the Audacity preferences. आपण हे ऑडसिटी प्राधान्ये वापरून सक्षम करा.
52 52 After enabling it for the first time, you will need to restart Audacity. प्रथमच सक्षम केल्यानंतर, आपल्याला ऑडसिटी पुन्हा सुरू करण्याची आवश्यकता असेल.
53 53 You can then check that it is enabled and was started by revisiting the preferences page. त्यानंतर आपण ते सक्षम केले असल्याचे तपासू शकता आणि प्राधान्ये पृष्ठावर पुन्हा जाऊन प्रारंभ केला होता.
54 54 Run Audacity ऑडीसिटी चालवा
55 55 Go into edit->preferences->modules संपादन-> प्राधान्ये-> विभागांमध्ये जा
56 56 Choose mod-script-pipe (which should show 'New') and change that to 'Enabled'. मोड-स्क्रिप्ट-पाइप निवडा (जे 'नवीन' दर्शवावे) आणि ते 'सक्षम' मध्ये बदला.
57 57 Restart Audacity धृष्टता पुन्हा सुरू करा
58 58 Check that it now does show 'Enabled'. ते आता 'सक्षम' दर्शविते हे तपासा.
59 59 This establishes that Audacity is finding mod-script pipe, and that the version is compatible (compiled on the same day). हे स्थापित करते की ऑडॅसिटी मॉड-स्क्रिप्ट पाईप शोधत आहे, आणि आवृत्ती सुसंगत आहे (त्याच दिवशी संकलित).

90 90 Known Issues & Missing Features ज्ञात समस्या आणि गहाळ वैशिष्ट्ये
91 91 Some current issues: काही सद्य:
92 92 Scripting only works with one project at a time. स्क्रिप्टिंग एका वेळी केवळ एका प्रोजेक्टसह कार्य करते.
93 93 For some menu commands, the project window must have focus for the command to succeed. काही मेनू आदेशांकरिता, कमांड यशस्वी होण्यासाठी प्रकल्प विंडोमध्ये फोकस असणे आवश्यक आहे.
94 94 It's not straightforward to get 'output' responses from commands like GetInfo. गेटइन्फो सारख्या आदेशावरून 'आउटपुट' प्रतिसाद मिळविणे सोपे नाही.
95 95 You will need to parse the results in Python. आपल्याला पायथनमधील निकालांचे विश्लेषण करणे आवश्यक आहे.
96 96 The commands could have stricter parameter validation and could give better error messages when they do not work. कमांड्स मध्ये कठोर पॅरामीटर प्रमाणीकरण असू शकते आणि कार्य करत नसताना चांगले त्रुटी संदेश देऊ शकतात.
97 97 There's no consistent way to abort or interrupt commands. कमतरता रद्द करण्याचा किंवा व्यत्यय आणण्याचा कोणताही सुसंगत मार्ग नाही.
98 98 The scripting module is not unloaded when Audacity quits. This means the script pipes are not properly deleted. ऑडीसिटी सोडल्यास स्क्रिप्टिंग मॉड्यूल अनलोड केले जात नाही. याचा अर्थ स्क्रिप्ट पाईप्स योग्यरित्या हटविलेले नाहीत.
99 99 There may be security problems relating to the use of pipes. You're advised not to use mod-script-pipe on a system with multiple simultaneous users. पाईप्सच्या वापराशी संबंधित सुरक्षा समस्या असू शकतात. आपल्याला एकाचवेळी एकाधिक वापरकर्त्यांसह सिस्टमवर मॉड-स्क्रिप्ट-पाईप न वापरण्याचा सल्ला देण्यात आला आहे.
