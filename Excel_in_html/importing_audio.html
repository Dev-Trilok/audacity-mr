22 22 Other importable audio formats इतर आयात करण्यायोग्य ऑडिओ स्वरूप
23 23 You can install the optional FFmpeg library to import a much larger range of audio formats including AC3, AMR(NB), M4A, MP4 and WMA (if the files are not DRM-protected to work only in particular software). FFmpeg will also import audio from most video files or DVDs that are not DRM-protected. आपण एसी 3, एएमआर (एनबी), एम 4 ए, एमपी 4 आणि डब्ल्यूएमएसह मोठ्या प्रमाणात ऑडिओ स्वरूप आयात करण्यासाठी पर्यायी FFmpeg लायब्ररी स्थापित करू शकता (जर फायली केवळ विशिष्ट सॉफ्टवेअरमध्ये कार्य करण्यासाठी डीआरएम-संरक्षित नसल्यास). FFmpeg डीआरएम-संरक्षित नसलेल्या बर्‍याच व्हिडिओ फायली किंवा डीव्हीडी वरून ऑडिओ आयात करेल.
24 24 On Mac only, Audacity can import M4A, MP4 and MOV files without FFmpeg. केवळ मॅकवर, ऑडॅसिटी एफएफम्पेगशिवाय एम 4 ए, एमपी 4 आणि एमओव्ही फायली आयात करू शकते.

27 27 WAV files from portable recorders may contain proprietary compressed audio. These should not be imported as Raw Data, but by installing the FFmpeg library. पोर्टेबल रेकॉर्डरमधील डब्ल्यूएव्ही फायलींमध्ये मालकी संकुचित ऑडिओ असू शकतात. हे रॉ डेटा म्हणून आयात केले जाऊ नये, परंतु एफएफएमपीएजी लायब्ररी स्थापित करुन.
28 28 LOF files (lists of files) are also supported if the list is correctly formatted and contains supported files. यादी योग्यरित्या स्वरूपित असल्यास आणि त्यामध्ये समर्थित फायली असल्यास एलओएफ फायली (फाइल्सच्या याद्या) देखील समर्थित आहेत.

32 32 If you try to do this you will get an error message. आपण हे करण्याचा प्रयत्न केल्यास आपणास एक त्रुटी संदेश मिळेल.
33 33 Three Ways to Import Audio ऑडिओ आयात करण्याचे तीन मार्ग
34 34 No matter which method you use to bring an audio file into Audacity, the file is always imported into a saved or unsaved Audacity project. The imported file always appears in a new track in that project. ऑडसिटीमध्ये ऑडिओ फाईल आणण्यासाठी आपण कोणती पद्धत वापरता याचा फरक पडत नाही, ही फाइल नेहमी सेव्ह किंवा जतन न केलेल्या ऑडसिटी प्रोजेक्टमध्ये आयात केली जाते. आयात केलेली फाईल त्या प्रकल्पातील नवीन ट्रॅकमध्ये नेहमी दिसून येते.
35 35 1. File > Import > Audio... 1. फाइल> आयात> ऑडिओ ...
36 36 If you select the File > Import > Audio... command, then choose one or more audio files, Audacity will import the selected file(s) into the existing project. This is useful to bring the content of one or more audio files into a project that already contains audio (for example, to mix several audio files together). आपण फाइल> आयात> ऑडिओ ... आदेश निवडल्यास, एक किंवा अधिक ऑडिओ फायली निवडा, ऑडसिटी विद्यमान प्रकल्पात निवडलेल्या फायली आयात करेल. आधीपासून ऑडिओ असलेल्या प्रकल्पात एक किंवा अधिक ऑडिओ फायलींची सामग्री आणण्यासाठी हे उपयुक्त आहे (उदाहरणार्थ, बर्‍याच ऑडिओ फायली एकत्र मिसळण्यासाठी).

40 40 3. Drag and drop 3. ड्रॅग आणि ड्रॉप करा
41 41 Note that you cannot use drag and drop (into an open Audacity Project window) to open Audacity projects - use File > Open... to open projects. लक्षात ठेवा आपण ऑडसिटी प्रोजेक्ट उघडण्यासाठी ड्रॅग आणि ड्रॉप (ओपन ऑडीसिटी प्रोजेक्ट विंडोमध्ये) वापरू शकत नाही - प्रकल्प उघडण्यासाठी फाईल> ओपन ... वापरा.

43 43 Drag and drop one or more audio files into an open Audacity project window: this is equivalent to File > Import > Audio.... ओपन ऑडीसिटी प्रोजेक्ट विंडोमध्ये एक किंवा अधिक ऑडिओ फायली ड्रॅग आणि ड्रॉप करा: ही फाइल> आयात> ऑडिओ .... च्या बरोबरीची आहे.
44 44 Drag and drop one or more audio files onto the Audacity icon: this is equivalent to File > Open.... ऑडसिटी चिन्हावर एक किंवा अधिक ऑडिओ फायली ड्रॅग आणि ड्रॉप करा: ही फाईल> ओपन .... च्या समतुल्य आहे.
45 45 On Linux: लिनक्स वर:
46 46 Drag and drop files into an open Audacity Project window: this is equivalent to File > Import > Audio.... ओपन ऑडेसिटी प्रोजेक्ट विंडोमध्ये फायली ड्रॅग आणि ड्रॉप करा: हे फाईल> इंपोर्ट> ऑडिओ .... च्या बरोबरीचे आहे.
47 47 Drag and drop one or more audio files onto the Audacity icon: only the first file will open. More than one file dragged and dropped onto the icon will do the same as the File > Open... command for the first file, but the other files will generate an error message saying that Audacity is already running. ऑडसिटी चिन्हावर एक किंवा अधिक ऑडिओ फायली ड्रॅग आणि ड्रॉप करा: केवळ पहिली फाईल उघडेल. एकापेक्षा अधिक फाईल ड्रॅग आणि आयकॉनवर टाकल्या गेल्या फाईलसाठी ती फाईल> ओपन ... कमांड प्रमाणेच करेल, परंतु इतर फायली ऑडॅसिटी आधीपासून चालू आहे असे सांगून एक त्रुटी संदेश निर्माण करेल.
48 48 You can also use File > Open... आपण फाईल> ओपन ...
49 49 This command behaves differently according to whether the project window in which you use the Open command has ever contained tracks of any type. ज्या कमांड तुम्ही ओपन कमांड वापरता त्या प्रोजेक्ट विंडोमध्ये कुठल्याही प्रकारचा मागोवा असतो का यानुसार ही कमांड वेगळी वागणूक देते.
50 50 If the project contains or has ever contained tracks at some point in its history (so that Edit > Undo and Edit > Redo are not empty), Open imports each file you select into its own new project window. For example, selecting three files would create three new project windows, each containing one of the files and each a separate project with its own history. जर प्रकल्पात त्याच्या इतिहासातील काही क्षणी ट्रॅक असतील किंवा असेल (जेणेकरून संपादन> पूर्ववत करा आणि संपादित करा> पुन्हा करा रिक्त नाही), आपण निवडलेल्या प्रत्येक फाईलला त्याच्या स्वत: च्या नवीन प्रकल्प विंडोमध्ये उघडा आयात करा. उदाहरणार्थ, तीन फाईल्स निवडण्यामुळे तीन नवीन प्रोजेक्ट विंडोज तयार होतील, त्यातील प्रत्येकामध्ये एक फाईल असेल आणि प्रत्येकजण स्वतःचा इतिहासासह एक स्वतंत्र प्रकल्प असेल.
51 51 This is handy if you want to apply different actions to different (or even the same) files and easily compare the results of each. आपण भिन्न (किंवा समान) फायलींवर भिन्न क्रिया लागू करू इच्छित असल्यास आणि त्या प्रत्येकाच्या निकालांची सहज तुलना करू इच्छित असल्यास हे सुलभ आहे.
52 52 If the project is empty and has never contained any tracks (for example, you used File > New to create a new, empty project or used File > Close to close an existing project), Open imports the first file into the empty project window, then imports each subsequent file into its own new project window. जर प्रकल्प रिक्त असेल आणि यामध्ये कधीही ट्रॅक नसतील (उदाहरणार्थ आपण नवीन, रिक्त प्रकल्प तयार करण्यासाठी फाइल> नवीन वापरला किंवा विद्यमान प्रकल्प बंद करण्यासाठी फाइल> क्लोज वापरला), उघडा प्रथम रिक्त प्रकल्प विंडोमध्ये प्रथम फाइल आयात करते, त्यानंतर प्रत्येक फाइल त्याच्या स्वतःच्या नवीन प्रकल्प विंडोमध्ये आयात करते.
53 53 As with the other three import methods, Open does not directly open your selected file(s). The imported audio data is converted into a large number of small files in AU audio format, preserving the current quality of the data. Conversion occurs even if you never explicitly save your work as an Audacity project.So to save your edits to an audio file for the computer you always have to choose an Export command from the File Menu, even if you want to overwrite the exact same file that you imported. इतर तीन आयात पद्धतींप्रमाणेच, ओपन आपली निवडलेली फाईल थेट उघडत नाही. आयात केलेला ऑडिओ डेटा सध्याची डेटाची गुणवत्ता जपून एयू ऑडिओ स्वरूपात मोठ्या संख्येने लहान फायलीमध्ये रूपांतरित झाला आहे. जरी आपण आपले कार्य ऑडसिटी प्रोजेक्ट म्हणून स्पष्टपणे कधीही जतन केले नाही तरीही रूपांतरण उद्भवते. संगणकासाठी आपली संपादने ऑडिओ फाईलमध्ये जतन करण्यासाठी आपल्याला नेहमीच फाइल मेन्यूमधून निर्यात आदेश निवडावा लागेल, जरी आपल्याला तीच फाईल अधिलिखित करायची असेल तर जे आपण आयात केले.
54 54 Metadata मेटाडेटा
55 55 Audio files can carry descriptive information in a metadata container, this allows information such as the title, artist, album, track number, and other information about the file to be stored in the audio file itself. ऑडिओ फायली मेटाडाटा कंटेनरमध्ये वर्णनात्मक माहिती ठेवू शकतात, यामुळे शीर्षक, कलाकार, अल्बम, ट्रॅक क्रमांक आणि फाइलविषयीची अन्य माहिती ऑडिओ फाइलमध्येच संग्रहित केली जाऊ शकते.
56 56 Each time an audio file is imported into the same project window, the metadata tags for that file replace the previously stored metadata for the project. So if the last imported file has no metadata, the project will no longer have any stored metadata. Using one of the above methods that imports each file into its own project window will ensure each file's metadata is stored. प्रत्येक वेळी त्याच प्रोजेक्ट विंडोमध्ये एखादी ऑडिओ फाईल आयात केली जाते तेव्हा त्या फाईलसाठी मेटाडेटा टॅग प्रकल्पासाठी पूर्वी संग्रहित मेटाडेटा पुनर्स्थित करतात. तर शेवटच्या आयातीत फाईलमध्ये मेटाडेटा नसल्यास, यापुढे यापुढे कोणताही संचयित मेटाडेटा नसेल. वरीलपैकी एक पद्धत वापरली जी प्रत्येक फाईलला त्याच्या स्वतःच्या प्रकल्प विंडोमध्ये आयात केली तर प्रत्येक फाईलचा मेटाडेटा संचयित होईल याची खात्री होईल.
57 57 Using the file type dropdown menu in the Open and Import Audio Dialogs ओपन आणि इम्पोर्ट ऑडिओ डायलॉग मधील फाईल टाइप ड्रॉपडाउन मेनू वापरणे

60 60 The file type choice filters the list of files in the window by restricting it to files of that type. फाइल प्रकारची निवड विंडोमधील फाइल्सची सूची त्या प्रकारच्या फाईल्सपुरती मर्यादित ठेवून फिल्टर करते.

For example, a WAV file could be imported by Audacity's native WAV importer or by the optional FFmpeg library if this was installed. फाईल प्रकार निवड ओपन किंवा इम्पोर्ट ऑडिओ संवाद वापरताना कोणता आयातकर्ता प्रथम फायलीचा प्रयत्न करतो हे ठरवते. उदाहरणार्थ, डब्ल्यूएव्ही फाईल ऑडॅसिटीच्या मुळ डब्ल्यूएव्ही आयातकाद्वारे किंवा ही स्थापित केली असल्यास पर्यायी एफएफएमपीजी लायब्ररीद्वारे आयात केली जाऊ शकते.
62 62 Additionally, rules for the order in which different importers attempt files of particular types can be created at Extended Import Preferences with an option to over-ride the Open and Import Audio file type choice. See Import Filtering and Importer Order for more details. या व्यतिरिक्त, ओपन अँड इम्पोर्ट ऑडिओ फाईल प्रकार निवडीवर जादा-राइडिंग पर्याय असलेल्या विस्तारित आयात प्राधान्ये येथे विविध आयातकर्त्यांनी विशिष्ट प्रकारच्या फायलींचा प्रयत्न केल्याच्या ऑर्डरचे नियम तयार केले जाऊ शकतात. अधिक तपशीलांसाठी आयात फिल्टरिंग आणि आयातकर्ता ऑर्डर पहा.

70 70 Import Methods आयात पद्धती
71 71 Make a copy before editing (safer) संपादन करण्यापूर्वी एक प्रत बनवा (अधिक सुरक्षित)
72 72 This is the default setting. When this option is chosen, Audacity copies the imported file(s) into the project's audio data. It is essential that files are copied into the project if the Audacity project is to be moved, opened on another computer or sent to someone else, or if you need to to move, rename or delete the original files. ही डीफॉल्ट सेटिंग आहे. जेव्हा हा पर्याय निवडला जातो तेव्हा ऑडॅसिटी आयात केलेल्या फायली (ओं) प्रोजेक्टच्या ऑडिओ डेटामध्ये कॉपी करते. ऑडसिटी प्रोजेक्ट हलविणे, दुसर्‍या संगणकावर उघडणे किंवा दुसर्‍या एखाद्याला पाठविणे किंवा फाइल हलविणे, नाव बदलणे किंवा मूळ फाइल्स हटविणे आवश्यक असल्यास त्या प्रकल्पात फाइल्सची प्रत बनवणे आवश्यक आहे.
73 73 This option does not make a backup copy of the original file in its original location. If you do not want to lose the original content of the file when exporting to the same file name and location, please change the file name when exporting. हा पर्याय मूळ फाइलच्या मूळ स्थानावरील बॅकअप प्रत बनवित नाही. त्याच फाईलच्या नावावर आणि स्थानावर निर्यात करताना तुम्हाला फाईलची मूळ सामग्री गमवायची नसेल, तर कृपया निर्यात करताना फाईलचे नाव बदला.
74 74 Read directly from the original (faster) मूळ (वेगवान) वरून थेट वाचा

Therefore with this option set, you must not move, rename or delete imported uncompressed files, or the folder they are in. The advantage of this option is that you can begin editing a long file almost immediately, since On-Demand Loading will be used to import the file. जेव्हा हा पर्याय निवडला जातो तेव्हा ऑडॅसिटी नेहमी आयात केलेल्या फायली आपण त्यामधून प्रोजेक्टमध्ये कॉपी केल्याशिवाय त्या त्या आयात केल्या त्या फोल्डरमधून नेहमीच वाचतात. म्हणूनच या ऑप्शन सेटसह, आपण आयातित कंप्रप्रेस फाइल किंवा त्यांचे असलेले फोल्डर हलवू नये, नाव बदलू किंवा हटवू नये. या पर्यायाचा फायदा म्हणजे आपण जवळजवळ त्वरित एक लांब फाईल संपादित करणे सुरू करू शकता, कारण ऑन-डिमांड लोडिंग वापरली जाईल फाईल आयात करण्यासाठी.

Therefore you must not move, rename or delete any "-old" files until you have completely finished working on the project. आपण त्याच फाईलचे नाव आणि मूळ फाईलच्या स्थानावर निर्यात केल्यास ऑडसिटी मूळ फाईलच्या सामग्रीची एक प्रत तयार करेल ("-ल्ड" आणि एक नंबर समाविष्ट करण्यासाठी पुनर्नामित केले जाईल) आणि नंतर या "-ल्ड" फाइलमधून प्रकल्प वाचेल . आपण प्रोजेक्टवर काम पूर्ण करेपर्यंत आपण कोणतीही "-લ્ડ" फायली हलवू, पुनर्नामित करू किंवा हटवू नका.
77 77 You can change your import preference at any time in Import / Export Preferences, and you can turn the warning dialog on and off in Warnings Preferences. आपण आयात / निर्यात प्राधान्ये कोणत्याही वेळी आपली आयात प्राधान्ये बदलू शकता आणि आपण चेतावणी प्राधान्यांमध्ये चेतावणी संवाद चालू आणि बंद करू शकता.

81 81 Importing Compressed Audio Files संकुचित ऑडिओ फायली आयात करीत आहे
82 82 When you import a compressed audio file into your project on Windows or Mac, either a lossy format like MP3, MP2, M4A or Ogg Vorbis or the lossless compressed FLAC format, there is no choice between "safer" and "faster" - the file's data is always copied into the project. जेव्हा आपण आपल्या प्रोजेक्टमध्ये विंडोज किंवा मॅकवर संकुचित ऑडिओ फाइल आयात करता, एमपी 3, एमपी 2, एम 4 ए किंवा ओग व्हॉर्बिस किंवा लॉसलेस कॉम्प्रेस केलेले एफएलएसी स्वरूप यासारखे हानिकारक स्वरूप असते, तेव्हा फायलीचे "सुरक्षित" आणि "वेगवान" दरम्यान पर्याय नसतो. डेटा नेहमी प्रकल्पात कॉपी केला जातो.
83 83 However by default on Linux, FLAC is imported by the Audacity importer for uncompressed files, so the warning for "Importing uncompressed audio files" will appear with the choice to copy in the audio or read it directly. Reading directly permits faster On-Demand Loading. तथापि लिनक्सवर डीफॉल्टनुसार, एफएलएसी ऑडसिटी आयातकर्त्याद्वारे संकुचित केलेल्या फायलींसाठी आयात केले जाते, म्हणून ऑडिओमध्ये कॉपी करण्यासाठी किंवा थेट वाचण्यासाठीच्या निवडीसह "कॉम्प्रेसप्रेस ऑडिओ फायली आयात करणे" ही चेतावणी दिसेल. वाचन थेट मागणीनुसार लोड करण्यास द्रुत परवानगी देते.
84 84 Re-encoding lossy compressed audio filesIf you import a lossy compressed audio file into Audacity, edit it then export it as a lossy compressed audio file, you will lose quality twice, once in the original encoding of the imported audio, then again when you export it from Audacity as a compressed file. So never do production work in lossy compressed audio formats, instead work with a lossless format such as WAV, AIFF or FLAC and for final use then export to compressed formats. लॉसी कॉम्प्रेस केलेले ऑडिओ फाईल्सचे पुन्हा एन्कोडिंग ऑडसिटीमध्ये आपण हानीकारक कॉम्प्रेस केलेल्या ऑडिओ फाईलची आयात करत असल्यास ती संपादित करा नंतर ती हानीकारक संकुचित ऑडिओ फाईल म्हणून निर्यात केल्यास आयात केलेल्या ऑडिओच्या मूळ एन्कोडिंगमध्ये एकदा आपण गुणवत्ता दोनदा गमावाल, नंतर पुन्हा निर्यात कराल तेव्हा ते एक संकुचित फाइल म्हणून ऑडॅसिटीपासून आहे. म्हणून हानीकारक कॉम्प्रेस केलेले ऑडिओ स्वरूपात उत्पादन कार्य कधीही करू नका, त्याऐवजी डब्ल्यूएव्ही, एआयएफएफ किंवा एफएलएसीसारख्या लॉसलेस फॉरमॅटसह कार्य करा आणि अंतिम वापरासाठी नंतर कॉम्प्रेस केलेल्या स्वरूपात निर्यात करा.
85 85 See this page in the Audacity Wiki for other applications that can be used to directly edit MP3 audio files, thereby avoiding lossy re-encoding. हे पृष्ठ अन्य अनुप्रयोगांसाठी ऑडसिटी विकीमध्ये पहा जे एमपी 3 ऑडिओ फायली थेट संपादित करण्यासाठी वापरले जाऊ शकतात, ज्यामुळे नुकसानीचे पुन्हा एन्कोडिंग टाळले जाईल.

89 89 Sample Rate and Sample Format of imported files नमुना दर आणि आयात केलेल्या फायलींचे नमुना स्वरूप
90 90 Audacity always imports files at their original sample rate, as displayed in the Track Information Area of the Track Control Panel. ट्रॅक कंट्रोल पॅनेलच्या ट्रॅक माहिती क्षेत्रामध्ये दर्शविल्यानुसार ऑडॅसिटी नेहमी त्यांच्या मूळ नमुन्या दरावर फायली आयात करते.
91 91 When importing a file into an empty project window (as always happens when using File > Open... as described below), the Project Rate control in Selection Toolbar changes if necessary to reflect the rate of the file. The Project Rate determines the sample rate a file will be exported at, so no further adjustment is needed to export that file at its original rate. However, once there is already an audio track of any origin in the project (either an imported file, a recording or generated audio), importing an audio file will never change the project rate. रिकाम्या प्रोजेक्ट विंडोमध्ये फाइल आयात करताना (नेहमीप्रमाणेच फाइल> ओपन वापरत असताना असे होते ... खाली वर्णन केल्यानुसार), फाईलचे दर प्रतिबिंबित करण्यासाठी आवश्यक असल्यास निवड टूलबारमधील प्रोजेक्ट रेट नियंत्रण बदलते. प्रोजेक्ट रेट एक फाइल निर्यात केली जाईल याचा नमुना दर निर्धारित करते, म्हणून त्या फाईलच्या मूळ दरावर निर्यात करण्यासाठी यापुढे समायोजित करण्याची आवश्यकता नाही. तथापि, एकदा एकदा प्रोजेक्टमध्ये कोणत्याही मूळचा ऑडिओ ट्रॅक (एकतर आयात केलेली फाईल, रेकॉर्डिंग किंवा व्युत्पन्न ऑडिओ) झाला की ऑडिओ फाईल आयात केल्यास प्रकल्प दर कधीही बदलणार नाहीत.
92 92 Generally, Audacity imports files at the sample format (bit depth) specified at Default Sample Format in Quality Preferences. This setting defaults to 32-bit float. Therefore by default an imported file will show in the Track Control Panel as 32-bit float, even if its original bit depth is lower. This is best for high-quality editing.
