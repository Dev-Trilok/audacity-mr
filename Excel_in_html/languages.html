4 4 Audacity can display in many non-English languages. By default, Audacity runs in the language the operating system uses (the language chosen in the Windows installer of Audacity only determines the interface language of the installer). To change the language Audacity runs in, use the Interface Preferences, or exit Audacity and modify the audacity.cfg settings file. Reinstalling Audacity does not change the language setting in audacity.cfg, unless you run the Windows installer of Audacity and explicitly choose Reset Preferences. धैर्य बर्‍याच इंग्रजी नसलेल्या भाषांमध्ये प्रदर्शित होऊ शकते. डीफॉल्टनुसार, ऑडीसिटी ऑपरेटिंग सिस्टम वापरत असलेल्या भाषेत चालते (ऑडसिटीच्या विंडोज इंस्टॉलरमध्ये निवडलेली भाषा केवळ इंस्टॉलरची इंटरफेस भाषा निर्धारित करते). ऑडॅसिटी ज्या भाषेत चालू आहे ती बदलण्यासाठी, इंटरफेस प्राधान्ये वापरा किंवा ऑडॅसिटीमधून बाहेर पडा आणि ऑडॅसिटी. सीएफजी सेटिंग्ज फाइल सुधारित करा. जोपर्यंत आपण ऑडॅसिटीचा विंडोज इंस्टॉलर चालवत नाही आणि स्पष्टपणे रीसेट प्राधान्ये निवडत नाही तोपर्यंत ऑडॅसिटी रीस्टॉल करणे ऑडेसिटी.सीएफजी मध्ये भाषा सेटिंग बदलत नाही.

10 10 Changing language when installing on Windows विंडोजवर स्थापित करताना भाषा बदलणे
11 11 When you install Audacity on Windows, during the installation you will get a dialog box which enables you to select the language to be used during the installation process. जेव्हा आपण विंडोजवर ऑडसिटी स्थापित करता, तेव्हा स्थापनेदरम्यान आपल्याला एक डायलॉग बॉक्स मिळेल जो आपल्याला स्थापना प्रक्रियेदरम्यान वापरण्याची भाषा निवडण्यास सक्षम करतो.
12 12 This choice of language is also set to be the working language on subsequent use of Audacity. भाषेची ही निवड त्यानंतरच्या ऑडॅसिटीच्या वापरासाठी कार्यरत भाषा देखील सेट केली जाईल.
13 13 This means that if you are on Windows and the language in your Audacity installation is not to your liking you can simply re-install Audacity and choose a different language. याचा अर्थ असा की आपण विंडोजवर असाल आणि आपल्या ऑडसिटी स्थापनेतील भाषा आपल्या आवडीनुसार नसेल तर आपण सहजपणे ऑडसिटी पुन्हा स्थापित करू शकता आणि भिन्न भाषा निवडू शकता.
14 14 Changing language in Interface Preferences इंटरफेस प्राधान्यांमध्ये भाषा बदलणे
15 15 Open the Interface Preferences, choose the specific language required from the "Language" box, then click OK. Most of the interface will change language at once, but a very few interface elements will change next time you restart Audacity. इंटरफेस प्राधान्ये उघडा, "भाषा" बॉक्समधून आवश्यक असलेली विशिष्ट भाषा निवडा, आणि नंतर ओके क्लिक करा. बहुतेक इंटरफेस एकाच वेळी भाषा बदलेल, परंतु पुढच्या वेळी आपण ऑडसिटी पुन्हा सुरू केल्यावर बरेच काही इंटरफेस घटक बदलेल.
16 16 Navigating the menus in an unfamiliar language अपरिचित भाषेत मेनू नेव्हिगेट करणे
17 17 If you cannot read the menus because the text is in an unfamiliar language or has ??? characters, you can navigate the menus sequentially to change the language. आपण मेनू वाचू शकत नाही कारण मजकूर अपरिचित भाषेत आहे किंवा आहे ??? वर्ण, आपण भाषा बदलण्यासाठी मेनू अनुक्रमे नॅव्हिगेट करू शकता.
18 18 Choose the second menu from the left in the Audacity menus at top left of the screen (on Mac, choose the first menu from the left) स्क्रीनच्या डावीकडील ऑडसिटी मेनूमध्ये डावीकडून दुसरा मेनू निवडा (मॅक वर, डावीकडून पहिला मेनू निवडा)
19 19 Choose the bottom item of that menu, (on Mac, choose the second item in that menu) which opens the Preferences त्या मेनूचा तळाचा आयटम निवडा, (मॅक वर, त्या मेनूमधील दुसरा आयटम निवडा) जो प्राधान्ये उघडेल
20 20 In the list on the left of Preferences, choose the fifth item from the top पसंतीच्या डावीकडे असलेल्या सूचीत, वरुन पाचवा आयटम निवडा
21 21 On the right, open the second dropdown box containing a list of the language choices उजवीकडे, भाषेच्या निवडीची यादी असलेला दुसरा ड्रॉपडाउन बॉक्स उघडा
22 22 Select the language you want and press Enter on your keyboard. आपल्याला पाहिजे असलेली भाषा निवडा आणि आपल्या कीबोर्डवरील एंटर दाबा.
23 23 Restart Audacity to fully complete the language change and re-order the Effect menu alphabetically. भाषा बदल पूर्णपणे पूर्ण करण्यासाठी ऑडेसिटी रीस्टार्ट करा आणि प्रभाव मेनूला वर्णक्रमानुसार पुन्हा क्रमवारी लावा.

41 41 Supported non-English languages इंग्रजी-नसलेल्या भाषा समर्थित
42 42 As well as US English, the Audacity software supports a significant number of other languages. There are no localized versions of Audacity. Instead, single versions of Audacity for the various operating platforms are shipped with gettext MO files. These files are then used to display the different languages. यूएस इंग्रजी तसेच ऑडसिटी सॉफ्टवेअर इतर भाषांच्या महत्त्वपूर्ण संख्येस समर्थन देते. ऑडॅसीची कोणतीही स्थानिक आवृत्त्या नाहीत. त्याऐवजी, विविध ऑपरेटिंग प्लॅटफॉर्मसाठी ऑडॅसिटीची एके आवृत्ती गेटटेक्स्ट एमओ फायलीसह पाठविली जाते. या फायली नंतर विविध भाषा प्रदर्शित करण्यासाठी वापरल्या जातात.
43 43 All translations are provided by volunteers and may at times be incomplete. Any untranslated parts of the software will appear in US English. We are always looking for new volunteers to help translate not only Audacity itself but also the main Audacity website and this Manual. सर्व भाषांतर स्वयंसेवकांनी प्रदान केले आहेत आणि काही वेळा ते अपूर्ण असतील. सॉफ्टवेअरचे कोणतेही अप्रतिबंधित भाग यूएस इंग्रजीमध्ये दिसतील. आम्ही केवळ ऑडॅसिटीच नव्हे तर मुख्य ऑडॅसिटी वेबसाइट आणि हे मॅन्युअल भाषांतरित करण्यास मदत करण्यासाठी नवीन स्वयंसेवक शोधत असतो.
