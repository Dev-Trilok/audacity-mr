10 10 Saving projects: Every time you save a standard Audacity project or a compressed copy of a project, Audacity will warn that this command is only for an Audacity project and that you must use an export command to produce an audio file that you can open in other applications. Once you understand this, you can turn off both these warnings from within either warning dialog. प्रकल्प जतन करीत आहे: प्रत्येक वेळी आपण प्रमाणित ऑडसिटी प्रोजेक्ट किंवा प्रोजेक्टची संकुचित प्रत जतन करता तेव्हा ऑडिटी चेतावणी देईल की ही आज्ञा केवळ ऑडॅसिटी प्रोजेक्टसाठी आहे आणि आपण दुसर्‍यामध्ये उघडू शकणारी ऑडिओ फाइल तयार करण्यासाठी आपल्याला एक्सपोर्ट कमांड वापरणे आवश्यक आहे अनुप्रयोग. एकदा आपल्याला हे समजल्यानंतर आपण दोन्ही चेतावणी एकतर चेतावणी संवादातून बंद करू शकता.
11 11 Saving empty project: Audacity will warn when you try to save (left-hand image) or close (right-hand image) a project that previously contained tracks but is now empty. This warning can only be turned off in Preferences (not by a checkbox in the warning) because of the risk of accidental data loss if the warning is disabled. रिक्त प्रकल्प जतन करीत आहे: जेव्हा आपण ट्रॅक असलेले परंतु आता रिक्त असलेले प्रकल्प (डावीकडील प्रतिमा) जतन करण्याचा किंवा उजवीकडील प्रतिमा बंद करण्याचा प्रयत्न करता तेव्हा धोक्याची सूचना होईल. ही चेतावणी केवळ प्राधान्यांमध्ये बंद केली जाऊ शकते (चेतावणीमधील चेकबॉक्सद्वारे नाही) चेतावणी अक्षम केल्यास अपघाती डेटा गमावण्याच्या धोक्यामुळे.
12 12 Low disk space at application start up: Audacity will warn on start up or when using File > New to create a new project if you have 100 MB or less space in the temporary folder for recording or editing. अनुप्रयोग सुरू असताना कमी डिस्कची जागाः आपल्याकडे रेकॉर्डिंग किंवा संपादनासाठी तात्पुरत्या फोल्डरमध्ये 100 एमबी किंवा त्यापेक्षा कमी जागा असल्यास नवीन प्रकल्प तयार करण्यासाठी फास्ट अप प्रारंभ करताना किंवा नवीन> फाइल वापरताना चेतावणी देईल.
13 13 When mixing down to stereo during export: Audacity will warn when a stereo audio file will be created as the result of an export operation that involves mixing two tracks or more. If you normally want to export to stereo audio files you can turn this warning off. निर्यातीच्या दरम्यान स्टीरिओमध्ये मिसळताना: दोन ट्रॅक किंवा त्यापेक्षा जास्त मिसळणार्‍या एक्सपोर्ट ऑपरेशनचा परिणाम म्हणून जेव्हा स्टिरीओ ऑडिओ फाइल तयार केली जाईल तेव्हा धोक्याचा इशारा होईल. आपण सामान्यपणे स्टिरिओ ऑडिओ फायलींवर निर्यात करू इच्छित असल्यास आपण ही चेतावणी बंद करू शकता.
14 14 When mixing down to mono during export: Audacity will warn when a mono audio file will be created as the result of an export operation that involves mixing two tracks or more. निर्यातीच्या दरम्यान मोनोमध्ये मिसळताना: दोन ट्रॅक किंवा त्याहून अधिक मिसळणार्‍या एक्सपोर्ट ऑपरेशनचा परिणाम म्हणून जेव्हा मोनो ऑडिओ फाइल तयार केली जाईल तेव्हा धोक्यात येईल.
15 15 When mixing with Custom FFmpeg export: Audacity will warn that your export will be to a single file accrding to your encoder settings. सानुकूल एफएफम्पेग एक्सपोर्टमध्ये मिसळताना: ऑडिटी चेतावणी देईल की आपला एक्सपोर्ट आपल्या एन्कोडर सेटिंग्जमध्ये दाखल असलेल्या एका फाइलमध्ये असेल.
