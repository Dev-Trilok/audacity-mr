4 4 A LOF file is a plain text file ending with the extension ".lof" that contains a list of audio files to import into an Audacity project window. It can optionally contain commands to create new project windows. एक एलओएफ फाइल एक साधा मजकूर फाईल आहे जी ".लोफ" विस्तारासह समाप्त होते ज्यामध्ये ऑडसिटी प्रोजेक्ट विंडोमध्ये आयात करण्यासाठी ऑडिओ फायलींची सूची असते. यात नवीन प्रोजेक्ट विंडोज तयार करण्यासाठी कमांड्स वैकल्पिकरित्या असू शकतात.
5 5 Syntax मांडणी
6 6 There are only two commands "file" and "window", each with associated options: तेथे फक्त दोन कमांडस आहेत "फाईल" आणि "विंडो", प्रत्येक संबंधित पर्यायांसहः
7 7 file: (required) specifies the full path to the file which then imports as a new track in the current window (the path must always be in double quotes) फाईल: (आवश्यक) फाइलचा संपूर्ण मार्ग निर्दिष्ट करते जी सध्याच्या विंडोमध्ये नवीन ट्रॅक म्हणून आयात करते (पथ नेहमी डबल कोट्समध्ये असणे आवश्यक आहे)
8 8 offset: an optional parameter to the file command, specifying the start time of the audio file in seconds ऑफसेट: फाइल कमांडसाठी पर्यायी पॅरामीटर, सेकंदात ऑडिओ फाईलचा प्रारंभ वेळ निर्दिष्ट करते
9 9 window: (optional) creates a new Audacity project window for a subsequent file in the list (the first file always creates a new window unless the existing project is empty) विंडो: (पर्यायी) सूचीतील त्यानंतरच्या फाइलसाठी नवीन ऑडसिटी प्रोजेक्ट विंडो तयार करते (विद्यमान प्रकल्प रिक्त असल्याशिवाय प्रथम फाइल नेहमीच नवीन विंडो तयार करते)
10 10 offset: an optional parameter to the window command, specifying the left-most time displayed in the window ऑफसेट: विंडो कमांडचा एक वैकल्पिक पॅरामीटर, विंडोमध्ये डावीकडील सर्वात जास्त वेळ दर्शविणारा निर्दिष्ट करा
11 11 duration: an optional parameter to the window command, specifying how much time should be displayed in the window कालावधी: विंडो आदेशाचे वैकल्पिक मापदंड, जे विंडोमध्ये किती वेळ दाखवायचे ते निर्दिष्ट करते
12 12 #: (optional) can be used anywhere on a line to precede a comment - everything after # on that line is ignored by the importer. #: (पर्यायी) टिप्पणीच्या आधी ओळीवर कोठेही वापरले जाऊ शकते - त्या ओळीवरील # नंतरचे सर्वकाही आयातकाकडे दुर्लक्ष केले जाते.
13 13 Tips: टिपा:
14 14 Each command must be placed on a separate line. प्रत्येक कमांड वेगळ्या ओळीवर ठेवला पाहिजे.
15 15 Files are imported in the order shown in the LOF file. एलओएफ फाईलमध्ये दर्शविलेल्या क्रमाने फायली आयात केल्या जातात.
16 16 If the file to be imported is in the same folder as the LOF file, you need only give the file name inside double quotes. जर आयात करायची फाईल एलओएफ फाईलच्या समान फोल्डरमध्ये असेल तर आपल्याला दुहेरी अवतरणातील फाईलचे नाव देणे आवश्यक आहे.
17 17 Example उदाहरण
18 18 Given an empty project window, the following LOF file located at C:\Desktop imports the audio file "sample1.wav" from the Desktop into that window, followed by "sample2.wav" and "sample3.wav" from other locations underneath. Then a new project window is created and the audio file "sample4.wav" from the Desktop is imported into it. रिक्त प्रोजेक्ट विंडो दिल्यास, सी येथे असलेली खालील एलओएफ फाइल डेस्कटॉप त्या विंडोमध्ये डेस्कटॉपवरून "नमूना 1.wav" ऑडिओ फाइल आयात करते आणि त्याखालील खाली "नमून 2.wav" आणि "नमून3.wav" ऑडिओ फाईल आयात करते. मग एक नवीन प्रोजेक्ट विंडो तयार होईल आणि त्यामध्ये डेस्कटॉप वरून "નમૂના4.wav" ऑडिओ फाईल आयात केली जाईल.
19 19 # everything following the hash character is ignored # हॅश कॅरेक्टरच्या मागे असलेल्या प्रत्येक गोष्टीकडे दुर्लक्ष केले जाते
20 20 window # an initial window command is implicit and optional विंडो # इनिशियल विंडो कमांड अंतर्निहित आणि पर्यायी आहे
21 21 file "sample1.wav" फाइल "नमुना1.wav"
22 22 # sample1.wav is displayed # नमुना 1.wav दिसेल
23 23 file "C:\Ken\sample2.wav" offset 5 फाइल "सी: \ केन \ नमूना 2.wav" ऑफसेट 5
24 24 # sample2.wav is displayed with a 5s offset # नमुना 2.wav 5 एसच्या ऑफसेटसह प्रदर्शित होईल
25 25 file "C:\Dave\sample3.wav" फाइल "सी: ave डेव \ नमूना 3.wav"
26 26 # sample3.wav is displayed with no offset # નમૂના3.wav ऑफसेटशिवाय प्रदर्शित आहे
27 27 window offset 5 duration 10 विंडो ऑफसेट 5 कालावधी 10
28 28 # open a new window, then zoom to display # नवीन विंडो उघडा, त्यानंतर प्रदर्शन करण्यासाठी झूम करा
29 29 # 10 seconds of sample4.wav from 5 seconds to 15 seconds, offsetting it by 2.5 seconds # 10 सेकंद नमुना 4.वाव्ह 5 सेकंद ते 15 सेकंद पर्यंत ते 2.5 सेकंदाने ऑफसेट करत आहे
30 30 file "sample4.wav" offset 2.5 फाइल "नमुना 4.wav" ऑफसेट 2.5
31 31 The file names shown are for a Windows system, but the same syntax applies on Macintosh and Linux with the obvious change to a suitable file path. दर्शविलेल्या फाइलची नावे विंडोज सिस्टमसाठी आहेत, परंतु समान सिंटॅक्स योग्य फाईल पथातील स्पष्ट बदलासह मॅकिंटोश आणि लिनक्सवर लागू होते.
