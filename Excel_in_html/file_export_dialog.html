5 5 is accessed from File > Export > Export Audio.... When you use File > Export > Export Selected Audio... to export a single audio file from a selection, the Export Selected Audio dialog appears. Both these dialogs are identical and let you specify folder location, file name, format and encoding options for your exported file.You can use File > Export > Export Multiple... to export multiple audio files (based on labels or multiple audio tracks) in one process. फाईल वरून प्रवेश केला आहे> निर्यात करा> ऑडिओ निर्यात करा .... आपण निवडताना एकल ऑडिओ फाईल निर्यात करण्यासाठी फाइल> निर्यात> निवडलेले ऑडिओ निर्यात करता तेव्हा ... निवडलेले ऑडिओ निर्यात करा संवाद आढळतो. हे दोन्ही संवाद एकसारखे आहेत आणि आपल्याला आपल्या निर्यात केलेल्या फायलीसाठी फोल्डरचे स्थान, फाईलचे नाव, स्वरूप आणि एन्कोडिंग पर्याय निर्दिष्ट करू द्या. आपण एकाधिक ऑडिओ फायली निर्यात करण्यासाठी फाइल> निर्यात> एकाधिक निर्यात करा ... वापरू शकता (लेबल किंवा एकाधिक ऑडिओ ट्रॅकवर आधारित) एका प्रक्रियेत.
6 6 Note carefully that any offset (leading empty space in the track before the audio starts) is ignored and excluded from the export and is not rendered as silence as you might expect. Empty space between clips in the track will be rendered as silence in the export. काळजीपूर्वक लक्षात घ्या की कोणतेही ऑफसेट (ऑडिओ सुरू होण्यापूर्वी ट्रॅकमध्ये रिक्त जागा दर्शविणारे) दुर्लक्षित केले जाते आणि निर्यातीतून त्याला वगळले जाते आणि आपल्याला अपेक्षेप्रमाणे मौन म्हणून प्रस्तुत केले जात नाही. ट्रॅकमधील क्लिप दरम्यान रिक्त जागा निर्यातीमध्ये शांतता म्हणून प्रस्तुत केली जाईल.
7 7 For Export Selected Audio empty space between clips in the selection will be rendered as silence in the export. निर्यात निवडलेल्या ऑडिओकरिता निवडमधील क्लिपमधील रिक्त जागा निर्यातीमध्ये शांतता म्हणून प्रस्तुत केली जाईल.
8 8 The simple workaround if you want to preserve the offset as silence in the exported file for Export Audio is to insert a short section of silence right at the beginning of tracks that you require the offset for. ऑडिओ एक्सपोर्ट करण्यासाठी निर्यात केलेल्या फाईलमध्ये मौन म्हणून आम्ही ऑफसेट जतन करू इच्छित असल्यास सोपा कार्यवाही म्हणजे आपल्याला ट्रॅकच्या सुरूवातीस शांततेचा एक छोटा विभाग समाविष्ट करणे आवश्यक आहे ज्यासाठी आपल्याला ऑफसेट आवश्यक आहे.
9 9 When the Export is made Audacity will render the empty space between this silence and the audio as silence. जेव्हा एक्सपोर्ट केले जाईल तेव्हा ऑडसिटी शांतता आणि ऑडिओ मधील रिक्त स्थान रेकॉर्ड करेल.
10 10 Export Audio or Export Selected Audio dialog ऑडिओ निर्यात करा किंवा निवडलेले ऑडिओ संवाद निर्यात करा
11 11 In summary, there are five steps to follow when exporting. थोडक्यात, निर्यात करताना पालनासाठी पाच चरण आहेत.
12 12 Choose the folder to export to in Save in: then type the file name you want. सेव्ह इन मध्ये निर्यात करण्यासाठी फोल्डर निवडा: त्यानंतर आपल्याला पाहिजे असलेले फाइल नाव टाइप करा.
13 13 Select the type of audio file to export to. निर्यात करण्यासाठी ऑडिओ फाईलचा प्रकार निवडा.
14 14 Set the Format Options (if those are available for your selected format) if you want to change the size, quality or other encoding choices for that selected file format. आपण त्या निवडलेल्या फाइल स्वरूपनासाठी आकार, गुणवत्ता किंवा इतर एन्कोडिंग निवडी बदलू इच्छित असल्यास स्वरूप पर्याय सेट करा (जर ते आपल्या निवडलेल्या स्वरुपासाठी उपलब्ध असतील तर).
15 15 Press Save. सेव्ह दाबा.
16 16 The Metadata Editor dialog will then appear (unless you have disabled it for the export step in the Import/Export Preferences). Edit Metadata lets you enter embedded information in the file such as artist, year or genre. Either leave the metadata blank or complete the fields you require, then press OK (not "Save..."). त्यानंतर मेटाडेटा संपादक संवाद दिसेल (जोपर्यंत आपण आयात / निर्यात प्राधान्यांमध्ये निर्यात चरणात अक्षम केले नाही). मेटाडेटा संपादित करा आपल्याला कलाकार, वर्ष किंवा शैली यासारख्या फाइलमध्ये एम्बेड केलेली माहिती प्रविष्ट करू देते. एकतर मेटाडेटा रिक्त सोडा किंवा आपल्याला आवश्यक फील्ड पूर्ण करा, नंतर ओके दाबा ("जतन करा ..." नाही).
17 17 File name फाईलचे नाव
18 18 Type the required file name. The correct file extension - preceded by a period (dot) - will be added automatically to the end of the file name, according to the format selected in the "Save as type" list (see below). For example, typing "08 Voodoo Chile" (without quotes) will produce a file "08 Voodoo Chile.wav". For M4A and WMA formats (for which the optional FFmpeg library is required), you can add alternative extensions as specified in "FFmpeg formats" below.Any other extensions can be added after a file name as required, but Audacity will display a warning that this is not the normal extension for this file type, and that some applications might not play files with non-standard extensions. आवश्यक फाइल नाव टाइप करा. "फाईल टाईप सेव्ह" या यादीमध्ये निवडलेल्या फॉरमॅटनुसार (बिंदू) आधी - फाइल कालावधीच्या आधी सही फाइल विस्तार आपोआप जोडला जाईल (खाली पहा). उदाहरणार्थ, "08 वूडू चिली" (कोटेशिवाय) टाईप केल्यामुळे "08 वूडू चिली.वाव" फाईल तयार होईल. एम 4 ए आणि डब्ल्यूएमए फॉरमॅट्ससाठी (ज्यासाठी पर्यायी एफएफएमपीईजी लायब्ररी आवश्यक आहे), आपण खाली "एफएफएमपीजी फॉरमॅट्स" मध्ये निर्दिष्ट केल्याप्रमाणे वैकल्पिक विस्तार जोडू शकता. आवश्यकतेनुसार फाइलच्या नावानंतर कोणतेही इतर विस्तार जोडले जाऊ शकतात, परंतु ऑडेसिटी चेतावणी दर्शवेल की या फाईल प्रकारासाठी हे सामान्य विस्तार नाही आणि काही अनुप्रयोग कदाचित मानक नसलेल्या विस्तारांसह फायली प्ले करू शकत नाहीत.

42 42 If you use Export Audio or Export Selected Audio to export multiple tracks as a mono or stereo mixdown, or as a multi-channel audio file by enabling Advanced Mixing Options. आपण मोनो किंवा स्टिरीओ मिक्सडाउन म्हणून किंवा प्रगत मिक्सिंग पर्याय सक्षम करून एकाधिक-चॅनेल ऑडिओ फाइल म्हणून निर्यात ऑडिओ किंवा निर्यात निवडलेले ऑडिओ वापरत असल्यास.

44 44 Cancelling the export process निर्यात प्रक्रिया रद्द करत आहे
45 45 You may abort the export process without writing the file by pressing Cancel in the Export Audio / Export Selected Audio dialog, or in the Advanced Mixing Options or Metadata Editor dialogs (if those dialogs appear). If you Cancel in any dialog, any changes you made to the "Save in:" directory or file type will not be preserved for next time you export. आपण निर्यात ऑडिओ / निर्यात निवडलेले ऑडिओ संवादात किंवा प्रगत मिश्रण पर्याय किंवा मेटाडेटा संपादक संवादात (ते संवाद दिसत असल्यास) दाबून फाइल न लिहिता निर्यात प्रक्रियेचा त्याग करू शकता. आपण कोणत्याही संवादात रद्द केल्यास आपण पुढील वेळी निर्यात करता तेव्हा "सेव्ह इन:" निर्देशिका किंवा फाइल प्रकारात केलेले कोणतेही बदल संरक्षित केले जाणार नाहीत.
