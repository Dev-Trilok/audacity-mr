4 4 Enables you to import audio files or label files into your project. Files in various audio formats can be imported into Audacity. आपल्या प्रोजेक्टमध्ये आपल्याला ऑडिओ फायली किंवा लेबल फायली आयात करण्यास सक्षम करते. ऑडसिटीमध्ये विविध ऑडिओ स्वरूपांमधील फायली आयात केल्या जाऊ शकतात.

6 6 Audio...  Ctrl +Shift + I ऑडिओ ... Ctrl + Shift + I
7 7 Launches a file selection window where you can choose to import one or more audio files into the current Audacity project. The file(s) will always be added as a new track to the project. This lets you mix two or more files together. फाईल निवड विंडो लाँच करते जिथे आपण चालू ऑडसिटी प्रोजेक्टमध्ये एक किंवा अधिक ऑडिओ फायली आयात करणे निवडू शकता. फायली नेहमी या प्रकल्पात नवीन ट्रॅक म्हणून जोडली जातील. हे आपल्याला दोन किंवा अधिक फायली एकत्रित करू देते.
8 8 See Importing Audio for more information. अधिक माहितीसाठी ऑडिओ आयात करणे पहा.
9 9 Labels... लेबले ...
10 10 Launches a file selection window where you can choose to import a single text file into the project containing point or region labels. For more information about the syntax for labels files, see Importing and Exporting Labels. फाईल निवड विंडो लाँच करते जेथे आपण पॉईंट किंवा प्रदेश लेबले असलेल्या प्रकल्पात एकच मजकूर फाइल आयात करणे निवडू शकता. लेबल फायलींसाठी वाक्यरचना बद्दल अधिक माहितीसाठी, लेबले आयात करणे आणि निर्यात करणे पहा.
11 11 MIDI... मिडी ...
12 12 Imports a MIDI (MIDI or MID extension) or Allegro (GRO) file to a Note Track where the Note track can be played. टीप ट्रॅकवर एमआयडीआय (एमआयडीआय किंवा एमआयडी विस्तार) किंवा द्रुतगतीने (जीआरओ) फाइल आयात करते जिथे टीप ट्रॅक प्ले केला जाऊ शकतो.
13 13 This should just work on Windows but for playback on Mac and Linux additional software may be required, see this section on the Playing and Recording page. हे फक्त विंडोजवर कार्य केले पाहिजे परंतु मॅक आणि लिनक्सवर प्लेबॅकसाठी अतिरिक्त सॉफ्टवेअरची आवश्यकता असू शकते, हा विभाग प्लेइंग आणि रेकॉर्डिंग पृष्ठावर पहा.
14 14 Simple cut-and-paste edits can also be performed. The result can be exported with the File > Export > Export MIDI command. साधी कट-पेस्ट संपादने देखील केली जाऊ शकतात. फाइल> निर्यात> निर्यात एमआयडीआय आदेशासह निकाल निर्यात केला जाऊ शकतो.
15 15 Raw Data... कच्ची माहिती...
16 16 Attempts to import an uncompressed audio file that might be "raw" data without any headers to define its format, might have incorrect headers or be otherwise partially corrupted, or might be in a format that Audacity is unable to recognize. Raw data in textual format cannot be imported. एक संकुचित ऑडिओ फाईल आयात करण्याचा प्रयत्न ज्याचे स्वरूप निश्चित करण्यासाठी कोणत्याही शीर्षलेखांशिवाय "कच्चा" डेटा असू शकतो, चुकीचे शीर्षलेख असू शकतात किंवा अन्यथा अंशतः दूषित होऊ शकतात किंवा ऑडॅसिटी ओळखण्यास अक्षम असलेल्या स्वरूपात असू शकतात. मजकूर स्वरूपात कच्चा डेटा आयात केला जाऊ शकत नाही.
17 17 First, select the file in question in the "Select any uncompressed audio file" dialog. Then select appropriate parameters to assist Audacity in the formatting of the data. The fields in the dialog require you select: प्रथम, "कोणतीही संकुचित ऑडिओ फाइल निवडा" संवादातील प्रश्नातील फाइल निवडा. नंतर डेटाच्या स्वरूपणात ऑडेसिटीला मदत करण्यासाठी योग्य मापदंड निवडा. संवादातील फील्डसाठी आपण निवडणे आवश्यक आहे:
18 18 Encoding (PCM, ADPCM, float...) एन्कोडिंग (पीसीएम, एडीपीसीएम, फ्लोट ...)
19 19 Byte order (this is almost always Little-endian if the file was created on Windows) बाइट ऑर्डर (ही फाईल विंडोजवर तयार केली असल्यास हे नेहमीच लिटल-एन्डियन असते)
20 20 Number of channels (expected to be found in the file and created as a result) चॅनेलची संख्या (फायलीमध्ये सापडण्याची आणि परिणामी तयार केलेली)
21 21 Start offset in bytes बाइटमध्ये ऑफसेट प्रारंभ करा
22 22 Percentage amount of the file to import आयात करण्यासाठी फाईलची टक्केवारी
23 23 Sample rate to be applied to the import (currently, rates between 100 Hz and 384000 Hz are supported) आयातीवर लागू केलेला नमुना दर (सध्या 100 हर्ट्ज ते 384000 हर्ट्झ दरम्यानचे दर समर्थित आहेत)
