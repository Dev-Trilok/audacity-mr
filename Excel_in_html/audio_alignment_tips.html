4 4 This page explains how to align multiple tracks so that their audio starts at the same point on the timeline above the waveform. For example you would often want to do this when you have recorded one track against another and need to ensure the recorded track is properly synchronized with the other track(s). हे पृष्ठ एकाधिक ट्रॅक संरेखित कसे करावे हे स्पष्ट करते जेणेकरून त्यांचा ऑडिओ त्याच वेगापासून वेव्हफॉर्मच्या वरील टाइमलाइनवर प्रारंभ होईल. उदाहरणार्थ जेव्हा आपण एखादा ट्रॅक दुसर्‍या विरूद्ध रेकॉर्ड केला असेल तेव्हा आपणास असे करणे आवडेल आणि रेकॉर्ड केलेला ट्रॅक इतर ट्रॅक (टे) सह योग्यरितीने संकालित केला गेला पाहिजे याची खात्री करा.

10 10 Using the Tracks Menu ट्रॅक मेनू वापरणे
11 11 The menu commands in the Tracks > Align Tracks submenu provide an automatic way of aligning selected tracks with the cursor, the selection, or with the start of the project: ट्रॅक मध्ये संरेखित मेनू आज्ञा> ट्रॅक संरेखित करा सबमेनू कर्सर, निवड किंवा प्रकल्पाच्या सुरूवातीस निवडलेले ट्रॅक संरेखित करण्याचा स्वयंचलित मार्ग प्रदान करते:
12 12 Align End to End: Aligns the selected tracks one after the other, based on their top-to-bottom order in the project window. The first selected track remains where it is, the start of the second selected track aligns with the end of the first selected track and so on. समाप्ती ते समाप्ती संरेखित करा: प्रकल्प विंडोमधील त्यांच्या वरच्या-खालच्या क्रमावर आधारित निवडलेले ट्रॅक एकामागून एक संरेखित करते. पहिला निवडलेला ट्रॅक जिथे आहे तिथेच राहतो, दुस selected्या निवडलेल्या ट्रॅकची सुरूवात पहिल्या निवडलेल्या ट्रॅकच्या शेवटी संरेखित होते आणि याप्रमाणे.
13 13 Align Together: एकत्र संरेखित करा:
14 14 If you select multiple tracks, this command moves the tracks so that they all start at the same time, so audio within them changes its synchronization. आपण एकाधिक ट्रॅक निवडल्यास, ही आज्ञा ट्रॅक हलवते जेणेकरून ते सर्व एकाच वेळी प्रारंभ होतील, म्हणून त्यांच्यामधील ऑडिओ त्याचे संकालन बदलू शकेल.
15 15 Start to Zero: Aligns the start of selected tracks with the start of the project. झिरोपासून प्रारंभ करा: प्रकल्प सुरू झाल्याने निवडलेल्या ट्रॅकची सुरूवात संरेखित करते.
16 16 Start to Cursor/Selection Start: Aligns the start of selected tracks with the current cursor position or with the start of the current selection. कर्सर / निवड प्रारंभ करा प्रारंभः वर्तमान कर्सर स्थानासह किंवा वर्तमान निवडीच्या प्रारंभासह निवडलेल्या ट्रॅकचा प्रारंभ संरेखित करते.
17 17 Start to Selection End: Aligns the start of selected tracks with the end of the current selection. निवडीची समाप्ती सुरू करा: सध्याच्या निवडीच्या समाप्तीसह निवडलेल्या ट्रॅकची सुरूवात संरेखित करते.
18 18 End to Cursor/Selection Start: Aligns the end of selected tracks with the current cursor position or with the start of the current selection. कर्सर / निवडीची समाप्ती समाप्त: निवडलेल्या ट्रॅकचा शेवटच्या कर्सर स्थानासह किंवा वर्तमान निवडीच्या प्रारंभासह संरेखित करते.
19 19 End to Selection End: Aligns the end of selected tracks with the end of the current selection. निवडीची समाप्ती शेवट: वर्तमान निवडीच्या समाप्तीसह निवडलेल्या ट्रॅकचा शेवट संरेखित करते.
20 20 If Sync-Lock Tracks is enabled, unselected tracks in a Sync-Locked Track Group will also move so that all the tracks in that group remain synchronized, except for the Align "End to End" and "Together" commands where only the selected tracks will move. Aligning end to end or together overrides Sync-Lock Tracks if necessary, so the selected tracks in that group may become desynchronized. संकालन-लॉक ट्रॅक सक्षम केलेले असल्यास, संकालित-लॉक केलेला ट्रॅक गटातील निवड रद्द केलेले ट्रॅक देखील हलतील जेणेकरून त्या गटातील सर्व ट्रॅक संकालित राहतील, "एंड टू एंड" संरेखित करा आणि फक्त निवडलेले ट्रॅक हलवेल समाप्तीची समाप्ती संरेखित करणे किंवा एकत्रितपणे आवश्यक असल्यास समक्रमण-लॉक ट्रॅक अधिलिखित करते, जेणेकरून त्या गटामधील निवडलेले ट्रॅक विभाजित होऊ शकतात.
21 21 Time Shift Tool टाइम शिफ्ट टूल
22 22 An easy way for sighted users to align tracks with each other दृष्टी असलेल्या वापरकर्त्यांचा एकमेकांशी ट्रॅक संरेखित करण्याचा सोपा मार्ग
23 23 is to use the Time Shift Tool म्हणजे टाइम शिफ्ट टूल वापरणे
24 24 which is located at the top of the Audacity screen in the Tools Toolbar. जी साधने टूलबारमधील ऑडसिटी स्क्रीनच्या शीर्षस्थानी आहे.
25 25 To use the Time Shift Tool, click on the टाइम शिफ्ट टूल वापरण्यासाठी, वर क्लिक करा
26 26 button so it appears to be depressed, or press its shortcut F5. If you wish to change its shortcut to something other than F5, you can do so on the Keyboard tab of Preferences. Then zoom in with Ctrl + 1 (or ⌘ + 1 on a Mac) so you can make a more precise alignment. Click in the track you want to move with the left button of the mouse, hold the button down while dragging the track to your required position, then release the mouse button. बटण जेणेकरून ते निराश झाल्यासारखे दिसते किंवा त्याचे शॉर्टकट F5 दाबा. आपण त्याचे शॉर्टकट एफ 5 व्यतिरिक्त अन्य कशासाठी बदलू इच्छित असल्यास आपण हे प्राधान्यांच्या कीबोर्ड टॅबवर करू शकता. त्यानंतर Ctrl + 1 (किंवा मॅकवरील ⌘ + 1) सह झूम वाढवा जेणेकरून आपण अधिक अचूक संरेखन करू शकता. माउसच्या डाव्या बटणासह आपण हलवू इच्छित ट्रॅकवर क्लिक करा, ट्रॅक आपल्या आवश्यक स्थानावर ड्रॅग करताना बटण दाबून ठेवा, त्यानंतर माउस बटण सोडा.
27 27 Aligning tracks without a mouse - for visually impaired users दृष्टिहीन वापरकर्त्यांसाठी - माउसशिवाय ट्रॅक संरेखित करीत आहे
28 28 Visually impaired users can use this Time Shifter plug-in to time-shift tracks without a mouse. This is one of a number of दृष्टिहीन वापरकर्ते हे टाइम शिफ्टर प्लग-इन माउसशिवाय टाइम-शिफ्ट ट्रॅकमध्ये वापरू शकतात. हे असंख्यांपैकी एक आहे
29 29 Nyquist plug-ins available for use in Audacity and works on both mono and stereo tracks. Instructions are given in Timeshif.TXT inside the zip folder. To install new plug-ins, place them in the "Plug-Ins" folder inside the Audacity installation folder. On Windows computers, your installation folder is usually under "Program Files". On Mac, it is usually under "Applications". Time shifter will appear in the Effect Menu underneath the separator after you restart Audacity. ऑडीसिटीमध्ये वापरासाठी Nyquist प्लग-इन उपलब्ध आहेत आणि दोन्ही मोनो आणि स्टिरीओ ट्रॅकवर कार्य करतात. पिन फोल्डरमध्ये टाइमशीफ.टीएक्सटी मध्ये सूचना दिल्या आहेत. नवीन प्लग-इन स्थापित करण्यासाठी, त्यांना ऑडसिटी स्थापना फोल्डरमध्ये "प्लग-इन" फोल्डरमध्ये ठेवा. विंडोज संगणकांवर, आपले इन्स्टॉलेशन फोल्डर सामान्यत: "प्रोग्राम फाइल्स" अंतर्गत असते. मॅक वर, हे सहसा ""प्लिकेशन्स" अंतर्गत असते. आपण ऑडेसिटी रीस्टार्ट केल्यानंतर टाइम शिफ्टर विभाजक खाली मेनूमध्ये इफेक्ट मेनूमध्ये दिसून येईल.
30 30 Marking align points संरेखित बिंदू चिन्हांकित करीत आहे
31 31 Some users find it easier to align tracks together when there is a vertical reference point extending down through all the tracks. जेव्हा सर्व ट्रॅकमधून एक अनुलंब संदर्भ बिंदू वाढविला जातो तेव्हा काही वापरकर्त्यांना ट्रॅक एकत्र संरेखित करणे सोपे होते.
32 32 You can make such a reference point by dragging the cursor down कर्सर खाली ड्रॅग करून आपण असा संदर्भ बिंदू बनवू शकता
33 33 into the other tracks, and optionally by marking cursor positions with labels. इतर ट्रॅकमध्ये आणि वैकल्पिकरित्या लेबलसह कर्सर स्थान चिन्हांकित करून.
34 34 To drag the cursor into the other tracks, make sure you are using the Selection Tool इतर ट्रॅकमध्ये कर्सर ड्रॅग करण्यासाठी, आपण निवड साधन वापरत असल्याचे सुनिश्चित करा
35 35 (shortcut F1), and place your cursor at a point where you want to line up the tracks. Then hold down Shift on your keyboard and click in the Track Control Panels (where the mute/solo buttons are) of all your other tracks. Now you'll see the cursor displayed in all the tracks, and you can switch to the Time Shift Tool and line up all the tracks against the cursor line. (शॉर्टकट एफ 1) वर क्लिक करा आणि आपला कर्सर अशा ठिकाणी ठेवा जेथे आपण ट्रॅक लावायचे इच्छिता. नंतर आपल्या कीबोर्डवरील शिफ्ट दाबून ठेवा आणि आपल्या इतर सर्व ट्रॅकच्या ट्रॅक कंट्रोल पॅनेलमध्ये (जिथे निःशब्द / एकल बटणे आहेत) क्लिक करा. आता आपणास सर्व ट्रॅकमध्ये प्रदर्शित केलेला कर्सर दिसेल आणि आपण टाइम शिफ्ट टूलवर स्विच करू शकता आणि सर्व ट्रॅक कर्सर लाइनच्या विरूद्ध तयार करू शकता.
36 36 To add a label at a cursor point, use Edit > Labels > Add Label at Selection. You can use the shortcut Ctrl + B (or ⌘ + B on a Mac) instead. The label is created in a new Label Track underneath the audio tracks. When you click on the label, all tracks will be selected and the cursor will be displayed through all the tracks, just like when you selected all the tracks by clicking on their Track Control Panels. However the label provides a permanent reference point you can come back to, and because you can have multiple labels in a Label Track, there can be multiple points you can line up with. कर्सर बिंदूवर लेबल जोडण्यासाठी, संपादन> लेबल> निवडीवर लेबल जोडा. त्याऐवजी आपण Ctrl + B (किंवा मॅक वर ⌘ + बी) शॉर्टकट वापरू शकता. ऑडिओ ट्रॅकच्या खाली नवीन लेबल ट्रॅकमध्ये लेबल तयार केले गेले आहे. जेव्हा आपण लेबलवर क्लिक कराल, तेव्हा सर्व ट्रॅक निवडले जातील आणि कर्सर सर्व ट्रॅकमधून दिसून येईल जसे की आपण त्यांच्या ट्रॅक कंट्रोल पॅनेलवर क्लिक करून सर्व ट्रॅक निवडले आहेत. तथापि हे लेबल कायम संदर्भ बिंदू प्रदान करते जिथे आपण परत येऊ शकता आणि आपल्याकडे एका लेबल ट्रॅकमध्ये एकाधिक लेबले असू शकतात, तेथे अनेक पॉइंट्स असू शकतात ज्या आपण एकत्र करू शकता.
37 37 You could also generate a Rhythm Track before you put other tracks on screen, using the Generate > Rhythm Track command. This gives you visible and audible bars in that Rhythm Track at the beat points of a chosen tempo. You could use a beat point to set a cursor position then drag the cursor into the other tracks as described above. You need to close this Rhythm Track when you export your work (by clicking on the [X] top left of the track) so that you do not export the audible clicks. आपण व्युत्पन्न> लय ट्रॅक आज्ञा वापरून इतर ट्रॅक स्क्रीनवर ठेवण्यापूर्वी आपण एक रिदम ट्रॅक देखील व्युत्पन्न करू शकता. हे आपल्याला निवडलेल्या टेम्पोच्या बीट बिंदूवर त्या रिदम ट्रॅकमधील दृश्यमान आणि ऐकण्यायोग्य बार देते. कर्सर स्थान सेट करण्यासाठी आपण बीट पॉईंट वापरू शकता आणि वर वर्णन केल्याप्रमाणे कर्सर ड्रॅगला इतर ट्रॅकमध्ये ड्रॅग करा. आपण आपले काम निर्यात करता तेव्हा हा ताल ट्रॅक बंद करणे आवश्यक आहे (ट्रॅकच्या वरील [X] वर डावीकडे क्लिक करून) जेणेकरून आपण ऐकण्यायोग्य क्लिक निर्यात करू शकणार नाही.
38 38 If you use the current version of Audacity, there is no visual guesswork involved in aligning tracks: you can Snap-To labels when you time shift a track towards them, and if you drag a selection region, the start point of the area on the time line is displayed dynamically in the Selection Toolbar as you drag. जर आपण ऑडॅसिटीची सद्य आवृत्ती वापरत असाल तर ट्रॅक संरेखित करण्यात कोणतेही दृश्य अंदाज कार्य करीत नाहीत: जेव्हा आपण त्यांच्याकडे एखादा ट्रॅक स्थानांतरित करता तेव्हा आपण स्नॅप-टू लेबले वापरू शकता आणि आपण निवड प्रदेश ड्रॅग केल्यास त्यावरील क्षेत्राचा प्रारंभ बिंदू आपण ड्रॅग करताना सिलेक्शन टूलबारमध्ये टाइम लाइन गतिकरित्या दर्शविली जाते.
