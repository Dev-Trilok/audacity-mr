4 4 Note Tracks contain data from MIDI (and Allegro) files - these can be imported with the File > Import > MIDI... command. टीप ट्रॅकमध्ये एमआयडीआय (आणि अल्लेग्रो) फायलींमधील डेटा असतो - हे फाइल> आयात> एमआयडीआय ... आदेशासह आयात केले जाऊ शकतात.
5 5 Note Tracks can be useful to visually compare a given MIDI (or Allegro) file with a sampled audio file (such as WAV) of the same music. टीप त्याच संगीताच्या नमुना असलेल्या ऑडिओ फाईल (जसे की डब्ल्यूएव्ही) सह दिलेल्या एमआयडीआय (किंवा अ‍ॅलेग्रो) फाईलची दृष्टि तुलना करण्यास ट्रॅक उपयुक्त ठरू शकतात.
6 6 They can be cut-and-paste edited. ते संपादित आणि कट-पेस्ट केले जाऊ शकतात.
7 7 They can then be exported using the File > Export > Export MIDI... command. When exporting so as to overwrite an existing GRO, MID or MIDI file, the original file is renamed with ".bak" suffix, for example "keyboards.mid.bak". त्यानंतर ते फाइल> निर्यात> निर्यात एमआयडीआय ... आदेशाचा वापर करुन निर्यात करता येऊ शकते. अस्तित्वातील जीआरओ, एमआयडी किंवा एमआयडीआय फाईल अधिलिखित करण्यासाठी निर्यात करताना मूळ फाइलचे नाव ".bak" प्रत्यय सह बदलले जाते, उदाहरणार्थ "कीबोर्ड. मिड.बाक".

18 18 A Note Track एक नोट ट्रॅक
19 19 Note tracks can be created by importing MIDI (or Allegro) files with File > Import > MIDI.... फाइल> आयात> एमआयडीआय सह एमआयडीआय (किंवा द्रुतगतीने) फायली आयात करुन टीप ट्रॅक तयार केले जाऊ शकतात.
20 20 Note carefully that when a MIDI or Allegro is imported you are unlikely to see all of the notes that are in the track with the initial view restricted to two octaves centered around the average note value. काळजीपूर्वक लक्षात घ्या की जेव्हा एखादी एमआयडीआय किंवा अल्लेग्रो आयात केली जाते तेव्हा आपणास सुरुवातीच्या दृश्यासह ट्रॅकमध्ये असलेल्या सर्व नोट्स सरासरी नोट मूल्याच्या भोवती केंद्रित केलेल्या दोन अष्टकांपर्यंत प्रतिबंधित नसतात.
21 21 Observe carefully the little black marks at the bottom right of the image. These indicate that there are additional notes in the Note Track below them. प्रतिमेच्या तळाशी उजवीकडे असलेल्या काळ्या रंगाच्या चिन्हे काळजीपूर्वक पहा. हे सूचित करते की त्यांच्याकडे असलेल्या टीप ट्रॅकमध्ये अतिरिक्त नोट्स आहेत.
22 22 More notes can me made visible by using Vertical Zooming. व्हर्टिकल झूमिंगद्वारे अधिक नोट्स मला दृश्यमान केल्या जाऊ शकतात.
23 23 Playing and editing Note tracks टीप ट्रॅक प्ले करणे आणि संपादित करणे
24 24 MIDI and Allegro playback मिडी आणि अ‍ॅलेग्रो प्लेबॅक
25 25 Note tracks containing MIDI data (or Allegro data) can now be played in Audacity. एमआयडीआय डेटा (किंवा अल्लेरो डेटा) असलेले टीप ट्रॅक आता ऑडॅसिटीमध्ये प्ले केले जाऊ शकतात.
26 26 Loop Play and Play Cut Preview also work with Note tracks. लूप प्ले आणि प्ले कट पूर्वावलोकन टीप ट्रॅकसह देखील कार्य करते.
27 27 But note that there will no use of the Playback meter while Note tracks are played. परंतु लक्षात घ्या की नोट ट्रॅक प्ले होत असताना प्लेबॅक मीटरचा वापर होणार नाही.
28 28 In order to achieve MIDI playback on Mac and Linux you will need to install, and run, additional software, a software synthesizer मॅक आणि लिनक्सवर एमआयडीआय प्लेबॅक साध्य करण्यासाठी आपल्याला अतिरिक्त सॉफ्टवेअर, एक सॉफ्टवेयर सिंथेसाइजर स्थापित करणे आणि चालविणे आवश्यक आहे.

32 32 Editing Note Tracks संपादन टीप ट्रॅक
33 33 Only limited editing can be performed on Note tracks such as cutting, pasting and deleting. केवळ कटिंग, पेस्ट करणे आणि हटविणे यासारख्या टीप ट्रॅकवर मर्यादित संपादन केले जाऊ शकते.
34 34 Track Control Panel ट्रॅक नियंत्रण पॅनेल
35 35 The Track Control Panel at the left of the note track has controls for this track. Left-clicking other than on a control performs track selection and move as follows. टीप ट्रॅकच्या डावीकडील ट्रॅक नियंत्रण पॅनेलमध्ये या ट्रॅकसाठी नियंत्रणे आहेत. नियंत्रणाशिवाय अन्य डावे क्लिक ट्रॅक निवड करते आणि खालीलप्रमाणे हलते.
36 36 Clicking selects all this track's audio, deselecting all other tracks. इतर सर्व ट्रॅकची निवड रद्द करून या क्लिकचा ऑडिओ निवडतो.
37 37 Holding Shift while clicking selects all this track's audio, keeping already selected tracks selected. आधीपासून निवडलेले ट्रॅक निवडलेले ठेवून या सर्व ट्रॅकचा ऑडिओ निवडताना क्लिक करताना शिफ्ट होल्डिंग.
38 38 (keyboard: use Up or Down to give the track focus, then Enter to select it; hold Shift then use Up or Down to extend selection into another track. Read keyboard selection for more). (कीबोर्ड: ट्रॅकला फोकस देण्यासाठी वर किंवा खाली वापरा, नंतर ते निवडण्यासाठी एन्टर करा; शिफ्ट दाबून धरा आणि दुसर्‍या ट्रॅकवर निवड विस्तृत करण्यासाठी वर किंवा खाली वापरा. ​​अधिकसाठी कीबोर्ड निवड वाचा)
39 39 Click-and-drag moves this track up or down when there are other tracks (keyboard: choose one of the "Move Track" commands in Note Track Dropdown Menu, opened by Shift + M on the currently focused track ). जेव्हा इतर ट्रॅक असतात तेव्हा क्लिक आणि ड्रॅग हा ट्रॅक वर किंवा खाली हलवितो (कीबोर्ड: नोटिस ट्रॅक ड्रॉपडाउन मेनूमधील “ट्रॅक ट्रॅक” कमांडपैकी एक निवडा, सध्या फोकस केलेल्या ट्रॅकवर शिफ्ट + एम द्वारे उघडलेला)
40 40 Controls नियंत्रणे
41 41 Close Button: बटण बंद करा:
42 42 Clicking this button will close the track, removing it from the project. या बटणावर क्लिक केल्याने हा प्रकल्पातून तो काढून टाकला जाईल.
43 43 Note Track Dropdown Menu: नोट ट्रॅक ड्रॉपडाउन मेनू:
44 44 Clicking the track name or using Shift + M on the currently focused track opens the Note Track Dropdown Menu giving options that affect this track only. ट्रॅकचे नाव क्लिक करणे किंवा सध्या फोकस केलेल्या ट्रॅकवर Shift + M वापरणे केवळ या ट्रॅकवर परिणाम करणारे टीप ट्रॅक ड्रॉपडाउन मेनू देते.
45 45 Newly created tracks are named "Note Track". नव्याने तयार केलेल्या ट्रॅकचे नाव "टीप ट्रॅक" आहे.
46 46 Mute Button: बटण नि: शब्द करा
47 47 Click to silence this track when playing. By default, the track will only be silenced if no Solo buttons are depressed. Click again to unsilence. खेळताना हा ट्रॅक शांत करण्यासाठी क्लिक करा. डीफॉल्टनुसार, कोणतीही सोलो बटणे उदासीन नसल्यास केवळ ट्रॅक शांत केला जाईल. पुन्हा कधीही क्लिक करा
48 48 Tracks on which the Mute button is pressed are never exported. ज्या ट्रॅकवर निःशब्द बटण दाबले जाते ते कधीही निर्यात केले जात नाहीत.
49 49 Solo Button: एकल बटण:
50 50 Click to play just this track. फक्त हा ट्रॅक खेळण्यासाठी क्लिक करा.
51 51 Click again to release the button. बटण सोडण्यासाठी पुन्हा क्लिक करा.
52 52 Whenever playing audio, Solo takes precedence over Mute by default - the Mute buttons have no effect whilst any Solo button is down. जेव्हा जेव्हा ऑडिओ प्ले होतो तेव्हा डीफॉल्टनुसार एकल नि: शब्द करण्यापेक्षा सोलो अग्रक्रम घेते - कोणतेही एकल बटण खाली नसल्यास निःशब्द बटणांवर कोणताही प्रभाव पडत नाही.
53 53 There are two other options for Solo button behavior in the Tracks Preferences. ट्रॅक पसंतींमध्ये सोलो बटणाच्या वर्तनासाठी इतर दोन पर्याय आहेत.
54 54 One of these is to remove the Solo button, leaving only a Mute button. यापैकी फक्त एक निःशब्द बटण सोडून सोलो बटण काढणे आहे.
55 55 Channel toggles: चॅनेल टॉगल:
56 56 Shows enabled channels, color coded to match the actual notes in the channels. Individual channels may be toggled by left-clicking, or all other channels can be muted/unmuted by right-clicking. चॅनेलमधील वास्तविक नोट्सशी जुळण्यासाठी सक्षम केलेले चॅनेल, रंग कोड कोड दर्शविते. डावी-क्लिक करून वैयक्तिक चॅनेल टॉगल केले जाऊ शकतात किंवा अन्य सर्व चॅनेल उजवे-क्लिक करून नि: शब्द / सशब्द केली जाऊ शकतात.
57 57 Channel numbers start at 1; channel 10 reserved for percussion by the General MIDI specification. चॅनेल क्रमांक 1 वाजता प्रारंभ होतात; जनरल एमआयडीआय स्पेसिफिकेशनद्वारे चक्र 10 आरक्षित करण्यासाठी राखीव आहे.
58 58 Velocity Slider: वेग स्लाइडर:
59 59 Set the velocity offset for this track. Hold down the shift key while dragging to adjust the slider in finer increments. Double-click the slider to bring up a window where you can make precise adjustments or enter a velocity value. या ट्रॅकसाठी वेग ऑफसेट सेट करा. बारीक वाढीमध्ये स्लायडर समायोजित करण्यासाठी ड्रॅग करताना शिफ्ट की दाबून ठेवा. विंडो आणण्यासाठी स्लाइडरवर डबल-क्लिक करा जिथे आपण अचूक समायोजन करू शकता किंवा वेग मूल्य प्रविष्ट करू शकता.
60 60 Velocity is in units of MIDI velocity, ranging from 1 (lowest) to 127 (highest). वेग एमआयडीआय वेगाच्या युनिटमध्ये आहे, ते 1 (सर्वात कमी) ते 127 (सर्वोच्च) पर्यंत आहेत.
61 61 The offset from this slider (ranging from -50 to +50) is combined with each note's individual velocity to produce the resultant velocity. या स्लाइडरमधील ऑफसेट (-50 ते +50 पर्यंतचे) परिणामी वेग तयार करण्यासाठी प्रत्येक नोटच्या वैयक्तिक वेगळ्यासह एकत्र केले जाते.
62 62 Track Collapse Button: ट्रॅक संकुचित बटण:
63 63 Click on this to make the track 'fold up' into a smaller size. ट्रॅकला लहान आकारात 'फोल्ड अप' करण्यासाठी या क्लिक करा.
64 64 Click again, or drag the lower edge of the track to restore the size. पुन्हा क्लिक करा किंवा आकार पुनर्संचयित करण्यासाठी ट्रॅकच्या खालच्या काठावर ड्रॅग करा.
65 65 Status Indicators स्थिती निर्देशक
66 66 Sync-Lock Indicator: संकालन-लॉक सूचक:
67 67 When present (at bottom-right of the Track Control Panel, adjacent to the Track Collapse button), indicates that this track is part of a Sync-Locked Track Group. उपस्थित असल्यास (ट्रॅक संक्षिप्त करा बटणालगत ट्रॅक नियंत्रण पॅनेलच्या उजवीकडे) दर्शवितो की हा ट्रॅक समक्रमित-लॉक केलेला ट्रॅक गटाचा भाग आहे.
68 68 Vertical Scale अनुलंब स्केल
69 69 The vertical scale displays note pitches like a piano's keys, ranging from the lowest to the highest possible MIDI notes (C -1 to G 9). अनुलंब प्रमाणात पियानोच्या किज सारख्या नोटांचे खेळपट्ट्या दर्शवतात, ज्यामध्ये कमीतकमी सर्वात कमी एमआयडीआय नोट्स (सी -1 ते जी 9 पर्यंत) असतात.
70 70 Each octave is labeled at C (and the notes above are in that octave). प्रत्येक अष्टक सी वर लेबल केलेले आहे (आणि वरील नोट्स त्या अष्टकातील आहेत)
71 71 The Vertical Scale can be used to control Vertical Zooming. व्हर्टिकल स्केलचा वापर व्हर्टिकल झूमिंग नियंत्रित करण्यासाठी केला जाऊ शकतो.
72 72 Observe carefully the little black marks above and below the colored notes. These indicate that there are additional notes in the Note Track above and below them. रंगीत नोटांच्या वर आणि खाली थोडे काळ्या खुणा काळजीपूर्वक पहा. हे सूचित करते की त्यांच्याकडे व त्याखालील टीप ट्रॅकमध्ये अतिरिक्त नोट्स आहेत.
73 73 More notes can me made visible by using Vertical Zooming. व्हर्टिकल झूमिंगद्वारे अधिक नोट्स मला दृश्यमान केल्या जाऊ शकतात.
74 74 Note Track Dropdown Menu नोंद ड्रॉपडाउन मेनू
75 75 Clicking the note track's name by the downward-pointing triangle (or using shortcut Shift + M or the keyboard Menu key when the note track has focus) opens the Note Track Dropdown Menu: डाउन-पॉइंटिंग त्रिकोणाद्वारे नोट ट्रॅकचे नाव क्लिक करणे (किंवा शॉर्टकट शिफ्ट + एम किंवा नोट ट्रॅकवर लक्ष केंद्रित करतेवेळी कीबोर्ड मेनू की वापरुन) नोट ट्रॅक ड्रॉपडाउन मेनू उघडेल:
76 76 Name नाव
77 77 Displays the "Track Name" dialog where you can give the track a new name. Useful in multi-track projects to provide a visual indication of the content of each track. "ट्रॅक नेम" संवाद प्रदर्शित करते जेथे आपण ट्रॅकला एक नवीन नाव देऊ शकता. प्रत्येक ट्रॅकच्या सामग्रीचे व्हिज्युअल संकेत देण्यासाठी मल्टी-ट्रॅक प्रकल्पांमध्ये उपयुक्त.
78 78 Up Octave and Down Octave अप ऑक्टोव्ह आणि डाऊन ऑक्टाव्ह
79 79 Moves the vertical scale up or down one octave. अनुलंब स्केल एका अष्टकोनी वर किंवा खाली हलवते.
80 80 Moving Tracks ट्रॅक हलवित आहे
81 81 Tracks can be moved up and down using the menu items. मेनू आयटम वापरून ट्रॅक वर आणि खाली हलविले जाऊ शकतात.
82 82 Move Track Up: Moves track up. ट्रॅक अप हलवा: ट्रॅक अप हलवते.
83 83 Move Track Down: Moves track down. ट्रॅक डाउन हलवा: ट्रॅक डाउन हलवते.
84 84 Move Track to Top: Moves track to become the topmost track in the project. ट्रॅक वर जा: प्रोजेक्ट मधील सर्वात वरचा ट्रॅक होण्यासाठी ट्रॅक हलवेल.
85 85 Move Track to Bottom: Moves track to become the bottom track in the project. ट्रॅक तळाशी हलवा: प्रकल्पातील तळाशी ट्रॅक होण्यासाठी ट्रॅक हलवते.
86 86 Note tracks can also be moved up or down by clicking between the controls in the track's Track Control Panel then dragging upwards or downwards. ट्रॅकच्या ट्रॅक कंट्रोल पॅनेलमधील नियंत्रणे दरम्यान वर किंवा खाली ड्रॅग करून नोट ट्रॅक वर किंवा खाली हलविले जाऊ शकतात.
