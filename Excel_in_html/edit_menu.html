4 4 The Edit Menu provides standard edit commands (Undo, Redo, Cut, Copy, Paste and Delete) plus many other commands specific to editing audio or labels. There are many commands available from this menu so some commands are grouped into submenus. संपादन मेनू मानक संपादन आदेश (पूर्ववत करा, पुन्हा करा, कट करा, कॉपी करा, पेस्ट करा आणि हटवा) तसेच ऑडिओ किंवा लेबले संपादित करण्यासाठी विशिष्ट इतर आदेश प्रदान करतात. या मेन्यू वरुन बर्‍याच कमांड उपलब्ध आहेत त्यामुळे काही कमांडस सबमेनसमध्ये ग्रुप केल्या आहेत.
5 5 Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
6 6 Mac users: Ctrl = ⌘ and Alt = Option. So, for example, Ctrl + Alt + K = ⌘ + Option + K मॅक वापरकर्ते: Ctrl = ⌘ आणि Alt = पर्याय. तर, उदाहरणार्थ, Ctrl + Alt + K = ⌘ + पर्याय + के
7 7 Undo  Ctrl + Z Ctrl + Z पूर्ववत करा
8 8 Undoes the last editing operation you performed to your project. आपण आपल्या प्रकल्पासाठी केलेले अंतिम संपादन ऑपरेशन पूर्ववत करते.
9 9 You can undo as many times as you want, all the way back to when you opened the window. आपण आपल्याला पाहिजे तितक्या वेळा पूर्ववत करू शकता, विंडो उघडल्यावर परत परत येऊ शकता.
10 10 To undo many operations, select History... from the View Menu. The name of this menu item will change to reflect what will be undone; if you just recorded some audio, the menu item name will be Undo Record. बर्‍याच ऑपरेशन्स पूर्ववत करण्यासाठी व्ह्यू मेनू वरून इतिहास ... निवडा. या मेनू आयटमचे नाव काय पूर्ववत केले जाईल हे प्रतिबिंबित करण्यासाठी बदलेल; आपण आत्ताच काही ऑडिओ रेकॉर्ड केल्यास, मेनू आयटमचे नाव पूर्ववत रेकॉर्ड होईल.
11 11 Redo  Ctrl + Y पुन्हा करा Ctrl + Y
12 12 Redoes any editing operations that were just undone. नुकतीच पूर्ववत केलेली कोणतीही संपादन ऑपरेशन्स पुन्हा करा.
13 13 After you perform a new editing operation, you can no longer redo the operations that were undone. आपण नवीन संपादन ऑपरेशन केल्यानंतर आपण यापुढे पूर्ववत केलेल्या ऑपरेशन्स पुन्हा करू शकत नाही.
14 14 Cut  Ctrl + X Ctrl + X कट करा
15 15 Removes the selected audio data and/or labels and places these on the Audacity clipboard. Audio or labels after the selection move leftwards. निवडलेला ऑडिओ डेटा आणि / किंवा लेबले काढून ऑडसिटी क्लिपबोर्डवर ठेवतो. निवडीनंतर ऑडिओ किंवा लेबल डावीकडे हलवा.
16 16 Delete  Ctrl + K Ctrl + K हटवा
17 17 Similar to Cut, but removes the audio data and/or labels that are currently selected without copying them to the Audacity clipboard. कट प्रमाणेच, परंतु ऑडिओ डेटा आणि / किंवा लेबले काढून टाकतात जे ऑडीसिटी क्लिपबोर्डवर कॉपी केल्याशिवाय सध्या निवडलेले आहेत.
18 18 Copy  Ctrl + C Ctrl + C कॉपी करा
19 19 Copies the selected audio data to the Audacity clipboard without removing it from the project. निवडलेला ऑडिओ डेटा प्रकल्पातून न काढता ऑडसिटी क्लिपबोर्डवर कॉपी करतो.
20 20 Paste  Ctrl + V Ctrl + V पेस्ट करा
21 21 Pastes audio which has been cut or copied to the Audacity clipboard, either inserting it into the selected track(s) at the cursor point, or replacing the current selection region(s). ऑडीसिटी क्लिपबोर्डवर कापला किंवा कॉपी केलेला ऑडिओ पेस्ट करा, एकतर कर्सर बिंदूवर निवडलेल्या ट्रॅकमध्ये यामध्ये समाविष्ट करुन किंवा विद्यमान निवड प्रदेश (चे) पुनर्स्थित करा.
22 22 Duplicate  Ctrl + D डुप्लिकेट सीटीआरएल + डी
23 23 Creates a new track containing only the current selection as a new clip. नवीन क्लिप म्हणून फक्त सद्य निवडीचा एक नवीन ट्रॅक तयार करते.
24 24 Remove Special विशेष काढा
25 25 The Remove Special submenu has commands for more "advanced" removal of audio and labels: हटवा विशेष सबमेनूमध्ये ऑडिओ आणि लेबले अधिक "प्रगत" काढण्याची आज्ञा आहेत:
26 26 Split Cut and Split Delete are a "special" cut or delete which make the audio or labels to right of the selection stay in current position instead of moving leftwards. स्प्लिट कट आणि स्प्लिट डिलीट हा एक "स्पेशल" कट किंवा डिलीट आहे जो ऑडिओ किंवा लेबलेला डावीकडे हलविण्याऐवजी निवडीच्या उजवीकडे ठेवतो.
27 27 Silence Audio replaces the current selection with silence. मौन ऑडिओ मौजूदा निवडीची जागा शांततेने घेते.
28 28 Trim Audio removes all audio from the current clip except the selected part, creating its own clip from the remaining, selected, audio. ट्रिम ऑडिओ निवडलेल्या भागाशिवाय वर्तमान क्लिपमधून सर्व ऑडिओ काढते, उर्वरित, निवडलेले, ऑडिओमधून स्वतःची क्लिप तयार करते.
29 29 Clip Boundaries क्लिप सीमा
30 30 The Clip Boundaries submenu has commands to create or remove separate clips in the audio track. A clip inside an audio track is a separate section of that track which has been split so that it can be manipulated somewhat independently of the other clips in the track. क्लिप बाउंडरीज सबमेनूमध्ये ऑडिओ ट्रॅकमध्ये स्वतंत्र क्लिप तयार करण्यासाठी किंवा काढण्यासाठी आज्ञा आहेत. ऑडिओ ट्रॅकच्या आत असलेली क्लिप त्या ट्रॅकचा वेगळा विभाग आहे जो विभाजित झाला आहे जेणेकरून ट्रॅकमधील इतर क्लिपच्या तुलनेत हे काहीसे स्वतंत्रपणे हाताळू शकते.
31 31 Labels लेबले
32 32 The Labels submenu has commands that enable you to add and edit labels. लेबले सबमेनूमध्ये अशी आज्ञा आहेत जी आपल्याला लेबले जोडण्यास आणि संपादित करण्यास सक्षम करतात.
33 33 This command invokes the Labels Editor which lets you add or remove Label Tracks and edit their labels entirely using the keyboard, so is particularly useful for visually impaired users. ही आज्ञा लेबल एडिटरची विनंती करते जी आपल्याला कीबोर्ड वापरुन लेबल ट्रॅक जोडू किंवा काढू देते आणि त्यांची लेबले पूर्णपणे संपादित करू देते, जेणेकरून दृष्टिहीन वापरकर्त्यांसाठी उपयुक्त आहे.
34 34 It brings up a dialog box showing all of your labels in a keyboard-accessible tabular view. Handy buttons in the dialog let you insert or delete a label, or import and export labels to a file. See Labels Editor for more details. कीबोर्ड-प्रवेशयोग्य टॅब्यूलर व्ह्यूमध्ये आपले सर्व लेबल दर्शविणारा एक संवाद बॉक्स आणतो. संवादातील सुलभ बटणे आपल्‍याला लेबल घालू किंवा हटवू देतील किंवा फायलीवर लेबल आयात आणि निर्यात करु देतील. अधिक माहितीसाठी लेबल संपादक पहा.
35 35 See Edit Menu: Labels for details of this submenu. संपादन मेनू पहा: या उपमेनूच्या तपशीलांसाठी लेबले.
36 36 Labeled Audio लेबल केलेला ऑडिओ
37 37 The Labeled Audio submenu has commands that offer a time saving way of performing Edit Menu operations on the audio of multiple fully selected range labels. लेबल केलेले ऑडिओ सबमेनूमध्ये असे आदेश आहेत जे एकाधिक पूर्णपणे निवडलेल्या श्रेणी लेबलांच्या ऑडिओवर संपादन मेनू ऑपरेशन्सची वेळ बचत मार्ग प्रदान करतात.
38 38 The commands apply to all labeled audio regions that are fully inside a selection drawn in a label track. The selection may extend beyond the label boundaries, but audio that is not labeled and audio whose region label is only partly within the selection will not be acted on. लेबल ट्रॅकमध्ये काढलेल्या निवडीच्या आत असलेल्या सर्व लेबल ऑडिओ प्रदेशांवर आदेश लागू होतात. निवड लेबलच्या सीमांच्या पलीकडे वाढू शकते परंतु लेबल न केलेले ऑडिओ आणि ज्याचे प्रदेश लेबल केवळ अंशतः निवडीमध्ये असेल त्या ऑडिओवर कारवाई केली जाणार नाही.
39 39 If none of the audio tracks are included in the selection, the Labeled Audio commands apply to all audio tracks in the project. However if you include only certain audio tracks in the selection, the Labeled Audio commands will only affect those selected audio tracks. कोणत्याही ऑडिओ ट्रॅक निवडीमध्ये समाविष्ट न केल्यास, लेबल केलेले ऑडिओ आदेश प्रकल्पातील सर्व ऑडिओ ट्रॅकवर लागू होतात. तथापि आपण निवडीमध्ये काही विशिष्ट ऑडिओ ट्रॅक समाविष्ट करत असल्यास, लेबल केलेले ऑडिओ आदेश केवळ त्या निवडलेल्या ऑडिओ ट्रॅकवर परिणाम करतील.
40 40 See Edit Menu: Labeled Audio for details of this submenu. या उपमेनूच्या तपशीलांसाठी संपादन मेनू: लेबल केलेला ऑडिओ पहा.
41 41 Metadata... मेटाडेटा ...
42 42 Use this function to edit the metadata tags that will be applied to exported files. निर्यात केलेल्या फायलींवर लागू होणारे मेटाडेटा टॅग संपादित करण्यासाठी हे कार्य वापरा.
43 43 Preferences...  Ctrl + P प्राधान्ये ... Ctrl + P
44 44 Displays the Preferences dialog. Preferences enable you change most of the default behaviors and settings of Audacity. प्राधान्ये संवाद दाखवतो. प्राधान्ये आपल्याला बहुतेक डीफॉल्ट वर्तन आणि ऑडसिटीची सेटिंग्ज बदलण्यास सक्षम करतात.
