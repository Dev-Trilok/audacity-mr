4 4 Nyquist Prompt lets you run and debug code snippets for your own plug-ins created using the Nyquist programming language. See the Manual's Nyquist page for more information about writing Nyquist code. Nyquist प्रॉम्प्ट आपल्याला Nyquist प्रोग्रामिंग भाषेचा वापर करुन तयार केलेल्या आपल्या स्वतःच्या प्लगइनसाठी कोड स्निपेट चालवू आणि डीबग करू देते. Nyquist कोड लिहिण्यासाठी अधिक माहितीसाठी महितीपुस्तिका चे Nyquist पृष्ठ पहा.
5 5 See also the Audacity wiki page on Nyquist Nyquist वर ऑडिटी विकी पृष्ठ देखील पहा
6 6 To use Nyquist Prompt, first select some audio. If the code you want to test generates audio, you can avoid the initial steps of preparing an audio selection by installing the Nyquist Generate Prompt plug-in instead. Nyquist प्रॉमप्ट वापरण्यासाठी, प्रथम काही ध्वनी निवडा. आपण चाचणी घेऊ इच्छित कोड ध्वनी व्युत्पन्न करत असल्यास आपण त्याऐवजी Nyquist जनरेट प्रॉम्प्ट प्लग-इन स्थापित करुन ध्वनी निवड तयार करण्याच्या प्रारंभिक चरणांना टाळू शकता.
7 7 Accessed by: Tools > Nyquist Prompt... याद्वारे प्रवेश केलेलेः साधने> न्यूक्विस्ट प्रॉमप्ट ...
8 8 Enter Nyquist Command Nyquist आज्ञा प्रविष्ट करा
9 9 Type the command as required. Enter on the keyboard moves the cursor to a new line. The last entered code is stored in the pluginsettings.cfg file in Audacity's folder for application data so is retained after exiting Audacity. आवश्यकतेनुसार आज्ञा टाइप करा. कीबोर्डवर एंटर केल्यामुळे कर्सर एका नवीन ओळीवर जाईल. अंतिम डेटा प्रविष्ट केलेला डेटा डेटासाठी ओड्यासिटीच्या फोल्डरमध्ये प्लगइनसमायोजन सीएफजी धारिकामध्ये संचयित केलेला आहे म्हणून ओड्यासिटी बाहेर आल्यानंतर ठेवला जातो.
10 10 Plugin GUI can be tested if you include complete Nyquist Plug-in Headers. The Nyquist Prompt interprets header comments to produce a plug-in of the defined type on the fly. By आपण पूर्ण Nyquist प्लग-इन शीर्षलेख समाविष्ट केल्यास प्लगइन GUI चाचणी केली जाऊ शकते. फ्लाय वर परिभाषित प्रकाराचे प्लग-इन तयार करण्यासाठी Nyquist प्रॉम्प्ट शीर्षलेख टिप्पण्यांचे अर्थ लावून देते. द्वारा
11 11 default the plug-in is a process type, though other type may be created by including the appropriate header. पूर्वनिर्धारित प्लग-इन एक प्रक्रिया प्रकार आहे, जरी अन्य प्रकार योग्य शीर्षलेख समाविष्ट करुन तयार केला जाऊ शकतो.
12 12 Use legacy (version 3) syntax लेगसी (आवृत्ती 3) वाक्यरचना वापरा
13 13 When selected, commands entered in the main text window will be interpreted as the "version 3" code. निवडल्यास, मुख्य मजकूर विंडोमध्ये प्रविष्ट केलेल्या आज्ञा्सचे वर्णन "आवृत्ती 3" कोड म्हणून केले जाईल.
14 14 When not selected (default), commands entered in the main text window will be interpreted as "version 4" code. निवडलेले नसल्यास (पूर्वनिर्धारित), मुख्य मजकूर विंडोमध्ये प्रविष्ट केलेल्या आज्ञा्सचे वर्णन "आवृत्ती 4" कोड म्हणून केले जाईल.
15 15 Version 4 code includes a number of variables that are not available in version 3. In particular, the S variable is no longer used to pass sound from Audacity to Nyquist. In version 4 the S variable has been replaced by *TRACK*. Full details about the difference between version 3 and version 4 code can be found in the Audacity wiki. आवृत्ती code कोडमध्ये असंख्य व्हेरिएबल्स समाविष्ट आहेत जे आवृत्ती in मध्ये उपलब्ध नाहीत. विशेषतः, एस व्हेरिएबल यापुढे ओड्यासिटी ते नायक्विस्टपर्यंत ध्वनी पास करण्यासाठी वापरला जात नाही. आवृत्ती 4 मध्ये एस व्हेरिएबल * गीतपट्टा * ने बदलला आहे. आवृत्ती 3 आणि आवृत्ती 4 कोडमधील भिन्नतेबद्दल पूर्ण तपशील ओड्यासिटी विकीमध्ये आढळू शकतो.
16 16 Buttons बटणे
17 17 Load Load a Nyquist plugin or script file saved with .ny extension. लोड करा एक Nyquist प्लगइन लोड करा किंवा .ny विस्तारासह जतन केलेली स्क्रिप्ट धारिका.
18 18 Save Save the current contents of the editor window as an .ny file. जतन एडिटर विंडोची सद्य सामग्री एक .ny धारिका म्हणून जतन करा.
19 19 Manage Opens a dropdown menu where you can save the window contents as a preset then reload those contents व्यवस्थापित करा ड्रॉपडाउन मेनू उघडतो जिथे आपण प्रीसेट म्हणून विंडोमधील सामग्री जतन करू शकता आणि नंतर त्या सामग्री पुन्हा लोड करा
20 20 by selecting the preset. You can also see some descriptive text about the plugin. For details see Manage presets. प्रीसेट निवडून. आपण प्लगइन विषयी काही वर्णनात्मक मजकूर देखील पाहू शकता. तपशीलांसाठी प्रीसेट्स व्यवस्थापित करा पहा.
21 21 Plugin authors should use this button to test if a Preview button in their plug-in would accurately preview the effect as applied to the waveform. Preview should not be used in effects that change over time (such as a fade effect) because Audacity's preview as currently implemented only applies the effect to the length of the previewed selection. प्लगइन लेखकांनी हे बटण त्यांच्या प्लग-इन मधील पूर्वावलोकन बटणावर वेव्हफॉर्मवर लागू केलेल्या प्रभावाचे अचूक पुनरावलोकन केले असेल तर हे तपासण्यासाठी वापरावे. कालांतराने बदलणार्‍या प्रभावांमध्ये पूर्वावलोकनाचा वापर केला जाऊ नये (जसे की फिकट प्रभाव) कारण ओड्यासिटीचे पूर्वावलोकन केवळ अंमलबजावणीच्या निवडीच्या लांबीवरच लागू होते.
22 22 Debug The Debug button works as it does in all Nyquist effects. Clicking this button shows a progress dialog for processing the code, followed by an uneditable "Nyquist Output" window listing any errors or other output. Click OK to close the output window and process the command, or use Enter on the keyboard. Hold Ctrl on Windows or ⌘ on Mac before pressing Enter. डीबग डीबग बटण सर्व Nyquist प्रभावांप्रमाणेच कार्य करते. या बटणावर क्लिक केल्याने कोडवर प्रक्रिया करण्यासाठी प्रगती संवाद दर्शविला जातो, त्यानंतर कोणतीही त्रुटी किंवा इतर आउटपुट सूचीबद्ध न करता येणारी "न्यक्युइस्ट आउटपुट" विंडो येते. आउटपुट विंडो बंद करण्यासाठी आज्ञावर ओके क्लिक करा किंवा कीबोर्डवरील एंटर वापरा. एंटर दाबण्यापूर्वी विंडोजवर किंवा tr मॅकवर सीटीआरएल दाबून ठेवा.
23 23 OK Applies the code to the waveform selection without debug output. Your entered code will be retained next time you open Nyquist Prompt in the Audacity session. Using the keyboard, hold Ctrl on Windows (or ⌘ on Mac) then press Enter. ओके डीबग आउटपुटशिवाय वेव्हफॉर्म निवडीसाठी कोड लागू करते. पुढच्या वेळी आपण ओड्यासिटी सत्रामध्ये Nyquist प्रॉम्प्ट उघडता तेव्हा आपला प्रविष्ट केलेला कोड कायम ठेवला जाईल कीबोर्ड वापरुन, Windows वर Ctrl दाबून ठेवा (किंवा Mac मॅक वर) त्यानंतर एंटर दाबा.
24 24 Cancel रद्द करा
25 25 No code is processed or debugged. Any code you entered after opening the window this time will be discarded. कोणत्याही कोडवर प्रक्रिया किंवा डीबग केले जात नाही. यावेळी आपण विंडो उघडल्यानंतर प्रविष्ट केलेला कोणताही कोड टाकून दिला जाईल.
26 26 The Nyquist prompt supports both LISP syntax and SAL syntax. If the code you enter cannot be recognized as valid Nyquist code in either LISP or SAL syntax, an error message will appear with a hint about making a correction. Nyquist प्रॉमप्ट दोन्ही LISP वाक्यरचना आणि SAL वाक्यरचना समर्थन देते. आपण प्रविष्ट केलेला कोड एलआयएसपी किंवा एसएएल सिंटॅक्समध्ये वैध Nyquist कोड म्हणून ओळखला जाऊ शकत नसल्यास, दुरुस्ती करण्याच्या इशारासह एक त्रुटी संदेश दिसेल.
