9 9 Persistence of changes made here येथे केलेले बदल चिकाटी
10 10 Changes you make when you press the OK button only persist for that track while the project window is open. This is the case even if you save a project. Use Spectrograms Preferences instead to make permanent changes to the default Spectrogram settings with which a new Spectrogram track will open. आपण ओके बटण दाबता तेव्हा आपण केलेले बदल केवळ प्रोजेक्ट विंडो उघडे असताना त्या ट्रॅकसाठी कायम राहतात. आपण एखादा प्रकल्प वाचविला तरीही हेच प्रकरण आहे. त्याऐवजी नवीन स्पेक्ट्रोग्राम ट्रॅक उघडेल त्या डीफॉल्ट स्पेक्ट्रोग्राम सेटिंग्जमध्ये कायम बदल करण्यासाठी स्पेक्ट्रोग्राम प्राधान्ये वापरा.

13 13 Use Preferences प्राधान्ये वापरा
14 14 The Spectrogram Settings dialog defaults to the "Use Preferences" checkbox being enabled, so on first opening the settings for a track the settings will be as already set in Spectrograms Preferences. Changing a setting in Spectrogram Settings will automatically disable the "Use Preferences" checkbox. Re-enabling the "Use Preferences" checkbox changes the settings in the dialog back to as they are in Spectrograms Preferences. स्पेक्ट्रोग्राम सेटिंग्ज संवाद "वापरा प्राधान्ये" चेकबॉक्स सक्षम करण्यावर डीफॉल्ट झाला आहे, म्हणून एखाद्या ट्रॅकसाठी सेटिंग्ज उघडण्यापूर्वी सेटिंग्ज स्पेक्ट्रोग्राम प्राधान्यांमध्ये आधीपासूनच सेट केल्या जातील. स्पेक्ट्रोग्राम सेटिंग्जमध्ये सेटिंग बदलल्याने स्वयंचलितपणे "वापरा प्राधान्ये" चेकबॉक्स अक्षम होईल. "प्राधान्ये वापरा" चेकबॉक्स पुन्हा सक्षम केल्याने डायलॉगमधील सेटिंग्ज स्पेक्ट्रोग्राम प्राधान्यांमध्ये असल्यामुळे त्या परत केल्या जातात.
15 15 Scale स्केल
16 16 Scale (in Spectrogram views): स्केल (स्पेक्ट्रोग्राम दृश्यांमध्ये):
17 17 Linear The linear vertical scale goes linearly from 0 kHz to 8 kHz frequency by default. रेखीय रेखीय अनुलंब स्केल डीफॉल्टनुसार 0 केएचझेड ते 8 केएचझेड फ्रिक्वेन्सीपासून रेषात्मकपणे जातो.
18 18 Logarithmic: This view is the same as the linear view except that the vertical scale is logarithmic. See Spectrogram View for a contrasting example of linear versus logarithmic spectrogram view. लोगारिथमिकः हे दृश्य रेखीय दृश्यासारखेच आहे परंतु त्याशिवाय अनुलंब स्केल लॉगरिथमिक आहे. रेखीय विरूद्ध लॉगॅरिथमिक स्पेक्ट्रोग्राम दृश्यासाठी विरोधाभासी उदाहरणासाठी स्पेक्ट्रोग्राम दृश्य पहा.
19 19 Mel: The name Mel comes from the word मेल: मेल नावाच्या शब्दावरून आला आहे
20 20 melody to indicate that the scale is based on pitch comparisons. See this Wikipedia page. स्केल खेळपट्टीच्या तुलनेत आधारित आहे हे दर्शविण्यासाठी चाल. हे विकिपीडिया पृष्ठ पहा.
21 21 Bark: This is a psychoacoustical scale based on subjective measurements of loudness. झाडाची साल: मोठ्याने जोर देण्याच्या व्यक्तिनिष्ठ मोजमापांवर आधारित हा एक मनोवैज्ञानिक स्तर आहे.
22 22 It is related to, but somewhat less popular than, the Mel scale. हे मेल स्केलशी संबंधित आहे, परंतु काहीसे कमी लोकप्रिय आहे.
23 23 See this Wikipedia page. हे विकिपीडिया पृष्ठ पहा.
24 24 ERB: The Equivalent Rectangular Bandwidth scale or ERB is a measure used in psychoacoustics, which gives an approximation to the bandwidths of the filters in human hearing. ईआरबीः समतुल्य आयताकृती बँडविड्थ स्केल किंवा ईआरबी मानसशास्त्रात वापरला जाणारा एक उपाय आहे, जो मानवी सुनावणीतील फिल्टरच्या बँडविड्थांना अंदाजे स्थान देतो.
25 25 It is implemented as a function ERBS(f) which returns the number of equivalent rectangular bandwidths below the given frequency "f". हे फंक्शन ईआरबीएस (एफ) म्हणून लागू केले गेले आहे जे दिलेली वारंवारता "एफ" च्या खाली समांतर आयताकृती बँडविड्थची संख्या परत करते.
26 26 See this Wikipedia page. हे विकिपीडिया पृष्ठ पहा.
27 27 Period: is the previously undocumented scale used by Pitch (EAC) view. पीरियडः पिच (ईएसी) दृश्याने वापरलेले पूर्वीचे Undocumented स्केल आहे.
28 28 It is for making the same displays of Pitch possible as in earlier versions of Audacity. हे ऑडॅसिटीच्या पूर्वीच्या आवृत्तींप्रमाणेच पिचचे समान प्रदर्शन शक्य करण्यासाठी आहे.
29 29 Minimum Frequency: This value corresponds to the bottom of the vertical scale in the spectrogram. Frequencies below this value will not be visible. The default value of "0" here will be treated as "1" when using "Spectrogram Logarithmic" view mode because a logarithmic scale cannot start at zero. किमान वारंवारता: हे मूल्य स्पेक्ट्रोग्राममधील अनुलंब स्केलच्या तळाशी संबंधित आहे. या मूल्याच्या खाली वारंवारता दृश्यमान होणार नाहीत. "स्पेक्ट्रोग्राम लोगारिथमिक" व्ह्यू मोडचा वापर करताना येथे "0" चे डीफॉल्ट मूल्य "1" म्हणून समजले जाईल कारण लॉगेरिथमिक स्केल शून्यावर प्रारंभ होऊ शकत नाही.
30 30 Maximum Frequency: This value corresponds to the top of the vertical scale. The value can be set to 100 Hz or any higher value. Irrespective of the entered value, the top of the scale will never exceed half the current sample rate of the track (for example, 22050 Hz if the track rate is 44100 Hz) because any given sample rate can only carry frequencies up to half that rate. A good use of this setting is in speech recognition or pitch extraction, where you can hide the visually unimportant highest frequencies and focus on the lower frequencies. जास्तीत जास्त वारंवारता: हे मूल्य अनुलंब स्केलच्या शीर्षाशी संबंधित आहे. मूल्य 100 हर्ट्ज किंवा कोणत्याही उच्च मूल्यावर सेट केले जाऊ शकते. प्रविष्‍ट केलेल्या मूल्याची पर्वा न करता, प्रमाणातील सर्वात वरचा भाग ट्रॅकच्या सद्य नमुना दरापेक्षा अर्धा कधीही ओलांडू शकणार नाही (उदाहरणार्थ, ट्रॅकचा दर 44100 हर्ट्ज असल्यास 22050 हर्ट्ज) कारण कोणताही नमुना दर त्या अर्ध्या दरापर्यंत वारंवारता बाळगू शकतो. . या सेटिंगचा चांगला वापर भाषण ओळखणे किंवा पिच एक्सट्रॅक्शनमध्ये आहे, जिथे आपण नेत्रहीन महत्त्वाचे नसलेले उच्च वारंवारता लपवू शकता आणि कमी वारंवारतेवर लक्ष केंद्रित करू शकता.
31 31 Colors रंग
32 32 Gain (dB): This enables you to increase / decrease the brightness of the display. गेन (डीबी): हे आपल्याला प्रदर्शनाची चमक वाढविण्यास / कमी करण्यास सक्षम करते.
33 33 For small signals where the display is mostly "blue" (dark) you can increase this value to see brighter colors and give more detail. लहान सिग्नलसाठी जिथे प्रदर्शन बहुतेक "निळे" (गडद) असते तेथे उजळ रंग पाहण्यासाठी आणि अधिक तपशील देण्यासाठी आपण हे मूल्य वाढवू शकता.
34 34 If the display has too much "white", decrease this value. जर डिस्प्लेमध्ये बरेच "पांढरे" असतील तर हे मूल्य कमी करा.
35 35 The default is 20dB and corresponds to a -20 dB signal at a particular frequency being displayed as "white". This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. डीफॉल्ट 20 डीबी आहे आणि -20 डीबी सिग्नलशी संबंधित आहे विशिष्ट आवृत्ति "व्हाइट" म्हणून प्रदर्शित केल्या जात आहे. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
36 36 Range (dB): Affects the range of signal sizes that will be displayed as colors. श्रेणी (डीबी): रंग म्हणून दर्शविल्या जाणार्‍या सिग्नल आकारांच्या श्रेणीवर परिणाम करते.
37 37 The default is 80 dB and means that you will not see anything for signals 80 dB below the value set for "Gain". This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. डीफॉल्ट d० डीबी आहे आणि याचा अर्थ असा आहे की आपण "गेन" साठी सेट केलेल्या मूल्याच्या खाली d० डीबी सिग्नलसाठी काहीही दिसणार नाही. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
38 38 Frequency Gain (dB/dec): A positive value here gives some extra gain to higher frequencies (above 1000 Hz), as they tend to be smaller and so cannot be seen as well. फ्रीक्वेंसी गेन (डीबी / डिक): इथले एक सकारात्मक मूल्य उच्च फ्रिक्वेन्सीना (1000 हर्ट्झपेक्षा जास्त) काही अतिरिक्त वाढ देते कारण ते कमी असतात आणि म्हणूनही पाहिले जाऊ शकत नाही.
39 39 You get less gain at lower frequencies as well. आपल्याला कमी फ्रिक्वेन्सीवर देखील कमी नफा मिळतो.
40 40 The default is 0 dB. This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. डीफॉल्ट 0 डीबी आहे. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
41 41 Grayscale: Shows gray shades in all spectrogram views instead of full color. ग्रेस्केल: संपूर्ण रंगाऐवजी सर्व स्पेक्ट्रोग्राम दृश्यांमध्ये करड्या छटा दाखवतात.
42 42 Algorithm अल्गोरिदम
43 43 Algorithm: अल्गोरिदम:
44 44 Frequencies (default): Audio frequency determines the pitch of a sound. फ्रिक्वेन्सी (डीफॉल्ट): ऑडिओ वारंवारता ध्वनीची खेळणी निश्चित करते.
45 45 Measured in Hz, higher frequencies have higher pitch. हर्ट्झमध्ये मोजले जाणारे, उच्च फ्रिक्वेन्सीमध्ये जास्त खेळपट्टी असते.
46 46 See this Wikipedia article. हा विकिपीडिया लेख पहा.
47 47 Reassignment: The method of reassignment sharpens blurry time-frequency data by relocating the data according to local estimates of instantaneous frequency and group delay. This mapping to reassigned time-frequency coordinates is very precise for signals that are separable in time and frequency with respect to the analysis window. पुन्हा नियुक्त करणे: पुन्हा नियुक्त करण्याची पद्धत त्वरित वारंवारता आणि गटाच्या विलंबाच्या स्थानिक अंदाजानुसार डेटा पुनर्स्थित करून अस्पष्ट वेळ-वारंवारता डेटा तीव्र करते. विश्लेषण केलेल्या विंडोच्या संदर्भात वेळ आणि वारंवारतेनुसार विभक्त होणार्‍या सिग्नलसाठी पुन्हा नियुक्त केलेल्या वेळ-वारंवारता समन्वयांचे हे मॅपिंग अगदी अचूक आहे.
48 48 Pitch (EAC): Highlights the contour of the fundamental frequency (musical pitch) of the audio, using the Enhanced Autocorrelation (EAC) algorithm. The EAC Algorithm was developed to produce a mathematical representation of the changes of pitch in a piece of audio. The aim was to allow automated comparison of sound files so that two versions of the same tune could be recognized as being similar, even if played in different keys, or on different instruments. पीच (ईएसी): वर्धित ऑटोकॉरेलेशन (ईएसी) अल्गोरिदम वापरुन ऑडिओच्या मूलभूत वारंवारतेचे (संगीतमय पिच) समोच्च हायलाइट करते. ईएसी अल्गोरिदम ऑडिओच्या तुकड्यात खेळपट्टीच्या बदलांचे गणितीय प्रतिनिधित्व करण्यासाठी विकसित केले गेले. ध्वनी फायलींच्या स्वयंचलित तुलनाची परवानगी देणे हाच हेतू होता जेणेकरून भिन्न टोनमध्ये किंवा भिन्न इंस्ट्रूमेंटवर प्ले केल्या गेल्या तरीही समान ट्यूनच्या दोन आवृत्त्या समान असल्याचे ओळखता येईल.
49 49 Window Size: The dropdown menu lets you choose the size of the Fast Fourier Transform (FFT) window which affects how much vertical (frequency) detail you see. विंडोचा आकारः ड्रॉपडाउन मेनूमुळे आपल्याला फास्ट फूरियर ट्रान्सफॉर्म (एफएफटी) विंडोचा आकार निवडता येतो ज्यामुळे आपण किती उभ्या (वारंवारता) तपशीलांवर परिणाम करता.
50 50 Larger FFT window sizes give more low frequency resolution and less temporal resolution, and are slower. मोठे एफएफटी विंडो आकार अधिक कमी वारंवारता रिझोल्यूशन आणि कमी ऐहिक रिझोल्यूशन देतात आणि हळू असतात.
51 51 Window type: Determines precisely how the spectrogram is computed. विंडोचा प्रकारः स्पेक्ट्रोग्रामची गणना कशी केली जाते हे तंतोतंतपणे निर्धारित करते.
52 52 'Rectangular' is slightly faster than other methods, but introduces some artifacts. 'आयताकृती' इतर पद्धतींपेक्षा किंचित वेगवान आहे, परंतु काही कलाकृतींचा परिचय देते.
53 53 All methods give broadly similar results. सर्व पद्धती मोठ्या प्रमाणात समान परिणाम देतात.
54 54 Zero padding factor: Larger values give finer interpolation of the colors along the vertical axis, at the expense of more computation time. शून्य पॅडिंग घटक: अधिक मूल्ये अधिक मोजण्याच्या वेळेच्या किंमतीवर, मोठी अक्षरे अनुलंब अक्षांसह रंगांचा अधिक चांगला प्रक्षोभन देतात.
55 55 Does not affect the time vs. frequency resolution tradeoff. वेळेच्या विरूद्ध वारंवारता रिझोल्यूशन ट्रेडऑफवर परिणाम होत नाही.
56 56 This option has no effect and is grayed out when the Pitch (EAC) algorithm is selected. जेव्हा या पिच (ईएसी) अल्गोरिदमची निवड केली जाते तेव्हा या पर्यायाचा कोणताही परिणाम होत नाही आणि राखाडी आहे.
57 57 Enable Spectral Selection वर्णक्रमीय निवड सक्षम करा
58 58 Check this box "on" if you want to enable spectral selections. आपण वर्णक्रमीय निवडी सक्षम करू इच्छित असल्यास हा बॉक्स "चालू" पहा.
59 59 These are used to make selections that include a frequency range as well as a time range on tracks in one of the Spectrogram views. Spectral selection is used with special spectral edit effects to make changes to the frequency content of the selected audio. Among other purposes, spectral selection and editing can be used for cleaning up unwanted sound, enhancing certain resonances, changing the quality of a voice or removing mouth sounds from voice work. हे निवडी करण्यासाठी वापरले जातात ज्यात स्पेक्ट्रोग्राम दृश्यांपैकी एकामधील वारंवारता श्रेणी तसेच ट्रॅकवर वेळ श्रेणी समाविष्ट असते. निवडलेल्या ऑडिओच्या वारंवारता सामग्रीमध्ये बदल करण्यासाठी स्पेक्ट्रल निवड विशेष वर्णक्रमीय संपादन प्रभावासह वापरली जाते. इतर कारणांपैकी अवांछित ध्वनी साफ करण्यासाठी, विशिष्ट अनुनादांना वर्धित करण्यासाठी, आवाजाची गुणवत्ता बदलण्यासाठी किंवा व्हॉइसच्या कामातून तोंडातील आवाज काढून टाकण्यासाठी वर्णक्रमीय निवड आणि संपादन वापरले जाऊ शकते.
