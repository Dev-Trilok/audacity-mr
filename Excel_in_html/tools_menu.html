11 11 For details see Add / Remove Effects, Generators and Analyzers. तपशीलांसाठी प्रभाव जोडा / काढा, जनरेटर आणि विश्लेषक काढा.

16 16 Macros... मॅक्रो ...
17 17 Shows the full-function Manage Macros dialog, from which you can edit macros, and apply them. पूर्ण-कार्य व्यवस्थापित करा मॅक्रो संवाद दर्शविते, ज्यामधून आपण मॅक्रो संपादित करू आणि त्यांना लागू करू शकता.
18 18 Macros (Compact Dialog) मॅक्रोज (कॉम्पॅक्ट संवाद)
19 19 Use the Shrink button in the full Manage Macros dialog to get the reduced apply-only dialog. केवळ लागू-कमी संवाद मिळविण्यासाठी पूर्ण मॅक्रो व्यवस्थापित करा संवादामधील संकोलित बटण वापरा.
20 20 This is a smaller version dialog which only has Apply Macro functionality and no Macro editing available. हा एक छोटा आवृत्ती आहे ज्यात केवळ लागू करा मॅक्रो कार्यक्षमता आहे आणि कोणतेही मॅक्रो संपादन उपलब्ध नाही.
21 21 You can get back to the full Manage Macros version by clicking the Expand button. विस्तृत करा बटणावर क्लिक करून आपण पूर्ण मॅक्रो व्यवस्थापित करा आवृत्तीवर परत येऊ शकता.
22 22 This reduced version is useful for presets. ही कमी केलेली आवृत्ती प्रीसेटसाठी उपयुक्त आहे.
23 23 It stays open after applying a macro, so it is a palette of custom functions, and you can pick another and apply that. हे मॅक्रो लागू झाल्यानंतर हे ओपन राहते, जेणेकरून ते सानुकूल फंक्शन्सचे एक पॅलेट आहे आणि आपण दुसरे निवडू शकता आणि ते लागू करू शकता.
24 24 Apply Macro मॅक्रो लागू करा

26 26 Selecting any of these Macros by clicking on it will cause that Macro to be applied to the current project. यातील कोणत्याही मॅक्रोवर क्लिक करून निवडल्यामुळे सध्याच्या प्रकल्पात मॅक्रो लागू होईल.
27 27 The Apply Macro command is a good way of having a palette of favorite presets for the effects you regularly use. आपण नियमितपणे वापरत असलेल्या परिणामांसाठी लागू मॅक्रो आदेश आवडत्या प्रीसेटचे पॅलेट ठेवण्याचा एक चांगला मार्ग आहे.
28 28 Screenshot... स्क्रीनशॉट ...

36 36 Run Benchmark... बेंचमार्क चालवा ...
37 37 Performs a mock editing routine to test how fast Audacity runs on your machine, expressed in terms of the estimated number of simultaneous tracks that could be played at once in a 44100 Hz, 16 bits per sample project. आपल्या मशीनवर ओड्यासिटी किती वेगवान चालते हे तपासण्यासाठी एक मॉक एडिटिंग रूटीन करते, 4400 हर्ट्ज, प्रति नमुना प्रोजेक्टमध्ये 16 बिट्स मध्ये एकाच वेळी खेळल्या जाणार्‍या एकाचवेळी गीतपट्टाच्या अंदाजित संख्येच्या संदर्भात व्यक्त केले जाते.
38 38 If the sample rate of the tracks differs from the Project Rate, causing real-time sample rate conversion, the number of tracks you can play may be lower than the Benchmark result. Using stereo or mono tracks, or using the default 32-bit गीतपट्टाचा नमुना दर रिअल-टाइम नमुना दर रूपांतरणामुळे प्रोजेक्ट रेटपेक्षा वेगळा असल्यास आपण प्ले करू शकता अशा गीतपट्टाची संख्या बेंचमार्क निकालापेक्षा कमी असू शकते. स्टिरिओ किंवा मोनो गीतपट्टा वापरणे किंवा पूर्वनिर्धारित 32-बिट वापरणे
39 39 sample format instead of 16-bit should not significantly change the number of tracks you can play. 16-बिट ऐवजी नमुना स्वरूपात आपण प्ले करू शकणार्‍या गीतपट्टाची संख्या लक्षणीय बदलू नये.

43 43 Regular Interval Labels... नियमित अंतराल लेबले ...
44 44 Places point labels in a label track so as to divide the associated audio into smaller, equally-sized segments. This can be useful for distributing a large file on the internet or ensuring that all exported files are the same size. You can either choose the number of labels to be created or the interval between them. Each label produced can contain specified label text and the labels can be given sequential numbers before or after the text. लेबल गीतपट्टामध्ये लेबले पॉईंट करतात जेणेकरून संबंधित ध्वनी लहान, समान-आकारातील विभागांमध्ये विभाजित करता येईल. इंटरनेटवर मोठी धारिका वितरित करण्यासाठी किंवा सर्व निर्यात केलेल्या धारिका समान आकाराच्या आहेत हे सुनिश्चित करण्यासाठी हे उपयुक्त ठरेल. आपण एकतर तयार केलेली लेबलची संख्या किंवा त्या दरम्यान मध्यांतर निवडू शकता. प्रत्येक उत्पादित लेबलमध्ये निर्दिष्ट लेबल मजकूर असू शकतो आणि लेबल मजकूराच्या आधी किंवा नंतर अनुक्रमांक दिले जाऊ शकतात.
45 45 Sample Data Export... नमुना डेटा निर्यात ...
46 46 Reads the values of successive samples from the selected audio and prints this data to a plain text, CSV or HTML file. Further information may be added as a "header" at the top of the file. निवडलेल्या ध्वनीवरील सलग नमुन्यांची मूल्ये वाचतो आणि हा डेटा एका साध्या मजकूर, सीएसव्ही किंवा एचटीएमएल धारिकावर मुद्रित करतो. धारिकाच्या शीर्षस्थानी पुढील माहिती "शीर्षलेख" म्हणून जोडली जाऊ शकते.
47 47 Sample Data Import... नमुना डेटा आयात ...
48 48 Reads numeric values from a plain ASCII text file and creates a PCM sample for each numeric value read. The values are read as 32-bit floating point numbers, where a range of +/- 1.0 represents 0 dB (full scale). प्लेन एएससीआयआय मजकूर धारिकामधून संख्यात्मक मूल्ये वाचते आणि प्रत्येक अंकीय मूल्य वाचनासाठी पीसीएम नमुना तयार करते. मूल्ये 32-बिट फ्लोटिंग पॉईंट क्रमांक म्हणून वाचली जातात, जिथे +/- 1.0 ची श्रेणी 0 डीबी (पूर्ण स्केल) दर्शवते.
49 49 The file MUST be plain ASCII text, and values should be separated by spaces, tabs or line-breaks. धारिका साधा ASCII मजकूर असणे आवश्यक आहे आणि मूल्ये रिक्त स्थान, टॅब किंवा लाइन-ब्रेकद्वारे विभक्त केल्या पाहिजेत.
50 50 The file name must have a '.txt' file extension. धारिका नावामध्ये '.txt' धारिका विस्तार असणे आवश्यक आहे.
