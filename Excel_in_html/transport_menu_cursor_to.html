4 4 These commands let you move the cursor to the start or end of the selection, track or any adjacent Clip that you may have. या कमांड्स तुम्हाला निवड, ट्रॅक किंवा तुमच्या जवळच्या कोणत्याही क्लिपच्या सुरूवातीस किंवा अंतात कर्सर हलवू देतात.
5 5 Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
6 6 Selection Start निवड प्रारंभ
7 7 When there is a selection, moves the cursor to the start of the selection and removes the selection without changing the zoom level. If the left edge of the selection is not visible, scrolls the display so that the cursor is visible. जेव्हा निवड असते तेव्हा कर्सर निवडीच्या प्रारंभाकडे हलवते आणि झूम पातळी बदलल्याशिवाय निवड काढून टाकते. जर निवडीची डावी बाजू दृश्यमान नसेल, तर प्रदर्शन स्क्रोल करा जेणेकरून कर्सर दृश्यमान होईल.
8 8 Cursor moved to the start of the previously selected region with the selection removed. कर्सर पूर्वी निवडलेल्या प्रदेशाच्या सुरूवातीस निवड काढून टाकला.
9 9 Selection End निवड समाप्त
10 10 When there is a selection, moves the cursor to the end of the selection and removes the selection without changing the zoom level. If the right edge of the selection is not visible, scrolls the display so that the cursor is visible. जेव्हा निवड असते, कर्सर निवडीच्या शेवटी हलवते आणि झूम पातळी बदलल्याशिवाय निवड काढून टाकते. जर निवडीची उजवी धार दिसत नसेल तर डिस्प्लेला स्क्रोल करा जेणेकरून कर्सर दृश्यमान होईल.
11 11 Cursor moved to the end of the previously selected region with the selection removed. कर्सर पूर्वी निवडलेल्या प्रदेशाच्या टोकाकडे गेला आणि निवड हटविली.
12 12 Track Start  J ट्रॅक प्रारंभ जे
13 13 Moves the cursor to the start of the selected track, removing any selection. If the left edge of the track is not visible, scrolls the display so that the cursor is visible. कोणतीही निवड काढून, निवडलेल्या ट्रॅकच्या सुरूवातीस कर्सर हलवते. ट्रॅकची डावी धार दिसत नसल्यास, प्रदर्शन स्क्रोल करते जेणेकरून कर्सर दृश्यमान होईल.
14 14 The shorter track was the one selected, so the cursor is moved to the 38 second mark at the beginning of that track with the selection removed. छोटा ट्रॅक निवडलेला होता, म्हणून कर्सर त्या ट्रॅकच्या सुरूवातीस 38 सेकंदांच्या मार्कवर नेऊन निवडला गेला.
15 15 Track End  K ट्रॅक एंड के
16 16 Moves the cursor to the end of the selected track, removing any selection. If the right edge of the track is not visible, scrolls the display so that the cursor is visible. कोणतीही निवड काढून, निवडलेल्या ट्रॅकच्या शेवटी कर्सर हलवते. ट्रॅकची उजवी धार दिसत नसल्यास, प्रदर्शन स्क्रोल करते जेणेकरून कर्सर दृश्यमान होईल.
17 17 Operating on the same track selection as above, the cursor is moved to the 1:28 mark at the end of the shorter selected track with the selection removed. वरील प्रमाणेच ट्रॅक निवडीवर कार्य करीत, कर्सर निवड निवडलेल्या ट्रॅकच्या छोट्या निवडानंतर 1:28 मार्क वर हलविला जाईल.
18 18 Previous Clip Boundary मागील क्लिप सीमा
19 19 Moves the cursor position backwards to the previous clip boundary. मागील क्लिप सीमेवर कर्सर स्थिती मागे सरकते.
20 20 If the cursor is inside a clip, but not exactly on the left boundary, this command will move the cursor to the left boundary of the clip. जर कर्सर क्लिपच्या आत असेल, परंतु अगदी डाव्या सीमेवर नसेल तर, ही कमांड कर्सरला क्लिपच्या डाव्या सीमेवर हलवेल.
21 21 If the cursor is exactly on the left boundary of a clip, this command will move the cursor to the right boundary of the previous clip. जर कर्सर अगदी क्लिपच्या डाव्या सीमेवर असेल तर ही कमांड मागील क्लिपच्या उजव्या सीमेवर कर्सर हलवेल.
22 22 If the cursor is between clips, this command will move the cursor to the right boundary of the previous clip. जर कर्सर क्लिपच्या दरम्यान असेल तर ही कमांड मागील क्लिपच्या उजव्या सीमेवर कर्सर हलवेल.
23 23 If the cursor is at the left edge of the first clip in the track, this command does nothing. जर कर्सर ट्रॅकमधील पहिल्या क्लिपच्या डाव्या काठावर असेल तर, ही आज्ञा काहीही करत नाही.
24 24 Both Previous Clip Boundary and Next Clip Boundary (below) use the clips in the selected track or tracks. If no tracks are selected, the clips in all tracks are used. मागील क्लिप सीमा आणि पुढील क्लिप सीमा (खाली) दोन्ही निवडलेल्या ट्रॅक किंवा ट्रॅकमध्ये क्लिप वापरतात. कोणतेही ट्रॅक निवडलेले नसल्यास, सर्व ट्रॅकमधील क्लिप वापरल्या जातील.
25 25 Next Clip Boundary पुढील क्लिप सीमा
26 26 Moves the cursor position forwards to the next clip boundary. कर्सर स्थान पुढील क्लिप सीमेवर हलवते.
27 27 If the cursor is inside a clip, but not exactly on the right boundary, this command will move the cursor to the right boundary of the clip. जर कर्सर क्लिपच्या आत असेल, परंतु अगदी उजव्या सीमेवर नसेल तर, ही कमांड कर्सरला क्लिपच्या उजव्या सीमेवर हलवेल.
28 28 If the cursor is exactly on the right boundary of a clip, this command will move the cursor to the left boundary of the next clip. जर कर्सर अगदी क्लिपच्या उजव्या सीमेवर असेल तर, ही कमांड कर्सरला पुढील क्लिपच्या डाव्या सीमेवर हलवेल.
29 29 If the cursor is between clips, this command will move the cursor to the left boundary of the next clip. जर कर्सर क्लिप दरम्यान असेल तर ही कमांड पुढील क्लिपच्या डाव्या सीमेवर कर्सर हलवेल.
30 30 If the cursor is at the right edge of the last clip in the track, this command does nothing. जर कर्सर ट्रॅकमधील शेवटच्या क्लिपच्या उजव्या काठावर असेल तर, ही आज्ञा काहीही करत नाही.
31 31 Both Previous Clip Boundary and Next Clip Boundary will scroll the display to show the cursor if the result of the command moves the cursor out of view. कमांडचा परिणाम कर्सरला दृश्यास्पद बाहेर हलवित असल्यास मागील क्लिप सीमा आणि पुढील क्लिप सीमा दोन्ही कर्सर दर्शविण्यासाठी प्रदर्शन स्क्रोल करेल.
32 32 Project Start  Home प्रोजेक्ट स्टार्ट होम
33 33 Moves the cursor to the beginning of the project. This is useful if you want to play everything, or record a new track starting from the beginning. Same as clicking the Skip to Start button प्रोजेक्टच्या सुरुवातीस कर्सर हलवते. आपण सर्वकाही प्ले करू इच्छित असल्यास, किंवा प्रारंभपासून नवीन ट्रॅक रेकॉर्ड करू इच्छित असल्यास हे उपयुक्त आहे. प्रारंभ करण्यासाठी वगळा बटणावर क्लिक करण्यासारखेच
34 34 in Transport Toolbar. ट्रान्सपोर्ट टूलबार मध्ये.
35 35 Project End  End प्रोजेक्ट एंड एंड
36 36 Moves the cursor to the end of the project. Same as clicking the Skip to End button प्रोजेक्टच्या शेवटी कर्सर हलवते. वगळा बटणावर क्लिक करण्यासारखेच
37 37 in Transport Toolbar. ट्रान्सपोर्ट टूलबार मध्ये.
