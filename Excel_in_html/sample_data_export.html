7 7 The number of samples processed will be the number of samples in the selection, up to a maximum of the number entered here. The default number of samples is 100. प्रक्रिया केलेल्या नमुन्यांची संख्या येथे नमूद केलेल्या संख्येपर्यंत, निवडीमधील नमुन्यांची संख्या असेल. नमुन्यांची पूर्वनिर्धारित संख्या 100 आहे.
8 8 Sample Data Export is limited internally to an absolute maximum of 1 million samples. नमुना डेटा निर्यात हे केवळ अंतर्गत जास्तीत जास्त 1 दशलक्ष नमुन्यांपर्यंत मर्यादित आहे.
9 9 Processing a large number of samples can create extremely large output files that may be difficult to open. Word processing application such as WordPad can usually open large TXT files, but for HTML output it is advisable to limit the number of samples to a maximum of a few hundred samples. मोठ्या संख्येने नमुन्यांची प्रक्रिया करणे अत्यंत मोठ्या आउटपुट धारिका तयार करू शकते ज्या उघडणे कठीण आहे. वर्ड प्रोसेसिंग suchप्लिकेशन जसे की वर्डपॅड सामान्यत: मोठ्या टीएक्सटी धारिका उघडू शकतो, परंतु एचटीएमएल आउटपुटसाठी नमुन्यांची संख्या जास्तीत जास्त काही शंभर नमुने मर्यादित करण्याचा सल्ला दिला जातो.
10 10 The track sample rate as indicated on the Track Control Panel indicates the number of samples per second. If the track and project rates are the same, the number of samples in a selection may be viewed in Selection Toolbar if you choose one of the selection formats that includes samples. गीतपट्टा नियंत्रण पॅनेलवर दर्शविल्याप्रमाणे गीतपट्टाचा नमुना दर प्रति सेकंद नमुन्यांची संख्या दर्शवितो. जर गीतपट्टा आणि प्रोजेक्टचे दर समान असतील तर आपण निवडलेल्या नमुन्यांपैकी एखादे नमुने निवडल्यास निवडीमधील नमुन्यांची संख्या निवड साधनपट्टीमध्ये पाहिली जाऊ शकते.

33 33 File data format धारिका डेटा स्वरूप
34 34 "Sample Data Export" supports exporting as plain text (TXT), comma separated values (CSV) or as a web page (HTML). The format is automatically selected based on the file extension of the exported file. The default format is plain ASCII text. "नमुना डेटा निर्यात" प्लेन टेक्स्ट (टीएक्सटी), स्वल्पविरामाने विभक्त मूल्ये (सीएसव्ही) किंवा वेब पृष्ठ (एचटीएमएल) म्हणून निर्यात करण्यास समर्थन देते. निर्यात केलेल्या धारिकाच्या धारिका विस्ताराच्या आधारावर स्वयंचलितपणे स्वरूपन निवडले जाते. पूर्वनिर्धारित स्वरूप साधा ASCII मजकूर आहे.
35 35 Data in your chosen format as listed below appears after any header information which you included. Text files where the output produces columns may not have perfectly lined-up columns due to formatting limitations of text files. खाली निवडलेल्या आपल्या निवडलेल्या स्वरूपातील डेटा आपण समाविष्ट असलेल्या कोणत्याही शीर्षलेख माहितीनंतर दिसून येईल. मजकूर धारिका्स जेथे आउटपुट स्तंभ तयार करतात तेथे मजकूर धारिकांच्या स्वरूपण मर्यादांमुळे योग्य प्रकारे स्तंभबद्ध स्तंभ नसू शकतात.
36 36 Sample List (txt): produces a plain text file with a list of sample values. नमुना यादी (टेक्स्ट): नमुना मूल्यांच्या सूचीसह एक साधी मजकूर धारिका तयार करते.
37 37 Indexed List (txt): produces a plain text file with an indexed list of sample values: sample number followed by sample value. अनुक्रमणिका यादी (टेक्स्ट): नमुना मूल्यांच्या अनुक्रमित सूचीसह एक साधी मजकूर धारिका तयार करते: नमुना मूल्य त्यानंतर नमुना मूल्य.
38 38 Time Indexed (txt): produces a plain text file with the time (relative to the start of the selection) followed by sample value वेळ इंडेक्स्ड (टेक्स्ट): वेळेसह एक साधी मजकूर धारिका तयार करते (निवडीच्या प्रारंभाशी संबंधित) त्यानंतर नमुना मूल्य
39 39 Data (csv): produces a CSV formatted file. This may be suitable for importing into spreadsheet applications for further analysis. डेटा (सीएसव्ही): एक सीएसव्ही स्वरूपित धारिका तयार करते. पुढील विश्लेषणासाठी हे स्प्रेडशीट अनुप्रयोगांमध्ये आयात करण्यासाठी योग्य ठरेल.
40 40 Web Page (html): produces an HTML 5 document that contains all of the header information and a table of sample data with sample number, time, linear and dB values. Web browsers that are not HTML 5 compliant may not display the page correctly. वेब पृष्ठ (एचटीएमएल): एक एचटीएमएल 5 दस्तऐवज तयार करते ज्यात सर्व शीर्षलेख माहिती आणि नमुना क्रमांक, वेळ, रेषीय आणि डीबी मूल्ये असलेल्या नमुना डेटा सारणीचा समावेश असतो. एचटीएमएल 5 अनुरूप नसलेले वेब ब्राउझर कदाचित पृष्ठ योग्य प्रकारे प्रदर्शित करू शकत नाहीत.
41 41 Include header information शीर्षलेख माहिती समाविष्ट करा
42 42 None: Prints only the sample data, preceded by any text you entered in the optional header text box (leave blank for none). काहीही नाही: पर्यायी शीर्षलेख मजकूर बॉक्समध्ये आपण प्रविष्ट केलेल्या कोणत्याही मजकूराच्या आधीचा नमुना डेटा फक्त छापतो (कशासाठीही रिक्त सोडू नका).
43 43 Minimal: Prints a minimal header that contains: किमान: एक किमान शीर्षलेख मुद्रित करते ज्यात:

45 45 Units (linear or dB) युनिट्स (रेखीय किंवा डीबी)
46 46 Optional header text (leave blank for none) पर्यायी शीर्षलेख मजकूर (कोणत्याहीसाठी रिक्त सोडू नका)
47 47 Standard: In addition to the Minimal header information, the following is also included: मानक: किमान शीर्षलेख माहिती व्यतिरिक्त, खालील देखील समाविष्ट केले आहे:
48 48 File name धारिकाचे नाव
49 49 Number of samples नमुने संख्या
50 50 Duration (seconds) कालावधी (सेकंद)
51 51 Mono or stereo मोनो किंवा स्टीरिओ
52 52 All: In addition to the Standard header information, the following is also included: सर्व: मानक शीर्षलेख माहिती व्यतिरिक्त, खालील देखील समाविष्ट केले आहे:
53 53 Peak amplitude (linear and dB) पीक मोठेपणा (रेखीय आणि डीबी)
54 54 Unweighted RMS level (dB) अदृष्य आरएमएस स्तर (डीबी)

56 56 Irrespective of these settings, the Standard information, file name, and any Optional header, are displayed in a confirmation message on successful completion of the export. या सेटिंग्सची पर्वा न करता, मानक माहिती, धारिकाचे नाव आणि कोणतेही वैकल्पिक शीर्षलेख, निर्यात यशस्वी झाल्यावर पुष्टीकरण संदेशात दर्शविले जातात.
