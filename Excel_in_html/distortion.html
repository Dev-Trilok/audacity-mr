4 4 Use the Distortion effect to make the audio sound distorted. By distorting the waveform the frequency content is changed, which will often make the sound 'crunchy' or 'abrasive'. ऑडिओ ध्वनी विकृत करण्यासाठी विरूपण प्रभाव वापरा. वेव्हफॉर्म विकृत करून वारंवारतेची सामग्री बदलली जाते, जे बर्‍याचदा ध्वनीला 'कुरकुरे' किंवा 'अपघर्षक' करते.
5 5 Technically this effect is a waveshaper. The result of waveshaping is equivalent to applying non-linear amplification to the audio waveform. Preset shaping functions are provided, each of which produces a different type of distortion. तांत्रिकदृष्ट्या हा प्रभाव वेव्हशेपर आहे. वेव्हशेपिंगचा परिणाम ऑडिओ वेव्हफॉर्मवर नॉन-रेखीय प्रवर्धन लागू करण्याइतकेच आहे. प्रीसेट शेपिंग फंक्शन्स प्रदान केल्या आहेत, त्यापैकी प्रत्येकजण भिन्न प्रकारचे विकृती उत्पन्न करते.
6 6 Presets are provided in the Manage menu, which suggest typical settings for a wide range of uses. प्रीसेट्स व्यवस्थापित करा मेनूमध्ये प्रदान केल्या आहेत, जे विस्तृत वापराच्या वैशिष्ट्यांसह सेटिंग्ज दर्शवितात.
7 7 Accessed by: Effect > Distortion... याद्वारे प्रवेश केलेले: प्रभाव> विकृती ...
8 8 For more "analog" sounding distortion, the track should be resampled to a high sampling rate so as to reduce aliasing. अधिक "अ‍ॅनालॉग" ध्वनी विकृततेसाठी, ट्रॅक उच्च सॅम्पलिंग दरामध्ये पुन्हा तयार केला जावा जेणेकरुन अलियासिंग कमी होईल.
9 9 Equalization may also be applied, before or after applying the distortion effect to modify the timbre of the distorted audio. विकृत ऑडिओची लांबी सुधारित करण्यासाठी विकृतीचा प्रभाव लागू करण्यापूर्वी किंवा नंतरही समता लागू केली जाऊ शकते.
10 10 Distortion type विकृतीचा प्रकार
11 11 These are the shaping functions. Each type defines how the input signal is mapped to the output level. हे आकार देणारी कार्ये आहेत. प्रत्येक प्रकार आउटपुट स्तरावर इनपुट सिग्नलला कसे मॅप केले जाते ते परिभाषित करते.
12 12 Each distortion type has one or more control parameters that determine how the shaping function is applied. There is a total of five slider controls. The controls are enabled or disabled automatically depending on which "type" of distortion is selected. प्रत्येक विकृत प्रकारात एक किंवा अधिक नियंत्रण पॅरामीटर्स असतात जे आकार कार्य कसे लागू करतात ते निर्धारित करतात. एकूण पाच स्लाइडर नियंत्रणे आहेत. विकृती कोणत्या "प्रकार" निवडली यावर अवलंबून नियंत्रणे स्वयंचलितपणे सक्षम किंवा अक्षम केली जातात.
13 13 For all distortion types, input signals above 0 dB are "hard clipped" at 0 dB in addition to the specified 'shaping'. सर्व विकृत प्रकारांकरिता, निर्दिष्ट केलेल्या 'शेपिंग' व्यतिरिक्त 0 डीबी वरील इनपुट सिग्नल 0 डीबी वर "हार्ड क्लीप केलेले" असतात.
14 14 Clipping Distortion क्लिपिंग विकृती
15 15 Hard Clipping हार्ड क्लिपिंग
16 16 This type of distortion cuts off peaks at the top and bottom of the waveform. Note that hard clipping is very likely to cause aliasing distortion, which may be undesirable. या प्रकारच्या विकृतीमुळे वेव्हफॉर्मच्या वरच्या आणि खालच्या भागातील शिखरे बंद होतात. लक्षात ठेवा की हार्ड क्लिपिंगमुळे अलियासिंग विकृती होण्याची शक्यता असते, जी अनिष्ट असू शकते.
17 17 Clipping level (-100 to 0 dB): Peaks greater than this level are cut off. The slider control has a logarithmic scale to allow settings close to 0 dB to be made more easily and accurately. कतरणे पातळी (-100 ते 0 डीबी): या स्तरापेक्षा उंचीची शिखर कापली आहेत. स्लाइडर नियंत्रणात 0 डीबी जवळील सेटिंग्ज अधिक सुलभ आणि अचूकपणे करता येण्याकरिता लॉगरिथमिक स्केल आहे.
18 18 Drive (0 to 100): When set to greater than 0, the waveform is amplified this amount (dB) prior to being clipped. ड्राइव्ह (० ते १००): 0 पेक्षा जास्त वर सेट केलेले असताना, वेव्हफॉर्म क्लिप होण्यापूर्वी ही रक्कम (डीबी) वाढविली जाते.
19 19 Make-up Gain (0 to 100): When set greater than 0, the output from the effect is amplified. When set to 100, the amplification is such that an input level of 0 dB (full track height) will produce an output level of 0 dB. मेक-अप गेन (0 ते 100): 0 पेक्षा जास्त सेट केल्यावर, परिणामाचे आउटपुट वाढविले जाईल. 100 वर सेट केल्यास, एम्प्लिफिकेशन असे आहे की 0 डीबीचे इनपुट लेव्हल (पूर्ण ट्रॅक उंची) 0 डीबीचे आउटपुट लेव्हल तयार करेल.
20 20 Soft Clipping सॉफ्ट क्लीपिंग
21 21 This type of distortion is similar to Hard Clipping except that 'corners' where clipping intersects with the waveform are rounded. This is achieved by progressively reducing the gain when the input is above the clipping threshold. The softer the clipping, the more rounded the clipped peaks and the less risk of aliasing distortion. या प्रकारचे विकृति हार्ड क्लिपिंगसारखेच आहे ज्याशिवाय 'कोपरे' क्लिपिंगला ज्याने वेव्हफॉर्मसह छेदते ते गोल केले जाते. जेव्हा इनपुट क्लिपिंग थ्रेशोल्डच्या वर असेल तेव्हा क्रमाने नफ्यावर कमी करुन हे साध्य केले जाते. क्लिपिंग जितके मऊ असेल तितके अधिक क्लिप केलेले शिखर आणि विकृती कमी होण्याचे कमी धोका.
22 22 Clipping threshold (-100 to 0 dB): The level at which gain reduction (clipping) starts. The slider control has a logarithmic scale to allow settings close to 0 dB to be made more easily and accurately. क्लिपिंग थ्रेशोल्ड (-100 ते 0 डीबी): ज्या स्तरावर वाढीची कपात (क्लिपिंग) सुरू होते. स्लाइडर नियंत्रणात 0 डीबी जवळील सेटिंग्ज अधिक सुलभ आणि अचूकपणे करता येण्याकरिता लॉगरिथमिक स्केल आहे.
23 23 Hardness (0 to 100): The 'hardness' of the clipping (100 % being "hard"). कडकपणा (0 ते 100): क्लिपिंगची 'कठोरता' (100% "कठोर")
24 24 Make-up Gain (0 to 100): When set greater than 0, the output from the effect is amplified. When set to 100, the amplification is such that an input level of 0 dB (full track height) will produce an output level of 0 dB. मेक-अप गेन (0 ते 100): 0 पेक्षा जास्त सेट केल्यावर, परिणामाचे आउटपुट वाढविले जाईल. 100 वर सेट केल्यास, एम्प्लिफिकेशन असे आहे की 0 डीबीचे इनपुट लेव्हल (पूर्ण ट्रॅक उंची) 0 डीबीचे आउटपुट लेव्हल तयार करेल.
25 25 Note that even without "Make-up Gain", the output level may be higher than the clipping threshold because the peaks are in effect 'squashed' flatter rather than being completely cut off. लक्षात घ्या की "मेक-अप गेन" शिवाय देखील आउटपुट लेव्हल क्लिपिंग थ्रेशोल्डपेक्षा जास्त असू शकते कारण शिखर पूर्णपणे कापण्याऐवजी 'स्क्वॅशड' चापटपणाचा प्रभाव असतो.
26 26 Overdrive Distortion ओव्हरड्राईव्ह विकृती
27 27 For all three 'Overdrive' types, the shaping function is a continuous smooth curve. तिन्ही 'ओव्हरड्राईव्ह' प्रकारांसाठी आकार देणे हे एक सतत गुळगुळीत वक्र असते.
28 28 Soft Overdrive मऊ ओव्हरड्राईव्ह
29 29 This is the "softest" of the overdrive types and produces the least amount of high harmonics. It can be useful for reducing the dynamic range (thus increasing the sustain of musical instruments) while adding the least amount of "crunch" to the sound. ओव्हरड्राईव्ह प्रकारांपैकी हा "मऊ" आहे आणि कमीतकमी उच्च हार्मोनिक्स तयार करतो. आवाजात "क्रंच" कमीतकमी प्रमाणात जोडताना डायनॅमिक रेंज कमी करण्यासाठी (अशा प्रकारे वाद्य वाजविण्याचा कार्यक्षमता वाढवणे) उपयुक्त ठरू शकते.
30 30 Distortion amount (0 to 100): The amount / strength of the distortion. विकृतीची रक्कम (0 ते 100): विकृतीचे प्रमाण / सामर्थ्य.
31 31 Output level (0 to 100): An output level adjustment. When set to 0 the output is silent. आउटपुट लेव्हल (0 ते 100): आउटपुट लेव्हल mentडजस्टमेंट. 0 वर सेट केल्यावर आउटपुट मूक होते.
32 32 Medium Overdrive मध्यम ओव्हरड्राईव्ह
33 33 Distortion amount (0 to 100): The amount / strength of the distortion. विकृतीची रक्कम (0 ते 100): विकृतीचे प्रमाण / सामर्थ्य.
34 34 Output level (0 to 100): An output level adjustment. When set to 0 the output is silent. आउटपुट लेव्हल (0 ते 100): आउटपुट लेव्हल mentडजस्टमेंट. 0 वर सेट केल्यावर आउटपुट मूक होते.
35 35 Hard Overdrive हार्ड ओव्हरड्राईव्ह
36 36 This is the "hardest" of the overdrive types and produces the greatest amount of high harmonics. This would typically be used for producing a heavily distorted effect. ओव्हरड्राईव्ह प्रकारांपैकी हा "सर्वात कठीण" आहे आणि मोठ्या प्रमाणात उच्च हार्मोनिक्स तयार करतो. हे सामान्यतः मोठ्या प्रमाणात विकृत प्रभाव तयार करण्यासाठी वापरले जाईल.
37 37 Distortion amount (0 to 100): The amount / strength of the distortion. विकृतीची रक्कम (0 ते 100): विकृतीचे प्रमाण / सामर्थ्य.
38 38 Output level (0 to 100): An output level adjustment. When set to 0 the output is silent. आउटपुट लेव्हल (0 ते 100): आउटपुट लेव्हल mentडजस्टमेंट. 0 वर सेट केल्यावर आउटपुट मूक होते.
39 39 Harmonic Distortion हार्मोनिक विकृती
40 40 These types of distortion create a series of odd or even harmonics. "Odd" harmonics have frequencies that are an odd number of times higher than the fundamental frequency, whereas even harmonics are an even number of times higher than the fundamental frequency. For example, the second harmonic of a 440 Hz sine wave is a sine wave with a frequency या प्रकारच्या विकृती विचित्र किंवा समरसतेची मालिका तयार करतात. "ऑड" हार्मोनिक्समध्ये वारंवारता असतात जी मूलभूत वारंवारतेपेक्षा विचित्र संख्येपेक्षा जास्त वेळा असतात, तर हार्मोनिक्ससुद्धा मूलभूत वारंवारतेपेक्षा अगदी पटीने जास्त असतात. उदाहरणार्थ, 440 हर्ट्झ साइन वेव्हची दुसरी हार्मोनिक वारंवारता असलेली साइन वेव आहे

Note that because real world sounds contain many frequencies, these distortion effects are unlikely to create recognizable "harmony" type sounds, but rather will produce distortion with a unique tonal character. च्या 880 हर्ट्ज लक्षात घ्या की वास्तविक जगाच्या ध्वनीमध्ये बर्‍याच फ्रिक्वेन्सी असतात, या विकृतीच्या प्रभावांमध्ये ओळखले जाणारे "सामंजस्य" प्रकारचे आवाज तयार होण्याची शक्यता नसते, परंतु त्याऐवजी एका विशिष्ट टोनल कॅरेक्टरसह विकृती निर्माण होते.
42 42 Cubic Curve (odd harmonics) क्यूबिक वक्र (विषम सुसंवाद)
43 43 This shaping function is often used in electric guitar distortion effects as it is possible to avoid aliasing distortion with only a modest amount of oversampling. This implementation does not use oversampling, though oversampling can be applied manually by resampling the track to a higher rate prior to applying this distortion effect. हे आकार देण्याचे कार्य बर्‍याचदा इलेक्ट्रिक गिटार विकृत प्रभावांमध्ये वापरले जाते कारण केवळ ओव्हरसाम्पलिंगच्या माफक प्रमाणात अलिएसिंग विकृती टाळणे शक्य आहे. हे अंमलबजावणी ओव्हरस्पीलिंगचा वापर करीत नाही, तथापि हा विकृतीकरण प्रभाव लागू करण्यापूर्वी ओव्हरस्म्पलिंग ट्रॅकला अधिक दरामध्ये पुन्हा स्थापित करून स्वहस्ते लागू केले जाऊ शकते.
44 44 One limitation of this type of distortion is that even when applied fully, the effect is still quite mild. Stronger effects can be achieved by applying the algorithm multiple times, which can be achieved by setting the "Repeat processing" control greater than 0. या प्रकारच्या विकृतीची एक मर्यादा अशी आहे की संपूर्णपणे लागू केली तरीही त्याचा प्रभाव अद्याप अगदी सौम्य आहे. अल्गोरिदम एकाधिक वेळा लागू करून मजबूत प्रभाव साध्य केले जाऊ शकतात, जे 0 पेक्षा मोठे "रिपीट प्रोसेसिंग" नियंत्रण सेट करून प्राप्त केले जाऊ शकतात.
45 45 When applied once only, each frequency component of the sound gains one harmonic overtone that is 3 times the original frequency. When the effect is applied multiple times, additional odd numbered harmonics are created. फक्त एकदाच लागू केल्यावर, ध्वनीचा प्रत्येक वारंवारता घटक मूळ आवृत्त्यांपेक्षा 3 पट अधिक एक हार्मोनिक ओव्हरटोन मिळवितो. जेव्हा प्रभाव एकाधिक वेळा लागू केला जातो तेव्हा अतिरिक्त विषम क्रमांकित हार्मोनिक्स तयार केले जातात.
46 46 Distortion amount (0 to 100): The amount / strength of the distortion. विकृतीची रक्कम (0 ते 100): विकृतीचे प्रमाण / सामर्थ्य.
47 47 Output level (0 to 100): An output level adjustment. When set to 0 the output is silent. आउटपुट लेव्हल (0 ते 100): आउटपुट लेव्हल mentडजस्टमेंट. 0 वर सेट केल्यावर आउटपुट मूक होते.
48 48 Repeat processing (0 to 5): The number of times to repeat the distortion algorithm. When set to 0 the effect is applied once only. पुनरावृत्ती प्रक्रिया (0 ते 5): विकृत अल्गोरिदमची पुनरावृत्ती करण्यासाठी किती वेळा. 0 वर सेट केल्यास केवळ एकदाच प्रभाव लागू केला जातो.
49 49 Even Harmonics जरी हार्मोनिक्स
50 50 This type of distortion produces a series of harmonics that are 2, 4, 6... times the frequency of the original waveform (even numbers). Producing even harmonics requires that the waveform is distorted asymmetrically, which if not corrected will produce substantial DC offset. To counter the DC offset, a DC blocking filter is provided. या प्रकारच्या विकृतीमुळे हार्मोनिक्सची मालिका तयार होते जी 2, 4, 6 असते ... मूळ वेव्हफॉर्मच्या वारंवारतेपेक्षा (अगदी संख्या). अगदी हार्मोनिक्स तयार करण्यासाठी वेव्हफॉर्म असमानमित विकृत करणे आवश्यक आहे, जे दुरुस्त न केल्यास पर्याप्त डीसी ऑफसेट तयार करेल. डीसी ऑफसेटचा सामना करण्यासाठी, डीसी ब्लॉकिंग फिल्टर प्रदान केले जाते.
51 51 DC blocking filter (checkbox) When checked, a low frequency filter is applied which will reduce the DC component of the processed audio. डीसी ब्लॉकिंग फिल्टर (चेकबॉक्स) चेक केल्यावर, कमी फ्रिक्वेंसी फिल्टर लागू केला जातो जो प्रक्रिया केलेल्या ऑडिओचा डीसी घटक कमी करेल.
52 52 Distortion amount (0 to 100): The amount / strength of the distortion. विकृतीची रक्कम (0 ते 100): विकृतीचे प्रमाण / सामर्थ्य.
53 53 Harmonic brightness (0 to 100): Higher values produce a greater number of harmonics. हार्मोनिक ब्राइटनेस (0 ते 100): उच्च मूल्ये मोठ्या संख्येने हार्मोनिक्स तयार करतात.
54 54 Hybrid Distortion संकरित विकृती
55 55 This type of distortion is distinct from the other presets in that the shaping functions curves in two directions (an "S" shape). Thus low level signals are "expanded" downwards, becoming quieter, while medium level signals are made louder creating dynamic range compression for louder sounds. या प्रकारची विकृती इतर प्रीसेटपेक्षा वेगळी आहे ज्यामध्ये आकार देणारी कार्ये दोन दिशानिर्देश ("एस" आकार) वक्र करते. अशा प्रकारे निम्न स्तरीय सिग्नल खालच्या दिशेने "विस्तारीत" केले जातात, शांत होतात, तर मध्यम पातळीचे सिग्नल जोरात केले जातात जेणेकरून मोठ्या आवाजात डायनॅमिक रेंज कॉम्प्रेशन तयार होते.
56 56 Expand and Compress विस्तृत करा आणि संकुचित करा
57 57 The transformation shape for this type of distortion is a sinusoidal "S" shape for both positive and negative going signals. The distortion of high peaks is similar to soft clipping or overdrive, while distortion of low level sounds is similar to crossover distortion. या प्रकारच्या विकृतीचा रूपांतर हा एक सकारात्मक आणि नकारात्मक दोन्ही सिग्नलसाठी साइनसॉइडल "एस" आकार आहे. उच्च शिखरांचे विकृती सॉफ्ट क्लीपिंग किंवा ओव्हरड्राइव्हसारखेच आहे, तर निम्न स्तरावरील ध्वनी विकृती क्रॉसओव्हर विकृतीच्या समान आहे.
58 58 Distortion amount (0 to 100): The amount / strength of the distortion. विकृतीची रक्कम (0 ते 100): विकृतीचे प्रमाण / सामर्थ्य.
59 59 Output level (0 to 100): An output level adjustment. When set to 0 the output is silent. आउटपुट लेव्हल (0 ते 100): आउटपुट लेव्हल mentडजस्टमेंट. 0 वर सेट केल्यावर आउटपुट मूक होते.

If the Wet level is 100% and Residue level 0%, the peak level of the result will be clipped down to this level. डीबी मर्यादा (-100 ते 0 डीबी): [डीफॉल्ट = -6] ही मोठेपणा पातळी आहे ज्याच्या वरील इनपुट सिग्नलवर प्रक्रिया केली जाते. जर ओले पातळी 100% आणि अवशेष पातळी 0% असेल तर निकालाची शिखर पातळी या पातळीवर खाली जाईल.
75 75 Wet level (-100 to 0dB): [default = 50] This is the percentage of the clipped signal that is fed to the output. It thus acts as a volume control for audio below the dB limit. When set to 100%, all sounds below the dB limit are fed to the output. At lower settings, the volume of the result will be reduced. ओले पातळी (-100 ते 0 डीबी): [डीफॉल्ट = 50] हे आउटपुटला दिले जाणा cli्या क्लिप केलेल्या सिग्नलची टक्केवारी आहे. हे डीबी मर्यादेच्या खाली ऑडिओसाठी व्हॉल्यूम नियंत्रण म्हणून कार्य करते. 100% वर सेट केल्यावर, डीबी मर्यादेच्या खाली असलेले सर्व आवाज आउटपुटला दिले जातात. कमी सेटिंग्जमध्ये, निकालाचे प्रमाण कमी होईल.
76 76 Residue level (-100 to 0dB): [default = 50] This allows a proportion of the signal that has been removed by clipping to be added back to the output and so soften the effect. When set to 0%, all of the clipped signal is discarded. At higher settings, some of the clipped signal will be restored, making the limit softer.
