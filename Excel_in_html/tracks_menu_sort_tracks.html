4 4 The Sort Tracks submenu provides commands to sort your tracks by Track Names or the tracks' start times. ट्रॅकची क्रमवारी सबमेनू आपल्या मागोवा ट्रॅक नावे किंवा ट्रॅकच्या प्रारंभ वेळांनुसार क्रमवारी लावण्यासाठी आदेश प्रदान करते.
5 5 Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
6 6 By Start time प्रारंभ वेळेत
7 7 Sorts all tracks in the project from top to bottom in the project window. After the sort the top track will have the earliest start time and the bottom track will have the latest start time. प्रोजेक्ट विंडोमध्ये वरपासून खालपर्यंत सर्व ट्रॅकची क्रमवारी लावते. क्रमवारी लावल्यानंतर शीर्ष ट्रॅकवर लवकरात लवकर प्रारंभ वेळ असेल आणि तळाशी असलेल्या ट्रॅकवर नवीनतम प्रारंभ वेळ असेल.
8 8 By Name नावाने
9 9 Sort all tracks in the project alphanumerically by track name. Numbers precede letters, and upper-case letters precede lower case letters. It does not use natural number sort, so "10" comes before "2" (but after "02"). प्रकल्पातील सर्व ट्रॅक अक्षरेद्वारे ट्रॅक नावाने क्रमवारी लावा. संख्या आधीची अक्षरे आणि अपर-केस अक्षरे लोअर केस अक्षरे आधी. हे नैसर्गिक संख्या क्रमवारी वापरत नाही, म्हणून "2" "2" च्या आधी येईल (परंतु "02" नंतर).
