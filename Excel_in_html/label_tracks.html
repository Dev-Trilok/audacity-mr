19 19 Features of a Label Track लेबल ट्रॅकची वैशिष्ट्ये
20 20 Labels can be used to mark then restore chosen points or regions of audio for playback or editing. प्लेबॅक किंवा संपादनासाठी निवडलेले बिंदू किंवा ऑडिओ प्रदेश पुनर्संचयित करण्यासाठी नंतर लेबलचा वापर केला जाऊ शकतो.
21 21 Labels can contain text for purposes of annotation or transcription. भाष्य किंवा प्रतिलेखनाच्या उद्देशाने लेबलांमध्ये मजकूर असू शकतो.
22 22 Labels and their text provide a convenient way to name different songs in a recorded track then export all songs at once to separate audio files using File > Export > Export Multiple.... लेबले आणि त्यांचा मजकूर रेकॉर्ड केलेल्या ट्रॅकमध्ये भिन्न गाण्यांना नावे ठेवण्यासाठी सोयीचा मार्ग प्रदान करतात नंतर फाइल> निर्यात> एकाधिक निर्यात करा .... वापरून ऑडिओ फायली विभक्त करण्यासाठी सर्व गाणी एकाच वेळी निर्यात करा.
23 23 Many of the tools in the Analyze Menu use labels to mark detected regions (such as regions of विश्लेषण मेनूमधील अनेक साधने आढळलेली प्रदेश चिन्हांकित करण्यासाठी लेबलांचा वापर करतात (जसे की प्रदेश
24 24 silence, sound or clipping). Point labels can mark detected beats. शांतता, आवाज किंवा क्लिपिंग). पॉइंट लेबले आढळलेल्या बीट्सचे चिन्हांकित करू शकतात.
25 25 The label text can be edited and you can resize region labels or move region or point labels. लेबल मजकूर संपादित केला जाऊ शकतो आणि आपण प्रदेश लेबलांचा आकार बदलू शकता किंवा प्रदेश किंवा बिंदू लेबल हलवू शकता.
26 26 The Label Track's Dropdown Menu can be used to name the track, move it up or down or set the font of all label text. लेबल ट्रॅकच्या ड्रॉपडाउन मेनूचा उपयोग ट्रॅकचे नाव ठेवण्यासाठी किंवा खाली हलविण्यासाठी किंवा सर्व लेबल मजकूराचा फॉन्ट सेट करण्यासाठी केला जाऊ शकतो.
27 27 Label tracks are included when saving an Audacity Project. ऑडसिटी प्रोजेक्ट जतन करताना लेबल ट्रॅक समाविष्ट केले जातात.
28 28 Label tracks can be used to define sets of Sync-Locked Track Groups. लेबल ट्रॅक सिंक्रोनाइझेशन-लॉक केलेले ट्रॅक ग्रुप्सचे सेट परिभाषित करण्यासाठी वापरले जाऊ शकतात.
29 29 Standard Edit Menu commands can be applied to the audio of multiple labeled regions (excluding audio outside those regions). To do this, drag a selection in the Label Track that fully includes (or extends beyond) the labels whose audio is to be affected, then choose Edit > Labels > Edit Labels... and your required command. एकाधिक लेबल केलेल्या प्रदेशांच्या ऑडिओवर मानक संपादन मेनू आज्ञा लागू केल्या जाऊ शकतात (त्या प्रदेशांच्या बाहेरील ऑडिओ वगळून). हे करण्यासाठी, लेबल ट्रॅकमधील एक निवडी ड्रॅग करा ज्यात ऑडिओवर परिणाम होण्याची शक्यता असलेल्या लेबलांचा पूर्णपणे समावेश आहे (किंवा त्यापलीकडे वाढविला आहे), नंतर संपादित करा> लेबले> लेबल संपादित करा ... आणि आपली आवश्यक आज्ञा निवडा.
30 30 Limitations of Labels लेबलांची मर्यादा
31 31 The text in a label should be less than 260 characters. Although more than 260 characters may be entered, if the project is saved, the label will not be loaded when the project is reopened. लेबलमधील मजकूर 260 वर्णांपेक्षा कमी असावा. 260 वर्णांपेक्षा जास्त वर्ण प्रविष्ट केले जाऊ शकतात, प्रकल्प जतन केल्यास प्रकल्प पुन्हा उघडल्यावर लेबल लोड केले जाणार नाही.
32 32 If there a very large number of labels in a project (hundreds or thousands), Audacity may become noticeably sluggish. प्रोजेक्टमध्ये (शेकडो किंवा हजारो) लेबलांची संख्या खूप असल्यास, ऑडॅसिटी सुस्त होऊ शकेल.
33 33 There is no simple way to convert a "region label" into a "point label". The easiest way is to delete the region label and add a new point label. "प्रांत लेबल" ला "पॉइंट लेबल" मध्ये रूपांतरित करण्याचा कोणताही सोपा मार्ग नाही. प्रदेश लेबल हटविणे आणि नवीन पॉईंट लेबल जोडणे हा सर्वात सोपा मार्ग आहे.
34 34 (Microsoft Windows) Labels do not support IME input. To enter Chinese / Japanese characters into a label, use copy and paste. (मायक्रोसॉफ्ट विंडोज) लेबल आयएमई इनपुटला समर्थन देत नाहीत. लेबलमध्ये चीनी / जपानी वर्ण प्रविष्ट करण्यासाठी, कॉपी आणि पेस्ट वापरा.
35 35 Point labels and region labels बिंदू लेबले आणि प्रदेश लेबले
36 36 Audacity supports labeling points, a single moment in the audio, and labeling regions, a span with a start and end time. ऑडॅसिटी लेबलिंग पॉइंट्स, ऑडिओमधील एक क्षण आणि लेबलिंग क्षेत्रांना प्रारंभ आणि समाप्ती कालावधीसह समर्थन देते.
37 37 A point label is just a region label with the same start and end. पॉईंट लेबल हे समान प्रारंभ आणि अंत असलेले फक्त एक प्रदेश लेबल आहे.
38 38 If you have several region labels one after the other, and each one starts where the previous one ends, they join together. आपल्याकडे एकापाठोपाठ एक प्रांताची अनेक लेबले असल्यास आणि मागील एक जिथे समाप्त होईल तेथे प्रत्येकजण प्रारंभ झाला तर ते एकत्र सामील होतील.
39 39 You can then adjust the end of one label and the start of the next at the same time. त्यानंतर आपण एकाच वेळी एका लेबलचा शेवट आणि पुढील प्रारंभ समायोजित करू शकता.
40 40 Example of a mono audio track with a label track containing a region label and a point label. प्रदेश लेबल आणि बिंदू लेबल असलेल्या लेबल ट्रॅकसह मोनो ऑडिओ ट्रॅकचे उदाहरण.

48 48 Creating and selecting Labels लेबले तयार करणे आणि निवडणे
49 49 Labels can be created by selecting a region or clicking at a point of interest then choosing Edit > Labels > Add Label at Selection (or use its keyboard shortcut Ctrl + B). An empty label appears then you can type to add text to the label. Press Enter on the keyboard to confirm the text and close the label. प्रदेश निवडून किंवा स्वारस्य असलेल्या बिंदूवर क्लिक करून नंतर संपादन> लेबले> निवड येथे लेबल जोडा (किंवा त्याचा कीबोर्ड शॉर्टकट Ctrl + B वापरा) द्वारे लेबले तयार केली जाऊ शकतात. रिक्त लेबल दिसते नंतर आपण लेबलमध्ये मजकूर जोडण्यासाठी टाइप करू शकता. मजकूरची पुष्टी करण्यासाठी कीबोर्डवरील एंटर दाबा आणि लेबल बंद करा.
50 50 When the label track has the yellow focus border as in the image above you can, if preferred, just type to create a label containing that text rather than first using the menu or shortcut to create the label. लेबल ट्रॅकला वरील प्रतिमांप्रमाणे पिवळी फोकस बॉर्डर असेल जेव्हा आपण प्राधान्य दिले तर लेबल तयार करण्यासाठी मेनू किंवा शॉर्टकट वापरण्याऐवजी त्या मजकूरासह एक लेबल तयार करा.
51 51 This default behavior can be turned off, if required, in हे डीफॉल्ट वर्तन आवश्यक असल्यास, बंद केले जाऊ शकते

53 53 You can also create labels while playing or recording. प्ले करताना किंवा रेकॉर्डिंग करताना आपण लेबले देखील तयार करू शकता.
54 54 To create a point label at the current position of the green playback cursor or red recording cursor, choose Edit > Labels > Add Label at Playback Position or its keyboard shortcut Ctrl + M (⌘ + . on Mac). ग्रीन प्लेबॅक कर्सर किंवा लाल रेकॉर्डिंग कर्सरच्या सद्य स्थितीत पॉईंट लेबल तयार करण्यासाठी, संपादन> लेबले> प्लेबॅक स्थानावर लेबल जोडा किंवा त्याचा कीबोर्ड शॉर्टकट Ctrl + M (on +. मॅक वर) निवडा.
55 55 To create a region label while playing or recording, click and drag the region then use the same Add Label at Selection command (or its shortcut Ctrl + B ) as when adding a region label when the track is stopped. Similarly you can click anywhere in a track while playing or recording and use Ctrl + B to label that point. खेळताना किंवा रेकॉर्डिंग करताना प्रदेश लेबल तयार करण्यासाठी, प्रदेश क्लिक करा आणि ड्रॅग करा नंतर ट्रॅक थांबवितांना प्रदेश लेबल जोडताना सेलेक्शन कमांडमधील समान जोडा लेबल वापरा (किंवा त्याचा शॉर्टकट Ctrl + B). त्याचप्रमाणे आपण प्ले करताना किंवा रेकॉर्डिंग करताना ट्रॅकवर कुठेही क्लिक करू शकता आणि त्या बिंदूचे लेबल लावण्यासाठी Ctrl + B वापरू शकता.
56 56 You can create extra label tracks using Tracks > Add New > Label Track but when exporting multiple files based on labels, audio is only exported for the labels in the uppermost label track in the project. आपण ट्रॅक्स> नवीन जोडा> लेबल ट्रॅकचा वापर करून अतिरिक्त लेबल ट्रॅक तयार करू शकता परंतु लेबलवर आधारित एकाधिक फायली निर्यात करताना, प्रोजेक्टमधील सर्वात वरच्या लेबल ट्रॅकमधील लेबलसाठी ऑडिओ केवळ निर्यात केला जातो.
57 57 Selecting a Label एक लेबल निवडत आहे
58 58 When you click inside a label to select it, the label background color changes to white, indicating that the label text is open for editing, and the cursor point or region of audio the label corresponds to is restored. The region will be visible in all audio tracks that are selected and the cursor will be visible in all audio tracks that had the yellow focus border. This applies even if the label track is above the audio track. जेव्हा आपण एखाद्या लेबलच्या निवडीसाठी त्यावर क्लिक करता तेव्हा लेबल पार्श्वभूमीचा रंग पांढरा बदलतो, हे दर्शविते की लेबल मजकूर संपादनासाठी खुला आहे आणि कर्सर बिंदू किंवा ऑडिओचा लेबल ज्या क्षेत्राशी संबंधित आहे तो पुनर्संचयित केला आहे. प्रदेश निवडलेल्या सर्व ऑडिओ ट्रॅकमध्ये दृश्यमान असेल आणि पिवळ्या रंगाची फोकस सीमा असलेल्या सर्व ऑडिओ ट्रॅकमध्ये कर्सर दृश्यमान असेल. हे लेबल ट्रॅक ऑडिओ ट्रॅकच्या वर असला तरीही लागू होते.

65 65 The label background becomes white, indicating you can type to edit the label's text. For a point label, the editing cursor is placed at the label moved to. For a region label, a selection is placed corresponding to the label moved to.
