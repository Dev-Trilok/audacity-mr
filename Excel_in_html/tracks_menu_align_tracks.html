4 4 The Align Tracks submenu provides a set of commands to help you align your selected tracks in various ways. संरेखित ट्रॅक सबमेनू आपल्याला निवडक ट्रॅक विविध प्रकारे संरेखित करण्यात मदत करण्यासाठी कमांडचा एक संच प्रदान करते.
5 5 In particular there are two very useful commands that enable you to align tracks end-to-end, one after the other, or align them all together. विशेषतः दोन अतिशय उपयुक्त कमांड्स आहेत जे आपल्याला मागोमाग एक मागून एक ट्रॅक संरेखित करण्यास सक्षम करतात किंवा त्या सर्व एकत्र संरेखित करतात.
6 6 Click on the image to learn more अधिक जाणून घेण्यासाठी प्रतिमेवर क्लिक करा
7 7 These commands provide an automatic way of aligning selected tracks with the cursor, the selection, or with the start of the project. हे आदेश कर्सर, निवड किंवा प्रोजेक्टच्या सुरूवातीस निवडलेल्या ट्रॅक संरेखित करण्याचा स्वयंचलित मार्ग प्रदान करतात.
8 8 With the exception of the first two commands ("Align End to End" and "Align Together"), each selected track is moved along equally so they remain synchronized with each other. If a label track is included in the selection, its labels will also move equally so they remain synchronized. पहिल्या दोन कमांडस वगळता ("अलाइन टू एंड एंड" आणि "एकत्र संरेखित करा"), प्रत्येक निवडलेला ट्रॅक समान रीतीने हलविला गेला आहे जेणेकरून ते एकमेकांशी समक्रमित राहतील. लेबल ट्रॅक निवडीमध्ये समाविष्ट केल्यास त्याची लेबले देखील तितकीच हलतील जेणेकरून ते संकालित होतील.
9 9 When aligning end to end or aligning together, the tracks are always moved individually to enforce the requested alignment (as described below). Label tracks are not affected by these two commands. शेवट संरेखित करताना किंवा एकत्र संरेखित करताना, विनंती केलेले संरेखन (खाली वर्णन केल्यानुसार) अंमलात आणण्यासाठी ट्रॅक नेहमीच स्वतंत्रपणे हलवले जातात. या दोन आदेशांद्वारे लेबल ट्रॅकवर परिणाम होणार नाही.
10 10 When there is a selection, the commands that align tracks with the cursor/selection will align with the start of the selection, and when there is a cursor, the commands that align tracks with the cursor/selection will align with the cursor. जेव्हा सिलेक्शन असेल तेव्हा कर्सर / सेलेक्शनद्वारे ट्रॅक संरेखित करणार्‍या कमांडस सिलेक्शनच्या सुरूवातीस संरेखित होतील आणि जेव्हा कर्सर असेल तेव्हा कर्सर / सिलेक्शनद्वारे ट्रॅक संरेखित करणार्‍या कमांड्स कर्सरसह संरेखित होतील.
11 11 If Sync-Lock Tracks is enabled, unselected tracks in a Sync-Locked Track Group will also move so that all the tracks in that group remain synchronized, except for the Align "End to End" and "Together" commands where only the selected tracks will move. Aligning end to end or together overrides Sync-Lock Tracks if necessary, so the selected tracks in that group may become desynchronized. संकालन-लॉक ट्रॅक सक्षम केलेले असल्यास, संकालित-लॉक केलेला ट्रॅक गटातील निवड रद्द केलेले ट्रॅक देखील हलतील जेणेकरून त्या गटातील सर्व ट्रॅक संकालित राहतील, "एंड टू एंड" संरेखित करा आणि फक्त निवडलेले ट्रॅक हलवेल समाप्तीची समाप्ती संरेखित करणे किंवा एकत्रितपणे आवश्यक असल्यास समक्रमण-लॉक ट्रॅक अधिलिखित करते, जेणेकरून त्या गटामधील निवडलेले ट्रॅक विभाजित होऊ शकतात.
12 12 If a track contains multiple clips and one of the Align commands is applied to that track then all the clips move and maintain their relative positions to each other within the track. The start of the first clip is the "start of the track" and the end of the last clip is the "end of the track". एखाद्या ट्रॅकमध्ये एकाधिक क्लिप्स असतील आणि त्या ट्रॅकवर संरेखित आज्ञा लागू केली गेली असेल तर सर्व क्लिप ट्रॅकमध्ये एकमेकांशी संबंधित स्थाने हलवतात आणि देखरेख करतात. पहिल्या क्लिपची सुरूवात "ट्रॅकची सुरुवात" असते आणि शेवटच्या क्लिपचा शेवट "ट्रॅकचा शेवट" असतो.
13 13 Align End to End समाप्तीस समाप्ती संरेखित करा
14 14 Aligns the selected tracks one after the other, based on their top-to-bottom order in the project window. The first selected track remains where it is, the start of the second selected track aligns with the end of the first selected track and so on. If you name a group of imported files from an album so that their alphabetical order is the same as the order you want to align them in, Align End to End will make the imported files play in their album order. निवडलेले ट्रॅक एकामागून एक प्रोजेक्ट विंडोमध्ये त्यांच्या टॉप-ते-डाउन ऑर्डरच्या आधारावर संरेखित करते. पहिला निवडलेला ट्रॅक जिथे आहे तिथेच राहतो, दुस selected्या निवडलेल्या ट्रॅकची सुरूवात पहिल्या निवडलेल्या ट्रॅकच्या शेवटी संरेखित होते आणि याप्रमाणे. आपण अल्बममधून आयात केलेल्या फायलींच्या एका गटास नाव दिले जेणेकरून त्यांची वर्णमाला क्रमानुसार आपण त्यांना संरेखित करू इच्छित असलेल्या ऑर्डर प्रमाणेच असल्यास, संरेखित करा एंड टू एंड फाइल आयात केलेल्या फायली त्यांच्या अल्बमच्या क्रमानुसार प्ले करेल.
15 15 After the align, all tracks in the project (not just the selected tracks) will be fitted to the window so that the end of the last track will be visible in the project window. संरेखनानंतर, प्रकल्पातील सर्व ट्रॅक (केवळ निवडलेले ट्रॅकच नाहीत) विंडोमध्ये बसविण्यात येतील जेणेकरून शेवटच्या ट्रॅकचा शेवट प्रकल्प विंडोमध्ये दिसून येईल.
16 16 Before doing Tracks > Align Tracks > Align End to End ट्रॅक करण्यापूर्वी> ट्रॅक संरेखित करा> संरेखित करा शेवट
17 17 After doing Tracks > Align Tracks > Align End to End ट्रॅक केल्यावर> ट्रॅक संरेखित करा> संरेखित करा शेवट
18 18 Note that zoom level has changed to display the entire project, and that the original selection has been retained. संपूर्ण प्रकल्प प्रदर्शित करण्यासाठी झूम पातळी बदलली आहे आणि मूळ निवड कायम ठेवली आहे हे लक्षात घ्या.
19 19 Align Together एकत्र संरेखित करा
20 20 If you select multiple tracks, this command moves the tracks so that they all start at the same time, so audio within them changes its synchronization. The resultant start time is an average of all of the tracks' original starting times. To adjust the starting times of tracks with complete flexibility, use Time Shift Tool. आपण एकाधिक ट्रॅक निवडल्यास, ही आज्ञा ट्रॅक हलवते जेणेकरून ते सर्व एकाच वेळी प्रारंभ होतील, म्हणून त्यांच्यामधील ऑडिओ त्याचे संकालन बदलू शकेल. परिणामी प्रारंभ वेळ सर्व ट्रॅकच्या मूळ सुरूवातीच्या वेळेची सरासरी असते. पूर्ण लवचिकतेसह ट्रॅकच्या सुरूवातीच्या वेळेस समायोजित करण्यासाठी, वेळ शिफ्ट टूल वापरा.
21 21 Before doing Tracks > Align Tracks > Align Together ट्रॅक करण्यापूर्वी> ट्रॅक संरेखित करा> एकत्र संरेखित करा
22 22 After doing Tracks > Align Tracks > Align Together ट्रॅक केल्यावर> ट्रॅक संरेखित करा> एकत्र संरेखित करा
23 23 Note that the three tracks now start at the same time, and the start time is the average of the original start times of the three tracks. लक्षात घ्या की तीन ट्रॅक आता एकाच वेळी प्रारंभ होतात आणि प्रारंभ वेळ तीन ट्रॅकच्या मूळ प्रारंभ वेळेची सरासरी आहे.
24 24 Start to Zero झिरोला प्रारंभ करा
25 25 Aligns the start of selected tracks with the start of the project. प्रोजेक्टच्या सुरूवातीस निवडलेल्या ट्रॅकची सुरूवात संरेखित करते.
26 26 Start to Cursor/Selection Start कर्सर प्रारंभ / निवड प्रारंभ
27 27 Aligns the start of selected tracks with the current cursor position or with the start of the current selection. सध्याच्या कर्सर स्थानासह किंवा सद्य निवडीच्या सुरूवातीसह निवडलेल्या ट्रॅकची सुरूवात संरेखित करते.
28 28 Start to Selection End सिलेक्शन एंड टू सिलेक्शन एन्ड
29 29 Aligns the start of selected tracks with the end of the current selection. सद्य निवडीच्या समाप्तीसह निवडलेल्या ट्रॅकचा प्रारंभ संरेखित करते.
30 30 End to Cursor/Selection Start कर्सर / निवड प्रारंभ
31 31 Aligns the end of selected tracks with the current cursor position or with the start of the current selection. सध्याच्या कर्सर स्थानासह किंवा सद्य निवडीच्या सुरूवातीसह निवडलेल्या ट्रॅकचा शेवट संरेखित करते.
32 32 End to Selection End निवडीचा शेवट
33 33 Aligns the end of selected tracks with the end of the current selection. सद्य निवडीच्या समाप्तीसह निवडलेल्या ट्रॅकचा शेवट संरेखित करते.
34 34 Move Selection with Tracks (on/off) निवड ट्रॅकसह हलवा (चालू / बंद)
35 35 When this option is off (not checked), the cursor or selection remains in its position relative to the Timeline. When it is on, the cursor or selection is moved to maintain its position relative to the tracks being moved. For example, Tracks >Align Tracks > Start to Cursor/Selection Start will align the selected tracks with the current cursor position or selection start, but then also move the cursor or selection along so that it remains at the same position relative to the new position of the aligned tracks. जेव्हा हा पर्याय बंद असतो (चेक केलेला नाही), तेव्हा टाइमलाइनच्या तुलनेत कर्सर किंवा निवड त्याच्या स्थितीत राहील. ते चालू असताना, ट्रॅक हलविण्याशी संबंधित नसल्यास कर्सर किंवा निवड त्याचे स्थान राखण्यासाठी हलविली जाते. उदाहरणार्थ, ट्रॅक> ट्रॅक संरेखित करा> प्रारंभ करणे कर्सर / निवड प्रारंभ निवडलेले ट्रॅक वर्तमान कर्सर स्थानासह किंवा निवड प्रारंभसह संरेखित करेल, परंतु नंतर कर्सर किंवा निवड देखील हलवेल जेणेकरून ते नवीन स्थानाशी संबंधित त्याच स्थितीत राहील. संरेखित ट्रॅकचे.
36 36 This option has no effect on the Align End to End and the Align Together commands. संरेखित समाप्तीची समाप्ती आणि संरेखित एकत्र आदेशांवर या पर्यायाचा कोणताही परिणाम होणार नाही.
