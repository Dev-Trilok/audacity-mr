10 10 MP3 Export Library Version: This shows the current version of the LAME MP3 encoding library being used by Audacity for MP3 export. If a suitable version of LAME is not accessible to Audacity, the message will instead state that the library is "not found". In that case, use the provided buttons to either locate LAME on your computer or download a free copy of it, following the instructions on the FAQ:Installation, Startup and Plugins page. एमपी 3 एक्सपोर्ट लायब्ररी आवृत्तीः हे एमपी 3 एक्सपोर्टसाठी ओड्यासिटी द्वारे वापरल्या जाणार्‍या लेम एमपी 3 एन्कोडिंग लायब्ररीची सद्य आवृत्ती दर्शविते. जर LAME ची योग्य आवृत्ती ओड्यासिटीमध्ये प्रवेशयोग्य नसेल तर संदेश त्याऐवजी लायब्ररी "सापडली नाही" असे नमूद करेल. अशा परिस्थितीत, प्रदान केलेल्या बटणांचा वापर एकतर आपल्या संगणकावरील लॅमे शोधण्यासाठी किंवा FAQ: स्थापना, प्रारंभ आणि प्लगइन्स पृष्ठावरील सूचनांचे अनुसरण करून त्याची एक विनामूल्य प्रत डाउनलोड करा.

13 13 FFmpeg Import/Export Library एफएफम्पेग आयात / निर्यात लायब्ररी
14 14 FFmpeg Library Version: This shows the current version of the FFmpeg library being used by Audacity for custom export. If a suitable version of FFmpeg is not accessible to Audacity, the message will instead state that the library is "not found". In that case, use the provided buttons to either locate the FFmpeg library on your computer or download a free copy of it, following the instructions on this page. FFmpeg ग्रंथालय आवृत्ती: हे FFmpeg लायब्ररीची सद्य आवृत्ती कस्टम एक्सपोर्टसाठी ओड्यासिटी द्वारे वापरली जात असल्याचे दर्शविते. जर एफएफम्पेगची योग्य आवृत्ती ओड्यासिटीवर प्रवेशयोग्य नसेल तर संदेश त्याऐवजी लायब्ररी "सापडली नाही" असे नमूद करेल. अशा परिस्थितीत, या पृष्ठावरील सूचनांचे अनुसरण करून आपल्या संगणकावर एफएफएमपीईजी लायब्ररी शोधण्यासाठी किंवा त्यावरील विनामूल्य प्रत डाउनलोड करण्यासाठी प्रदान केलेल्या बटणे वापरा.
