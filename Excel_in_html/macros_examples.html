23 23 When later needed in your workflow, choose Tools > Apply Macro..., select the "NR&EQ" Macro then click "Apply Macro to:" Project to apply the Macro to the selected track(s) in the current project window. नंतर आपल्या वर्कफ्लोमध्ये आवश्यक असल्यास, साधने> मॅक्रो लागू करा ... निवडा, "एनआर अँड ईक्यू" मॅक्रो निवडा त्यानंतर सध्याच्या प्रकल्प विंडोमध्ये निवडलेल्या ट्रॅकवर मॅक्रो लागू करण्यासाठी प्रोजेक्ट क्लिक करा.
24 24 If a Noise Profile exists, that Noise Profile will be used. It is often best to capture a suitable Noise Profile before running a Macro. जर ध्वनी प्रोफाइल अस्तित्वात असेल तर ते आवाजाचे प्रोफाइल वापरले जाईल. मॅक्रो चालविण्यापूर्वी योग्य आवाजाचे प्रोफाइल हस्तगत करणे चांगले.
25 25 If a Noise Profile does not exist: ध्वनी प्रोफाइल अस्तित्वात नसल्यास:
26 26 If the Macro is applied to the current project (as above), the current selection is used to create the Noise Profile. Therefore, other effect commands in the Macro will also only apply to that selection. If an export command is added, the entire file will be exported. सध्याच्या प्रकल्पावर मॅक्रो लागू केला असल्यास (वरील प्रमाणे), सद्य निवडीचा उपयोग नॉइस प्रोफाइल तयार करण्यासाठी केला जातो. म्हणूनच, मॅक्रो मधील इतर प्रभाव आदेश केवळ त्या निवडीस लागू होतील. निर्यात आदेश जोडल्यास, संपूर्ण फाइल निर्यात केली जाईल.
27 27 If the Macro is applied to files, the first file (all of it) is used to create the Noise Profile. It may be useful to prepare a file containing a suitable Noise Profile and name it so that it is alphabetically the first file of those to be run in the Macro. जर फायलींवर मॅक्रो लागू केला गेला असेल तर प्रथम फाईल (त्या सर्व) नॉइस प्रोफाइल तयार करण्यासाठी वापरली जाते. योग्य ध्वनी प्रोफाइल असलेली फाईल तयार करणे आणि त्यास नावे ठेवणे उपयुक्त ठरेल जेणेकरून मॅक्रोमध्ये चालविल्या जाणार्‍या त्यातील वर्णमाला वर्णक्रमानुसार असेल.
28 28 Useful Commands उपयुक्त आज्ञा
29 29 The 'Select' command with 'Relative To=Selection' can be used to extend and contract a selection. 'रिलेटिव टू = सेलेक्शन' सह 'सिलेक्ट' कमांडचा उपयोग निवड वाढवण्यासाठी आणि कॉन्ट्रॅक्ट करण्यासाठी केला जाऊ शकतो.
30 30 Command आज्ञा
31 31 Description वर्णन
32 32 "Select: RelativeTo=Selection Start=-1 End=1" "निवडा: रिलेटिव्ह टू = सेलेक्शन स्टार्ट = -1 एंड = 1"
33 33 This command expands a selection by two seconds: ही कमांड दोन सेकंदाने निवड विस्तृत करते:
34 34 "Select: RelativeTo=Selection Start=1 End=-1" "निवडा: रिलेटिव्ह टू = सिलेक्शन स्टार्ट = 1 एंड = -1"
35 35 This command contracts a selection by two seconds: ही कमांड दोन सेकंदाने निवडीची कंत्राट घेते:
36 36 "Select: RelativeTo=Selection Start=1 End=1" "निवडा: रिलेटिव्ह टू = निवड प्रारंभ = 1 समाप्त = 1"
37 37 This command moves a selection right by one second: ही कमांड सिलेक्शनला एका सेकंदाने उजवीकडे हलवते:

44 44 Spectral Magickes Wacky-Macro स्पेक्ट्रल मॅजिक्स वकी-मॅक्रो
45 45 This macro takes a single mono track, and converts it to a stereo track, with one channel a spectrogram and the other a wave. हा मॅक्रो एकल मोनो ट्रॅक घेते आणि एका चॅनेलसह एक स्पेक्ट्रोग्राम आणि दुसर्‍या लाटेत स्टिरिओ ट्रॅकमध्ये रुपांतरित करतो.
46 46 It is one example of how to creatively abuse the Macro system, since audacity is not set up to handle mixed views on wave tracks. वेव्ह ट्रॅकवर मिश्रित दृश्ये हाताळण्यासाठी धृष्टता तयार केलेली नसल्यामुळे मॅक्रो सिस्टमचा क्रिएटिव्ह दुरुपयोग कसा करावा याचे हे एक उदाहरण आहे.
47 47 Note the use of a TrackCount of 0.5 to select just one of two channels in a stereo track. स्टिरिओ ट्रॅकमधील दोनपैकी फक्त एक चॅनेल निवडण्यासाठी ०.० च्या ट्रॅककॉन्टचा वापर लक्षात घ्या.
48 48 The TrackCount of 0 is used to unselect all tracks. सर्व ट्रॅकची निवड रद्द करण्यासाठी 0 चा ट्रॅककउंट वापरला जातो.
