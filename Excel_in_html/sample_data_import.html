1 1 Sample Data Import नमुना डेटा आयात
2 2 From Audacity Development Manual ऑडसिटी डेव्हलपमेंट मॅन्युअल वरुन
3 3 Jump to: navigation, search येथे जा: सुचालन, शोध
4 4 Sample Data Import reads numeric values from a plain ASCII text file and creates a PCM sample for each numeric value read. The values are read as 32-bit floating point numbers, where a range of +/- 1.0 represents 0 dB (full scale). नमुना डेटा आयात साध्या एएससीआयआय मजकूर फाईलमधून संख्यात्मक मूल्ये वाचतो आणि प्रत्येक अंकीय मूल्य वाचनासाठी पीसीएम नमुना तयार करतो. मूल्ये 32-बिट फ्लोटिंग पॉईंट क्रमांक म्हणून वाचली जातात, जिथे +/- 1.0 ची श्रेणी 0 डीबी (पूर्ण स्केल) दर्शवते.
5 5 The file MUST be plain ASCII text, and values should be separated by spaces, tabs or line-breaks. फाईल साधा ASCII मजकूर असणे आवश्यक आहे आणि मूल्ये रिक्त स्थान, टॅब किंवा लाइन-ब्रेकद्वारे विभक्त केल्या पाहिजेत.
6 6 The file name must have a '.txt' file extension. फाईल नावामध्ये '.txt' फाईल विस्तार असणे आवश्यक आहे.
7 7 See also the companion tool Sample Data Export. सहचर साधन नमुना डेटा निर्यात देखील पहा.
8 8 Accessed by: Tools > Sample Data Import... याद्वारे प्रवेश केलेलेः साधने> नमुना डेटा आयात ...
9 9 Select file फाईल निवडा
10 10 Use this to select a file name and file location to be used for the Sample Data Import नमुना डेटा आयात करण्यासाठी वापरण्यासाठी फाइल नाव आणि फाइल स्थान निवडण्यासाठी याचा वापर करा
11 11 File name फाईलचे नाव
12 12 The name of the file to import the data from. A file name extension of .txt is required. वरून डेटा आयात करण्यासाठी फाईलचे नाव. .Txt चा फाइल नाव विस्तार आवश्यक आहे.
13 13 The file to be imported must contain only plain ASCII text. Files other than plain ASCII text are NOT SUPPORTED. आयात करण्यासाठी फाइलमध्ये फक्त साधा ASCII मजकूर असणे आवश्यक आहे. साध्या एएससीआयआय मजकूरशिवाय इतर फायली समर्थित नाहीत.
14 14 File location (path to file) फाइल स्थान (फाईलचा मार्ग)
15 15 This is the directory (folder) where Sample Data Import will look for the named text file. If the file cannot be found, the plug-in will abort and an error will be shown. ही निर्देशिका (फोल्डर) आहे जेथे नमुना डेटा आयात नामित मजकूर फाईल शोधेल. फाईल सापडली नाही तर प्लग-इन रद्द होईल आणि एक त्रुटी दर्शविली जाईल.
16 16 Default file name and path डीफॉल्ट फाइल नाव आणि पथ
17 17 The default filename/filepath is normally: डीफॉल्ट फाइलनाव / फाइलपथ सामान्यत: असते:
18 18 Windows: C:\Users\<username>\Documents\Audacity\sample-data.txt विंडोज: सी: \ वापरकर्ते \ <वापरकर्तानाव> u दस्तऐवज \ ऑडॅसिटी \ नमुना-डेटा.txt
19 19 Mac: /Users/<username>/Documents/sample-data.txt मॅक: / वापरकर्ते / वापरकर्तानाव> / दस्तऐवज / नमुना- डेटा.टी.एस.टी.
20 20 Linux: /home/<username>/Documents/sample-data.txt लिनक्स: /home/<username>/Documents/sample-data.txt
21 21 Using the Select a file button is the recommended way to select the input file. The file path text may also be typed or edited manually, though care needs to be taken to avoid errors. फाइल निवडा बटण वापरणे इनपुट फाइल निवडण्याचा सूचविलेला मार्ग आहे. फाइल पथ मजकूर स्वहस्ते टाइप किंवा संपादित देखील केला जाऊ शकतो, तरीही त्रुटी टाळण्यासाठी काळजी घेणे आवश्यक आहे.
22 22 Invalid data handling अवैध डेटा हाताळणी
23 23 The specified text file should normally contain numbers only (see: Data Format below). If invalid data is present in the file (non-numeric text), there are two ways in which this invalid data may be handled: निर्दिष्ट मजकूर फाईलमध्ये सामान्यत: फक्त संख्या असते (पहा: खाली डेटा स्वरूप). अवैध डेटा फाइलमध्ये असल्यास (संख्यात्मक मजकूर) नसल्यास, दोन अवैध मार्गांनी हा अवैध डेटा हाताळला जाऊ शकतोः
24 24 Throw error (default): थ्रो त्रुटी (डीफॉल्ट):
25 25 Any text that cannot be read as a numeric value will abort the effect and display an error message. The error message will attempt to display the invalid text. अंकीय मूल्य म्हणून वाचता येत नाही असा कोणताही मजकूर परिणाम रद्द करेल आणि एक त्रुटी संदेश प्रदर्शित करेल. त्रुटी संदेश अवैध मजकूर प्रदर्शित करण्याचा प्रयत्न करेल.
26 26 Read as zero: शून्य म्हणून वाचा:
27 27 Any text that cannot be read as a numeric value will produce a zero (silent) sample value. This may be useful when attempting to read data that has been generated by a third party application that could, for example, produce "infinite" values. अंकीय मूल्य म्हणून वाचता येणार नाही असा मजकूर शून्य (मूक) नमुना मूल्य तयार करेल. तृतीय पक्षाच्या अनुप्रयोगाद्वारे व्युत्पन्न केलेला डेटा वाचण्याचा प्रयत्न करताना हे उपयुक्त ठरेल, उदाहरणार्थ, "असीम" मूल्ये निर्माण करू शकेल.
28 28 Data format डेटा स्वरूप
29 29 Sample data must be plain ASCII text only and should be numeric values only. The decimal separator must be a period. The values are read as 32-bit float numbers, where a range of +/- 1.0 represents 0 dB (full scale). नमुना डेटा फक्त साधा ASCII मजकूर असणे आवश्यक आहे आणि केवळ संख्यात्मक मूल्ये असावी. दशांश विभाजक एक कालावधी असणे आवश्यक आहे. मूल्ये 32-बिट फ्लोट संख्या म्हणून वाचली जातात, जिथे +/- 1.0 ची श्रेणी 0 डीबी (पूर्ण प्रमाणात) दर्शवते.
30 30 Values must be separated by one of: मूल्ये यापैकी एकाने विभक्त करणे आवश्यक आहे:
31 31 One or more spaces. एक किंवा अधिक जागा
32 32 One or more tabs. एक किंवा अधिक टॅब.
33 33 One or more line breaks. एक किंवा अधिक लाइन खंडित.
34 34 Any combination of the above. वरीलपैकी कोणतेही संयोजन.
35 35 Assuming that no errors are encountered, data import terminates at the end of the file. कोणत्याही त्रुटी आल्या नाहीत असे गृहित धरून डेटा आयात फाईलच्या शेवटी संपते.
36 36 Number of channels वाहिन्यांची संख्या
37 37 If a stereo track is selected before opening the effect, then the data is assumed to be stereo, otherwise the data will be treated as mono. जर प्रभाव उघडण्यापूर्वी स्टिरीओ ट्रॅक निवडला गेला असेल तर डेटा स्टिरीओ असा गृहित धरला जाईल, अन्यथा डेटा मोनो म्हणून गणला जाईल.
38 38 Stereo data must be interleaved (alternate left/right sample values). स्टिरीओ डेटा इंटरलीव्ह केलेला असणे आवश्यक आहे (वैकल्पिक डावे / उजवे नमुना मूल्य).
39 39 Data comments डेटा टिप्पण्या
40 40 Comments may be included in the file by preceding the comment with a semicolon. All text between a semicolon and the end of the line is ignored. Files already exported from "Sample Data Export" that are compatible with Sample Data Import except for their text headers can thus be imported perfectly with the default "Throw error" handling by preceding each header line with a semicolon. अर्धविराम सह टिप्पणी अगोदर फाइल मध्ये टिप्पण्या समाविष्ट केल्या जाऊ शकतात. अर्धविराम आणि रेषाच्या शेवटी मधील सर्व मजकूर दुर्लक्षित केला जाईल "नमुना डेटा निर्यात" वरून आधीच निर्यात केलेल्या फायली ज्या त्यांच्या नमुना शीर्षलेख वगळता नमुना डेटा आयातशी सुसंगत आहेत अशा प्रकारे अर्धविरामसह प्रत्येक शीर्षलेख ओळीच्या आधी डीफॉल्ट "थ्रो त्रुटी" हाताळणीसह उत्तम प्रकारे आयात केली जाऊ शकते.
41 41 Obtaining suitable data योग्य डेटा मिळवत आहे
42 42 One of the simplest ways to produce data in a suitable format is to use the Sample Data Export effect. Alternatively, data may be generated from third party applications, from spreadsheet data, or even manually typed in a text editor. सॅम्पल डेटा एक्सपोर्ट इफेक्टचा वापर करणे योग्य स्वरुपात डेटा तयार करण्याचा सर्वात सोपा मार्ग आहे. वैकल्पिकरित्या, तृतीय पक्षाच्या अनुप्रयोगांमधून डेटा व्युत्पन्न केला जाऊ शकतो स्प्रेडशीट डेटामधून किंवा मजकूर संपादकात स्वहस्ते टाइप केला जाऊ शकतो.
43 43 Obtain data from 'Sample Data Export' 'नमुना डेटा निर्यात' वरून डेटा मिळवा
44 44 To obtain suitable data from Sample Data Export, ensure the following settings: नमुना डेटा निर्यात वरून योग्य डेटा मिळविण्यासाठी, खालील सेटिंग्जची खात्री करा:
45 45 Measurement scale: "Linear" मापन मोजमाप: "रेखीय"
46 46 File data format: "Sample List (txt)" फाईल डेटा स्वरूप: "नमुना यादी (टेक्स्ट)"
47 47 Include header information: "None" शीर्षलेख माहिती समाविष्ट करा: "काहीही नाही"
48 48 Optional header text: none (empty) पर्यायी शीर्षलेख मजकूर: काहीही नाही (रिक्त)
49 49 Channel layout for stereo: "L-R on Same Line" or "Alternate Lines" स्टिरीओसाठी चॅनेल लेआउट: "समान रेषावरील एल-आर" किंवा "वैकल्पिक रेखा"
50 50 File name: must end with ".txt" or leave empty for ".txt" to be added automatically फाइल नाव: ".txt" ने समाप्त होणे आवश्यक आहे किंवा ".txt" स्वयंचलितपणे जोडण्यासाठी रिक्त सोडा
