13 13 Sending Projects to others इतरांना प्रकल्प पाठवित आहे
14 14 It's quite possible to send an Audacity Project to another person or open it on another computer of your own, so that the Project can be opened in another copy of Audacity with all the tracks, label and envelope information in situ. However when opening the AUP Project file you need to have its associated _data folder with the same name as the AUP file present in the same folder. For example, to send a Project to someone else you could send the AUP file and _data folder in a compressed .zip archive. Free Windows tools to create .zip archives include 7-Zip or IZArc. Free Mac tools include Keka or Apple's built-in compression utility. Zip compression is lossless, so does not reduce the space on disk very much. Ask your recipient to extract both the AUP file and _data folder to the same directory then open the AUP file from inside that folder. दुसर्या व्यक्तीला ऑडसिटी प्रोजेक्ट पाठविणे किंवा आपल्या स्वत: च्या दुसर्‍या संगणकावर उघडणे शक्य आहे, जेणेकरून प्रोजेक्ट ऑडसीटीच्या दुसर्‍या प्रतात सर्व ट्रॅक, लेबल आणि लिफाफ्याच्या माहितीसह परिस्थितीत उघडता येईल. तथापि एओपी प्रोजेक्ट फाईल उघडताना आपणास त्याचे संबंद्ध _डेटा फोल्डर त्याच फोल्डरमध्ये एयूपी फाइलच्या समान नावाने असणे आवश्यक आहे. उदाहरणार्थ, एखाद्या दुसर्‍यास प्रोजेक्ट पाठविण्यासाठी आपण संकुचित .zip संग्रहणामध्ये AUP फाइल आणि _डेटा फोल्डर पाठवू शकता. .Zip संग्रहण तयार करण्यासाठी विनामूल्य Windows साधनांमध्ये 7-झिप किंवा IZArc चा समावेश आहे. विनामूल्य मॅक टूल्समध्ये केका किंवा Appleपलची अंगभूत कॉम्प्रेशन युटिलिटी समाविष्ट आहे. झिप कॉम्प्रेशन हानिरहित आहे, म्हणून डिस्कवरील जागा फारच कमी करत नाही. आपल्या प्राप्तकर्त्यास समान निर्देशिका मध्ये AUP फाइल आणि _डेटा फोल्डर दोन्ही काढण्यास सांगा आणि त्या फोल्डरच्या आत AUP फाइल उघडा.
15 15 In practice, the problem with sending such a folder to someone else is the size of the _data folder. At Audacity's default 32-bit float sample format / 44100 Hz sample rate using lossless uncompressed audio, stereo Projects take 20 MB of space per minute, which rules out sending the project by email. सराव मध्ये, असे फोल्डर दुसर्‍या कोणास पाठविण्यातील अडचण म्हणजे _डेटा फोल्डरचा आकार. ऑडॅसिटीच्या डीफॉल्ट 32-बिट फ्लोट नमुना स्वरूपात / 44100 हर्ट्ज नमुना दर लॉलेसलेस कॉम्प्रप्रेसित ऑडिओ वापरुन, स्टीरिओ प्रोजेक्ट्स प्रति मिनिट 20 एमबी जागा घेतात, जे ईमेलद्वारे प्रोजेक्ट पाठविण्यास नकार देतात.
16 16 To send your project, first close it (saving changes) so that Audacity can discard the surplus data files which allow undo and redo of edits while projects are open. This way, only the audio for the current state of the project is included. If you forget to close the project before zipping, Audacity will complain about these surplus files, calling them "orphan block files". If this happens, the safest course is to tell Audacity to "continue without deleting" the orphans, at least until you are sure the audio of the project is correct. आपला प्रकल्प पाठविण्यासाठी प्रथम ते बंद करा (बदल जतन करा) जेणेकरून ऑडसिटी अतिरेकी डेटा फायली टाकू शकेल ज्या प्रकल्पांना उघडे असताना संपादन पूर्ववत आणि पुन्हा करण्याची परवानगी देतील. अशाप्रकारे, प्रकल्पाच्या सद्य स्थितीसाठी केवळ ऑडिओ समाविष्ट केला आहे. आपण झिप करण्यापूर्वी प्रकल्प बंद करणे विसरल्यास ऑडसिटी या अतिरीक्त फायलींबद्दल तक्रार करेल, त्यांना "अनाथ ब्लॉक फायली" म्हणत असेल. असे झाल्यास, सर्वात सुरक्षित मार्ग म्हणजे ऑडसेटला अनाथांना "हटविल्याशिवाय पुढे जा" असे सांगावे लागेल, जोपर्यंत आपल्याला खात्री नाही की प्रकल्पातील ऑडिओ बरोबर आहे.
17 17 Then use a zip application like those above to place the AUP file and _data folder inside a zip file. Then find a suitable free internet file transfer service to send your zip file. Recommendable services include the following. Check the limitations for free accounts on size and number of uploads, maximum permitted number of downloads, and length of time files are stored for. त्यानंतर झिप फाईलमध्ये एयूपी फाइल आणि _डेटा फोल्डर ठेवण्यासाठी वरीलप्रमाणे झिप अनुप्रयोग वापरा. नंतर आपली झिप फाइल पाठविण्यासाठी एक योग्य विनामूल्य इंटरनेट फाइल ट्रान्सफर सेवा शोधा. शिफारस केलेल्या सेवांमध्ये पुढील गोष्टींचा समावेश आहे. आकार आणि अपलोडवरील संख्येवर विनामूल्य खात्यांसाठी मर्यादा तपासा, डाउनलोडची जास्तीत जास्त परवानगी संख्या आणि फायली ज्या कालावधीसाठी संग्रहित आहेत त्या तपासा.

24 24 Don't forget old-fashioned solutions too if you are still on slow dial-up internet. You can always combine your AUP Project file and _data folder in a .zip file, burn this to a data CD with Windows Media Player or similar then send it by postal mail. आपण अद्याप स्लो डायल-अप इंटरनेट वर असल्यास जुन्या पद्धतीची निराकरणे देखील विसरू नका. आपण आपल्या एओपी प्रोजेक्ट फाईल आणि _डाटा फोल्‍डरला नेहमीच एक .zip फाईलमध्ये एकत्र करू शकता, विंडोज मीडिया प्लेयर किंवा तत्सम असलेल्या डेटा सीडीमध्ये बर्न करा किंवा पोस्टल मेलद्वारे पाठवा.
25 25 You can produce a small, compressed project for distribution to others using File > Save Project > Save Compressed Copy of Project.... आपण फाईल> सेव्ह प्रोजेक्ट> प्रोजेक्टची संकुचित प्रत जतन करा .... वापरून इतरांना वितरणासाठी एक छोटा, संकुचित प्रकल्प तयार करू शकता.
26 26 The audio in the _data folder is in the form of one OGG file per track of the project. _डेटा फोल्डरमधील ऑडिओ प्रोजेक्टच्या प्रत्येक ट्रॅकवर एक ओजीजी फाईलच्या रूपात आहे.
27 27 The OGG files are very slightly lossy, whereas when saving a project by File > Save Project > Save Project or File > Save Project > Save Project As... the .au files in the _data folder are lossless. ओजीजी फायली खूपच हानीकारक असतात, परंतु फाइल> प्रकल्प जतन करा> प्रकल्प जतन करा किंवा फाइल जतन करा> प्रकल्प जतन करा> प्रकल्प म्हणून जतन करा ... _डाटा फोल्डरमधील .au फाइल्स लॉसलेस आहेत.
28 28 Dependencies on other audio files इतर ऑडिओ फायलींवर अवलंबन
29 29 If you save a standard uncompressed project which contains audio imported into Audacity from a WAV or AIFF file, you must have previously set Audacity to make a copy of that original audio when you imported it, so that it's included in the _data folder. आपण एक प्रमाणित संकुचित प्रकल्प जतन केले ज्यामध्ये डब्ल्यूएव्ही किंवा एआयएफएफ फाईलमधून ऑडसिटीमध्ये ऑडिओ आयात केलेला असेल तर आपण आयात केल्यावर त्या मूळ ऑडिओची प्रत तयार करण्यासाठी आपण यापूर्वी ऑडॅसीटी सेट केली पाहिजे, जेणेकरून ते _डेटा फोल्डरमध्ये समाविष्ट असेल.
30 30 Note that when importing compressed audio like MP3 or OGG, Audacity always copies the audio into the _data folder. लक्षात ठेवा एमपी 3 किंवा ओजीजी सारख्या संकुचित ऑडिओची आयात करताना ऑडॅसिटी नेहमी ऑडिओ _data फोल्डरमध्ये कॉपी करते.
31 31 When you import a WAV or AIFF there will by default be a warning where you can choose to copy in the file or not. जेव्हा आपण डब्ल्यूएव्ही किंवा एआयएफएफ आयात करता तेव्हा डीफॉल्टनुसार एक चेतावणी होते जिथे आपण फाइलमध्ये कॉपी करणे निवडू शकता किंवा नाही.
32 32 You can always check if you need to copy in files to the current project, and copy them in if necessary at Help > Diagnostics > Check Dependencies.... आपणास सद्य प्रोजेक्टमध्ये फायलींमध्ये कॉपी करण्याची आवश्यकता असल्यास आपण नेहमी तपासू शकता आणि मदत> निदान> चेक अवलंबित्व वर आवश्यक असल्यास त्यामध्ये कॉपी करा. ....
33 33 You can change your preference for future copying in or not in Import/Export Preferences, and you can turn the warning on or off in Warnings Preferences. आपण आयात / निर्यात प्राधान्ये मध्ये भविष्यकाळात कॉपी करण्यासाठी आपले प्राधान्य बदलू शकता आणि आपण चेतावणी प्राधान्यांमध्ये चेतावणी चालू किंवा बंद करू शकता.
34 34 Sending an exported audio file निर्यात केलेली ऑडिओ फाईल पाठवित आहे
35 35 You can also send an exported audio file from your Project to the other user. Only Project–specific information such as label and envelope data will be lost by exporting instead of saving a project. आपण आपल्या प्रोजेक्टमधून निर्यात केलेली ऑडिओ फाईल अन्य वापरकर्त्याला देखील पाठवू शकता. केवळ प्रकल्प label विशिष्ट माहिती जसे की लेबल आणि लिफाफा डेटा प्रकल्प जतन करण्याऐवजी निर्यात करुन गमावला जाईल.
36 36 To export your Project as an audio file, choose the File > Export command. Choices: आपला प्रकल्प ऑडिओ फाईल म्हणून निर्यात करण्यासाठी फाइल> निर्यात आदेश निवडा. निवडी:
37 37 Exporting as a WAV or AIFF file creates a file with no loss of audio quality, but this takes 10 MB of disk space per minute for CD quality (44100 Hz, 16-bit stereo). डब्ल्यूएव्ही किंवा एआयएफएफ फाइल म्हणून निर्यात केल्याने ऑडिओ गुणवत्तेची हानी नसलेली एक फाईल तयार होते, परंतु सीडी गुणवत्तेसाठी प्रति मिनिट 10 एमबी डिस्क स्पेस (44100 हर्ट्ज, 16-बिट स्टिरिओ) घेते.
38 38 Exporting from Audacity as FLAC provides lossless compression and reliably reduces file size by about 40% compared to WAV or AIFF. ऑडसिटीकडून एफएलएसी म्हणून निर्यात करणे लॉसलेस कॉम्प्रेशन प्रदान करते आणि डब्ल्यूएव्ही किंवा एआयएफएफच्या तुलनेत फायलीचे आकार सुमारे 40% ने कमीतकमी कमी करते.
39 39 Exporting as an OGG or MP3 greatly reduces the size of the exported file, at the expense of some quality loss. ओजीजी किंवा एमपी 3 म्हणून निर्यात केल्याने काही गुणवत्तेच्या तोटाच्या किंमतीवर निर्यात केलेल्या फाईलचा आकार मोठ्या प्रमाणात कमी होतो.
40 40 OGG tends to have slightly higher quality than MP3 for the same file size, but not all software players can accept OGG, and most websites hosting audio files expect MP3. ओजीजीकडे समान फाईल आकारासाठी एमपी 3 पेक्षा किंचित उच्च दर्जाची आहे परंतु सर्व सॉफ्टवेअर प्लेयर ओजीजी स्वीकारू शकत नाहीत आणि ऑडिओ फाईल होस्ट करणार्‍या बर्‍याच वेबसाइट एमपी 3 ची अपेक्षा करतात.
41 41 Audacity's default MP3 export bit rate of 128 kbps gives very reasonable sound quality for about 1 MB of space per minute. To compress the MP3 to a still smaller file (at the cost of further quality loss), choose "MP3 Files" in the Export File window, press the Options button then reduce the bit rate in the "Quality" dropdown. For example, a 64 kbps MP3 would take up 0.5 MB (500 kb) per minute, because 64 kbps is half the default bit rate of 128 kbps. ऑडिसिटीचा डीफॉल्ट एमपी 12 एक्सपोर्ट बिट दर 128 केबीपीएस प्रति मिनिट सुमारे 1 एमबी जागेसाठी अगदी वाजवी आवाज गुणवत्ता देते. एमपी 3 ला अद्याप लहान फाईलमध्ये संकुचित करण्यासाठी (पुढील गुणवत्ता हानीच्या किंमतीनुसार), निर्यात फाइल विंडोमध्ये "एमपी 3 फायली" निवडा, पर्याय बटण दाबा आणि नंतर "गुणवत्ता" ड्रॉपडाऊनमधील बिट दर कमी करा. उदाहरणार्थ, 64 केबीपीएस प्रति मिनिट 0.5 एमबी (500 केबी) घेईल, कारण 64 केबीएस 128 केबीपीएसचा डीफॉल्ट बिट दर अर्धा आहे.
42 42 To export MP3 files you must add the LAME encoder to your computer. Instructions for doing this are at Lame Installation in the FAQ. एमपी files फाईल्स एक्सपोर्ट करण्यासाठी तुम्हाला तुमच्या संगणकात लॅम एन्कोडर जोडणे आवश्यक आहे. असे करण्याच्या सूचना एफएक्यू मधील लाम इंस्टॉलेशनवर आहेत.
43 43 For all formats except MP3, you can reduce the exported file size further by reducing its sample rate. Do this by changing the project rate dropdown bottom left of the screen. Reducing the project rate below 44100 Hz is only recommended for speech. एमपी 3 वगळता सर्व स्वरूपांसाठी आपण निर्यात केलेला फाईलचा आकार कमी करुन आणखी कमी करू शकता. स्क्रीनच्या डावीकडील प्रकल्प दर ड्रॉपडाऊन बदलून हे करा. प्रकल्प दर 44100 हर्ट्झपेक्षा कमी करणे केवळ भाषणांसाठी सूचविले जाते.
44 44 Sending files by email ईमेलद्वारे फायली पाठवित आहे
45 45 You should always attach your audio file to the email message. Some email clients can embed the audio inside HTML email messages so that the recipient hears the audio on opening the email without having to open an attachment. However whether the recipient will hear the audio depends on the recipient having an HTML email client (or having HTML enabled without any security restrictions in place). Many email users disable HTML email due to its perceived security risk. आपण आपली ऑडिओ फाइल नेहमीच ईमेल संदेशाशी संलग्न केली पाहिजे. काही ईमेल क्लायंट एचटीएमएल ईमेल संदेशांमध्ये ऑडिओ एम्बेड करू शकतात जेणेकरून एखादा संलग्नक न उघडता प्राप्तकर्ता ईमेल उघडताना ऑडिओ ऐकेल. तथापि प्राप्तकर्ता ऑडिओ ऐकेल की नाही हे प्राप्तकर्त्यावर HTML ईमेल क्लायंट असण्यावर अवलंबून आहे (किंवा त्या ठिकाणी कोणत्याही सुरक्षा निर्बंधाशिवाय HTML सक्षम करणे). बर्‍याच ईमेल वापरकर्त्यांनी त्याच्या सुरक्षिततेच्या जोखमीमुळे HTML ईमेल अक्षम केले.
46 46 Sending files greater than 5 MB by email is usually out of the question due to server bandwidth and storage restrictions. Some webmail services do allow larger messages to be sent. For example, Gmail allows sending messages up to 25 MB, but this still depends on the recipient's email service accepting large attachments (Gmail now accepts incoming attachments up to 50 MB). The best solution is often to use a web-based transfer service like http://www.yousendit.com, http://www.sendspace.com or http://minus.com/, or burn the exported audio file to an audio CD and send it by postal mail. ईमेलद्वारे 5 एमबी पेक्षा जास्त फाईल्स पाठविणे बहुधा सर्व्हर बँडविड्थ आणि स्टोरेज निर्बंधामुळे प्रश्न उद्भवू शकत नाही. काही वेबमेल सेवा मोठ्या संदेश पाठविण्याची परवानगी देतात. उदाहरणार्थ, जीमेल 25 एमबी पर्यंत संदेश पाठविण्याची परवानगी देते, परंतु हे अद्याप प्राप्तकर्त्याच्या ईमेल सेवेवर अवलंबून असते जे मोठ्या संलग्नक स्वीकारत आहे (जीमेल आता 50 एमबी पर्यंतचे इनटेचमेंट स्वीकारते). बर्‍याचदा चांगला उपाय म्हणजे http://www.yousendit.com, http://www.sendspace.com किंवा http://minus.com सारख्या वेब-आधारित हस्तांतरण सेवाचा वापर करणे किंवा यावर निर्यात केलेली ऑडिओ फाइल बर्न करणे एक ऑडिओ सीडी आणि पोस्टल मेलद्वारे पाठवा.
47 47 If you want to burn to an audio CD you need to export 44100 Hz, 16-bit stereo WAV or AIFF files, and tell your burning software to burn an "audio" or "music" CD. For instructions on how to do this, see How to burn CDs. Audio CDs will retain the full quality of the original track in Audacity, can contain 74 – 80 minutes of music and can be played in computers and on any standalone CD player (and some standalone DVD players). आपण ऑडिओ सीडीवर बर्न इच्छित असल्यास आपल्याला 44100 हर्ट्ज, 16-बिट स्टीरिओ डब्ल्यूएव्ही किंवा एआयएफएफ फायली निर्यात करणे आवश्यक आहे आणि आपल्या बर्निंग सॉफ्टवेअरला "ऑडिओ" किंवा "संगीत" सीडी जाळण्यासाठी सांगा. हे कसे करावे यावरील सूचनांसाठी, सीडी कशी बर्न करावी ते पहा. ऑडिओ सीडी ऑडसिटीमध्ये मूळ ट्रॅकची पूर्ण गुणवत्ता राखून ठेवू शकतात, यात 74 ते 80 मिनिटांचे संगीत असू शकते आणि ते संगणकावर आणि कोणत्याही स्वतंत्र सीडी प्लेयरवर (आणि काही स्टँडअलोन डीव्हीडी प्लेयर) प्ले केले जाऊ शकते.
48 48 Exporting from multi-track projects मल्टी-ट्रॅक प्रकल्पांमधून निर्यात करत आहे
49 49 If you have multiple tracks in your Project, Audacity by default will mix these down to a single mono or stereo track when exporting to an audio file. To export separate tracks you can: आपल्याकडे आपल्या प्रकल्पात एकाधिक ट्रॅक असल्यास, ऑडिओ फाइलवर निर्यात करताना डीफॉल्टनुसार ऑडॅसिटी हे एका मोनो किंवा स्टिरीओ ट्रॅकमध्ये मिसळेल. स्वतंत्र ट्रॅक निर्यात करण्यासाठी आपण हे करू शकता:
50 50 Use File > Export > Export Multiple... and "split files based on tracks" फाइल> निर्यात> एकाधिक निर्यात करा ... आणि "ट्रॅकवर आधारित फायली विभाजित करा" वापरा.
51 51 Select the first track, choose File > Export > Export Selected Audio... then repeat in turn for the other tracks. पहिला ट्रॅक निवडा, फाइल> निर्यात> निवडलेला ऑडिओ निवडा ... त्यानंतर इतर ट्रॅकच्या बदल्यात पुन्हा करा.
52 52 The recipient can then shift-select and multiple import each file into separate tracks of an empty Audacity Project. प्राप्तकर्ता नंतर रिक्त ऑडॅसिटी प्रोजेक्टच्या स्वतंत्र ट्रॅकमध्ये प्रत्येक फाईल शिफ्ट-सिलेक्ट आणि एकाधिक आयात करू शकतो.
53 53 Alternatively, current Audacity can export the individual tracks in a Project as one multi-channel audio file. So if you had a Project with six mono or left/right tracks, it could be exported as a one six-channel audio file, and any user of Audacity 1.2 or later can open that file and see the channels displayed as individual tracks. वैकल्पिकरित्या, वर्तमान ऑडीसिटी एका मल्टी-चॅनेल ऑडिओ फाईलच्या रूपात प्रकल्पातील स्वतंत्र ट्रॅक निर्यात करू शकते. तर आपल्याकडे सहा मोनो किंवा डावी / उजवी ट्रॅक असलेली एखादी प्रोजेक्ट असल्यास ती एक सहा-चॅनेल ऑडिओ फाइल म्हणून निर्यात केली जाऊ शकते आणि ऑडसिटी 1.2 किंवा नंतरचा कोणताही वापरकर्ता ती फाईल उघडू शकतो आणि चॅनेल स्वतंत्र ट्रॅक म्हणून प्रदर्शित करू शकतो.
54 54 To use multi-channel export, go to Import/Export Preferences and enable "Use custom mix". मल्टी-चॅनेल निर्यात वापरण्यासाठी आयात / निर्यात प्राधान्ये वर जा आणि "सानुकूल मिश्रित वापरा" सक्षम करा.
55 55 Sending to someone using a different version of Audacity ऑडॅसिटीची भिन्न आवृत्ती वापरत असलेल्या एखाद्यास पाठवित आहे
56 56 If you or the other party created the project with a legacy version of Audacity, please note the following restrictions. जर आपण किंवा इतर पक्षाने ऑडसिटीच्या लेगसी आवृत्तीसह प्रकल्प तयार केला असेल तर कृपया खालील प्रतिबंध लक्षात घ्या.
57 57 If the person you are sending the Project to has a legacy 1.2.x version of Audacity you will need to save the Project in a 1.2.x version of Audacity, because 1.2.x versions of Audacity cannot open Projects created in 1.3 or later versions. ज्याला आपण प्रोजेक्ट पाठवित आहात त्याच्याकडे ऑडॅसिटीचा 1.2.x आवृत्तीचा वारसा असेल तर आपणास ऑडिटीच्या 1.2.x आवृत्तीत प्रकल्प जतन करावा लागेल कारण ऑडसिटीच्या 1.2.x आवृत्ती 1.3 किंवा नंतरच्या आवृत्तीत तयार केलेले प्रकल्प उघडू शकत नाहीत .
58 58 If the person you are sending the Project to has Audacity 1.3.2 or later, they should be able to open a project created in 1.2.x, but ask them to back up the AUP file and _data folder on receipt to be sure. To make a backup, copy the AUP file and _data folder and paste them into a different folder. Once they do save that project in 1.3.x or later and send it back to you, you will need 1.3.x or later yourself to open it. ज्याला आपण प्रोजेक्ट पाठवित आहात त्याच्याकडे ऑडसिटी १. later.२ किंवा नंतरचे असल्यास, त्यांनी 1.2.x मध्ये तयार केलेला प्रकल्प उघडण्यास सक्षम असावा, परंतु पावतीवर एयूपी फाइल आणि _डेटा फोल्डरचा बॅक अप घेण्यास सांगा. बॅकअप घेण्यासाठी, AUP फाईल आणि _डेटा फोल्डर कॉपी करा आणि त्यांना भिन्न फोल्डरमध्ये पेस्ट करा. एकदा त्यांनी या प्रकल्पाची 1.3.x किंवा नंतर जतन केली आणि ती तुम्हाला परत पाठविल्यानंतर आपणास तो उघडण्यासाठी 1.3.x किंवा नंतर स्वत: ची आवश्यकता असेल.

If that person sends you a Project created in 1.0.0, you should generally be able to open it in 1.1.0 to 1.2.6, but Audacity's conversion of a 1.0.0 Project to the new format used by 1.1.0 or later does not always work. You should not open Projects created in 1.0.0 in current versions of Audacity. आपण ज्या व्यक्तीला प्रोजेक्ट पाठवत आहात तो ऑडसिटी 1.0.0 वापरण्यासाठी (मॅक ओएस 9 वरील वापरकर्ते ऑडसिटीच्या या आवृत्तीत मर्यादित आहेत), आपण प्रोजेक्ट 1.0.0 मध्ये तयार करणे आवश्यक आहे, कारण 1.0.0 1.1 मध्ये तयार केलेले प्रकल्प उघडू शकत नाही. 0 किंवा नंतर जर ती व्यक्ती आपल्याला 1.0.0 मध्ये तयार केलेला प्रकल्प पाठवित असेल तर आपण सामान्यत: ते 1.1.0 ते 1.2.6 मध्ये उघडण्यास सक्षम असावे परंतु ऑडसिटीने 1.0.0 प्रोजेक्टचे रुपांतर 1.1.0 किंवा नंतरच्या नवीन स्वरूपात केले. नेहमी कार्य करत नाही. आपण ऑडसिटीच्या वर्तमान आवृत्त्यांमध्ये 1.0.0 मध्ये तयार केलेले प्रकल्प उघडू नका.
60 60 Sending to someone using a different operating system or platform भिन्न ऑपरेटिंग सिस्टम किंवा प्लॅटफॉर्म वापरत असलेल्या एखाद्यास पाठवित आहे
